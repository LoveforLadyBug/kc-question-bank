## 문제

카카오클라우드 MemStore가 기존 디스크 기반 데이터베이스 대비 갖는 가장 큰 기술적 특징은?

## 보기

- A. 데이터를 서버 메모리에 저장하여 빠른 읽기/쓰기 성능을 제공한다.
- B. 데이터를 여러 물리 디스크에 분산 저장하여 내구성을 극대화한다.
- C. SQL 표준 쿼리를 완벽하게 지원하여 복잡한 조인 연산이 가능하다.
- D. 스키마 없이 JSON 문서를 저장하는 도큐먼트 지향 구조를 제공한다.

## 정답

A

## 해설

MemStore는 인메모리 데이터 스토어 서비스로, 데이터를 서버 메모리(RAM)에 저장하기 때문에 디스크 I/O가 발생하지 않아 매우 빠른 읽기/쓰기 성능을 제공합니다.
이러한 특성 때문에 캐싱, 세션 관리, 실시간 리더보드 등 지연 시간이 낮아야 하는 워크로드에 적합합니다.

## 오답 포인트

- B: 다중 물리 디스크 분산 저장은 인메모리 방식의 특징이 아니라 분산 스토리지 시스템의 특징입니다.
- C: SQL 표준 쿼리와 복잡한 조인 연산은 RDBMS(MySQL, PostgreSQL)의 특징이지 MemStore의 특징이 아닙니다.
- D: JSON 도큐먼트 저장은 NoSQL 도큐먼트 DB의 특징이며, MemStore의 인메모리 방식과 다릅니다.
