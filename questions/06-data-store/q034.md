## 문제

카카오클라우드 MemStore 서비스의 주요 목적으로 가장 적절한 것은?

## 보기

- A. 대용량 파일을 분산 저장하고 버전 관리를 제공하는 오브젝트 스토리지 서비스
- B. 서버 운영·관리 부담을 줄이고 데이터베이스를 쉽게 사용할 수 있도록 지원하는 인메모리 데이터 스토어 서비스
- C. 관계형 데이터를 영구 저장하고 복잡한 SQL 쿼리를 처리하는 RDBMS 서비스
- D. 블록 스토리지 형태로 데이터를 저장하고 가상 머신에 마운트하는 스토리지 서비스

## 정답

B

## 해설

MemStore는 서버 이중 운영·관리 부담을 덜어주고, 데이터베이스를 쉽게 사용할 수 있도록 지원하는 완전 관리형 인메모리 데이터 스토어 서비스입니다.
인메모리 방식은 디스크가 아닌 서버 메모리에 데이터를 저장하여 매우 빠른 읽기/쓰기 성능을 제공합니다.

## 오답 포인트

- A: 오브젝트 스토리지는 별도의 스토리지 서비스이며, MemStore의 역할과 다릅니다.
- C: 관계형 데이터 영구 저장 및 SQL 처리는 MySQL 또는 PostgreSQL 서비스의 역할입니다.
- D: 블록 스토리지는 가상 머신용 스토리지 서비스로, MemStore와 전혀 다른 범주입니다.
