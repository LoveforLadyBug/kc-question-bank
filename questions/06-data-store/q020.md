## 문제

카카오클라우드 Data Store의 세 가지 서비스(MySQL, PostgreSQL, MemStore)와 그 기반 기술을 올바르게 연결한 것은?

## 보기

- A. MySQL → ORDBMS, PostgreSQL → RDBMS, MemStore → 인메모리
- B. MySQL → RDBMS, PostgreSQL → ORDBMS, MemStore → 인메모리
- C. MySQL → 인메모리, PostgreSQL → RDBMS, MemStore → ORDBMS
- D. MySQL → RDBMS, PostgreSQL → 인메모리, MemStore → ORDBMS

## 정답

B

## 해설

카카오클라우드 Data Store에서 MySQL은 오픈 소스 RDBMS 기반, PostgreSQL은 오픈 소스 ORDBMS 기반, MemStore는 완전 관리형 인메모리 데이터 스토어로 각각 구성됩니다.
세 서비스는 서로 다른 기반 기술을 갖추고 있어, 고객은 워크로드 특성에 따라 적합한 서비스를 선택할 수 있습니다.

## 오답 포인트

- A: MySQL과 PostgreSQL의 기반 기술(RDBMS, ORDBMS)이 서로 바뀌어 있어 잘못된 연결입니다.
- C: MySQL은 인메모리 서비스가 아닌 RDBMS 기반이며, MemStore가 인메모리 서비스에 해당합니다.
- D: PostgreSQL은 인메모리 서비스가 아닌 ORDBMS 기반이며, MemStore는 ORDBMS가 아닌 인메모리 서비스입니다.
