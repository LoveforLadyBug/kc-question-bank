## 문제

카카오클라우드 Data Store의 MySQL 서비스와 PostgreSQL 서비스를 비교할 때, 두 서비스의 차이점으로 올바른 것은?

## 보기

- A. MySQL은 인메모리 방식, PostgreSQL은 디스크 기반 방식이다.
- B. MySQL은 RDBMS 기반, PostgreSQL은 ORDBMS 기반으로 구축된 서비스이다.
- C. MySQL은 완전 관리형이지만, PostgreSQL은 사용자가 직접 운영해야 한다.
- D. MySQL은 오픈 소스가 아니며, PostgreSQL만 오픈 소스 기반이다.

## 정답

B

## 해설

MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)을 기반으로, PostgreSQL은 오픈 소스 객체-관계형 데이터베이스(ORDBMS)를 기반으로 각각 구축된 완전 관리형 서비스입니다.
두 서비스 모두 오픈 소스 기반의 완전 관리형 서비스라는 공통점이 있으나, 기반 데이터베이스 유형(RDBMS vs ORDBMS)에서 차이가 있습니다.

## 오답 포인트

- A: 인메모리 방식은 MemStore에 해당하며, MySQL과 PostgreSQL은 모두 디스크 기반 관계형 데이터베이스입니다.
- C: MySQL과 PostgreSQL 모두 카카오클라우드에서 완전 관리형으로 제공되며, 사용자가 직접 운영할 필요가 없습니다.
- D: MySQL도 오픈 소스로 제공되는 RDBMS이며, 두 서비스 모두 오픈 소스 기반입니다.
