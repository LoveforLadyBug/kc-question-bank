## 문제

카카오클라우드 Data Store에서 인메모리 데이터 처리에 특화된 서비스는 무엇인가?

## 보기

- A. MySQL
- B. PostgreSQL
- C. MemStore
- D. Object Storage

## 정답

C

## 해설

MemStore는 카카오클라우드 Data Store 내에서 인메모리 데이터 저장 및 처리를 목적으로 설계된 완전 관리형 서비스입니다.
MySQL과 PostgreSQL은 디스크 기반의 관계형 데이터베이스 서비스이며, Object Storage는 Data Store와 별개의 서비스입니다.

## 오답 포인트

- A: MySQL은 RDBMS 기반의 디스크 저장형 관계형 데이터베이스 서비스로, 인메모리 처리에 특화되지 않습니다.
- B: PostgreSQL은 ORDBMS 기반의 관계형 데이터베이스 서비스로, 인메모리 전용 서비스가 아닙니다.
- D: Object Storage는 Data Store가 아닌 별도의 스토리지 서비스 범주에 속합니다.
