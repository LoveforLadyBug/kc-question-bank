## 문제

카카오클라우드 Data Store의 PostgreSQL 서비스가 기반으로 하는 데이터베이스 유형은?

## 보기

- A. 관계형 데이터베이스 관리 시스템 (RDBMS)
- B. 비관계형 데이터베이스 (NoSQL)
- C. 객체-관계형 데이터베이스 (ORDBMS)
- D. 인메모리 데이터 스토어

## 정답

C

## 해설

카카오클라우드 PostgreSQL 서비스는 오픈 소스로 제공되는 객체-관계형 데이터베이스(ORDBMS)인 PostgreSQL을 기반으로 구축된 완전 관리형 서비스입니다.
ORDBMS는 관계형 모델에 객체 지향 개념을 결합한 형태로, 복잡한 데이터 타입과 상속 등의 기능을 지원하는 것이 특징입니다.

## 오답 포인트

- A: RDBMS 기반은 PostgreSQL이 아닌 MySQL 서비스에 해당하는 설명입니다.
- B: PostgreSQL은 관계형 모델을 확장한 ORDBMS이며, NoSQL 데이터베이스가 아닙니다.
- D: 인메모리 데이터 스토어는 MemStore 서비스에 대한 설명입니다.
