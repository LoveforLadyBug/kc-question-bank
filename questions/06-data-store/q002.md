## 문제

카카오클라우드 Data Store에 포함된 서비스를 모두 올바르게 나열한 것은?

## 보기

- A. MySQL, PostgreSQL, Redis
- B. MySQL, PostgreSQL, MemStore
- C. MariaDB, PostgreSQL, MemStore
- D. MySQL, MongoDB, MemStore

## 정답

B

## 해설

카카오클라우드 Data Store에는 MySQL, PostgreSQL, MemStore 세 가지 서비스가 포함되어 있으며, 모두 고성능과 고가용성을 보장합니다.
Redis, MariaDB, MongoDB는 Data Store의 공식 구성 서비스에 해당하지 않습니다.

## 오답 포인트

- A: Redis는 카카오클라우드 Data Store의 공식 구성 서비스가 아닙니다. 인메모리 서비스는 MemStore입니다.
- C: MariaDB는 Data Store에 포함되지 않으며, MySQL이 관계형 DB 서비스로 제공됩니다.
- D: MongoDB는 Data Store에 포함되지 않으며, 문서형 NoSQL 서비스는 별도입니다.
