## 문제

카카오클라우드 MemStore 서비스의 주요 목적으로 가장 적절한 것은?

## 보기

- A. 대용량 파일을 오브젝트 형태로 영구 저장하는 스토리지 제공
- B. 서버 운영·관리 부담을 낮추고 데이터베이스를 쉽게 사용할 수 있도록 지원하는 완전 관리형 인메모리 데이터 스토어
- C. 관계형 스키마를 기반으로 복잡한 트랜잭션을 처리하는 RDBMS 서비스
- D. 블록 스토리지 형태로 VM 인스턴스에 마운트하여 사용하는 저장 장치

## 정답

B

## 해설

MemStore는 서버 이중화·운영 관리 부담을 덜어주고, 데이터베이스를 쉽게 사용할 수 있도록 돕는 완전 관리형 인메모리 데이터 스토어 서비스입니다.
인메모리 방식으로 동작하기 때문에 디스크 기반 데이터베이스보다 빠른 읽기·쓰기 속도가 필요한 캐시·세션 저장 등의 사용 사례에 적합합니다.

## 오답 포인트

- A: 오브젝트 영구 저장은 카카오클라우드 Object Storage의 역할입니다.
- C: 복잡한 트랜잭션 처리를 위한 RDBMS는 MySQL 또는 PostgreSQL의 특징입니다.
- D: 블록 스토리지는 카카오클라우드 Block Storage 서비스의 특징입니다.
