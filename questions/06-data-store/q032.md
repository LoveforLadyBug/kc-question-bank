## 문제

카카오클라우드 Data Store의 MySQL 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 오픈 소스 객체-관계형 데이터베이스(ORDBMS)를 기반으로 구축된 관리형 서비스이다.
- B. 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)을 기반으로 구축된 관리형 서비스이다.
- C. 서버 메모리에 데이터를 저장하는 인메모리 데이터 스토어 서비스이다.
- D. NoSQL 기반의 분산 데이터베이스를 완전 관리형으로 제공하는 서비스이다.

## 정답

B

## 해설

카카오클라우드 MySQL 서비스는 오픈 소스로 제공되는 관계형 데이터베이스 관리 시스템(RDBMS)인 MySQL을 기반으로 구축된 완전 관리형 데이터베이스 서비스입니다.
사용자는 서버 설치·운영 없이 MySQL 환경을 바로 사용할 수 있으며, 고성능·고가용성이 보장됩니다.

## 오답 포인트

- A: 객체-관계형 데이터베이스(ORDBMS)를 기반으로 하는 서비스는 MySQL이 아닌 PostgreSQL입니다.
- C: 서버 메모리에 데이터를 저장하는 인메모리 서비스는 MemStore에 대한 설명입니다.
- D: NoSQL 기반 분산 데이터베이스는 Data Store의 구성 서비스에 해당하지 않습니다.
