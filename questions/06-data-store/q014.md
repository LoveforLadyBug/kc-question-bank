## 문제

카카오클라우드 Data Store의 MemStore 서비스가 제공하는 핵심 가치로 가장 적절한 것은?

## 보기

- A. 오픈 소스 RDBMS 기반의 트랜잭션 처리 최적화
- B. 객체-관계형 모델을 통한 복잡한 쿼리 지원
- C. 서버 운영 관리 부담을 낮추고 데이터베이스를 쉽게 사용할 수 있도록 지원하는 인메모리 서비스
- D. 대용량 파일 데이터를 영구 저장하기 위한 블록 스토리지 연동

## 정답

C

## 해설

MemStore는 서버의 운영 및 관리 부담을 덜어주고 데이터베이스를 쉽게 사용할 수 있도록 돕는 완전 관리형 인메모리 데이터 스토어 서비스입니다.
인메모리 방식은 디스크 기반 데이터베이스보다 빠른 읽기/쓰기 성능을 제공하므로, 캐싱이나 세션 관리 등의 고속 처리가 필요한 워크로드에 적합합니다.

## 오답 포인트

- A: RDBMS 기반 트랜잭션 처리 최적화는 MySQL 서비스의 특성에 가까운 설명입니다.
- B: 객체-관계형 모델과 복잡한 쿼리 지원은 PostgreSQL 서비스의 특성입니다.
- D: 블록 스토리지 연동은 Data Store가 아닌 별도의 스토리지 서비스 영역에 해당합니다.
