## 문제

카카오클라우드 Data Store에 포함된 세 가지 서비스와 각 서비스의 기반 기술을 올바르게 연결한 것은?

## 보기

- A. MySQL → ORDBMS, PostgreSQL → RDBMS, MemStore → 인메모리 스토어
- B. MySQL → RDBMS, PostgreSQL → ORDBMS, MemStore → 인메모리 스토어
- C. MySQL → RDBMS, PostgreSQL → NoSQL, MemStore → 블록 스토리지
- D. MySQL → 인메모리 스토어, PostgreSQL → RDBMS, MemStore → ORDBMS

## 정답

B

## 해설

카카오클라우드 Data Store에서 MySQL은 오픈 소스 RDBMS 기반, PostgreSQL은 오픈 소스 ORDBMS 기반, MemStore는 완전 관리형 인메모리 데이터 스토어로 각각 다른 기술 유형에 해당합니다.
세 서비스의 기술적 특성을 구분하는 것은 적절한 서비스를 선택하는 데 핵심적인 기준이 됩니다.

## 오답 포인트

- A: MySQL과 PostgreSQL의 유형(RDBMS, ORDBMS)이 서로 바뀌어 잘못 연결되었습니다.
- C: PostgreSQL은 NoSQL이 아닌 ORDBMS이며, MemStore는 블록 스토리지가 아닌 인메모리 스토어입니다.
- D: MySQL은 인메모리 스토어가 아닌 RDBMS이며, MemStore는 ORDBMS가 아닌 인메모리 스토어입니다.
