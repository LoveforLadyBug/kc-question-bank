## 문제

다음 중 카카오클라우드 MemStore 서비스를 선택하기에 가장 적합한 시나리오는?

## 보기

- A. 복잡한 조인 쿼리가 필요한 대규모 트랜잭션 처리 시스템
- B. 지리 정보 및 사용자 정의 데이터 타입을 활용하는 분석 플랫폼
- C. 빠른 응답 속도가 요구되는 세션 데이터 또는 캐시 저장소
- D. 장기 보관이 필요한 대용량 로그 데이터의 영구 저장소

## 정답

C

## 해설

MemStore는 인메모리 방식으로 동작하여 디스크 기반 데이터베이스 대비 월등히 빠른 읽기/쓰기 성능을 제공하므로, 세션 관리나 캐싱처럼 낮은 지연 시간이 요구되는 워크로드에 최적화되어 있습니다.
또한 서버 운영 관리 부담을 낮춰 개발자가 인프라보다 비즈니스 로직에 집중할 수 있도록 돕습니다.

## 오답 포인트

- A: 복잡한 조인과 대규모 트랜잭션 처리에는 MySQL과 같은 RDBMS 기반 서비스가 더 적합합니다.
- B: 지리 정보 및 사용자 정의 데이터 타입 처리는 ORDBMS인 PostgreSQL의 강점에 해당합니다.
- D: 대용량 로그 데이터의 영구 저장에는 내구성을 갖춘 디스크 기반 스토리지 또는 데이터베이스 서비스가 적합합니다.
