## 문제

카카오클라우드 Data Store의 MySQL 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 오픈 소스 관계형 데이터베이스(RDBMS)인 MySQL을 기반으로 구축된 완전 관리형 서비스이다.
- B. 오픈 소스 객체-관계형 데이터베이스(ORDBMS)인 MySQL을 기반으로 구축된 서비스이다.
- C. MySQL은 카카오클라우드에서 자체 개발한 독점 데이터베이스 엔진을 사용한다.
- D. MySQL 서비스는 인메모리 방식으로 동작하여 영구 저장을 지원하지 않는다.

## 정답

A

## 해설

카카오클라우드 MySQL 서비스는 오픈 소스로 제공되는 관계형 데이터베이스 관리 시스템(RDBMS)인 MySQL을 기반으로 구축된 완전 관리형 데이터베이스 서비스입니다.
완전 관리형이므로 고객은 서버 운영·패치 등의 부담 없이 데이터베이스를 사용할 수 있습니다.

## 오답 포인트

- B: 객체-관계형 데이터베이스(ORDBMS)는 MySQL이 아닌 PostgreSQL의 특징입니다.
- C: MySQL은 카카오클라우드 독점 엔진이 아니라 오픈 소스 MySQL 엔진을 기반으로 합니다.
- D: MySQL은 디스크 기반 영구 저장을 지원하는 RDBMS이며, 인메모리 방식은 MemStore의 특징입니다.
