## 문제

카카오클라우드 Data Store에 포함된 관리형 데이터베이스 서비스 세 가지를 올바르게 나열한 것은?

## 보기

- A. MySQL, PostgreSQL, Redis
- B. MySQL, PostgreSQL, MemStore
- C. MySQL, MongoDB, MemStore
- D. PostgreSQL, MariaDB, MemStore

## 정답

B

## 해설

카카오클라우드 Data Store는 고성능과 고가용성을 보장하는 MySQL, PostgreSQL, MemStore 세 가지 완전 관리형 서비스를 포함합니다.
각 서비스는 서로 다른 데이터베이스 유형(관계형, 객체-관계형, 인메모리)을 기반으로 하며, 고객의 다양한 요구사항에 맞는 환경을 제공합니다.

## 오답 포인트

- A: Redis는 카카오클라우드 Data Store의 구성 서비스가 아닙니다. 인메모리 서비스로는 MemStore가 제공됩니다.
- C: MongoDB는 NoSQL 문서형 데이터베이스로, Data Store 구성 서비스에 포함되지 않습니다.
- D: MariaDB는 Data Store 구성 서비스에 포함되지 않으며, MySQL이 관계형 데이터베이스 서비스로 제공됩니다.
