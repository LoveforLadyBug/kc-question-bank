## 문제

카카오클라우드 Data Store에서 PostgreSQL 서비스의 기반 기술을 가장 정확하게 설명한 것은?

## 보기

- A. 오픈 소스 RDBMS인 MySQL을 기반으로 구축된 관리형 서비스이다.
- B. 오픈 소스 인메모리 캐시 엔진을 기반으로 구축된 관리형 서비스이다.
- C. 오픈 소스 객체-관계형 데이터베이스(ORDBMS)인 PostgreSQL을 기반으로 구축된 관리형 서비스이다.
- D. 분산 NoSQL 데이터베이스를 기반으로 구축된 관리형 서비스이다.

## 정답

C

## 해설

카카오클라우드 PostgreSQL 서비스는 오픈 소스 객체-관계형 데이터베이스(ORDBMS)인 PostgreSQL을 기반으로 구축된 완전 관리형 데이터베이스 서비스입니다.
ORDBMS는 전통적인 관계형 모델에 객체 지향 기능(사용자 정의 타입, 상속 등)을 결합한 데이터베이스 유형입니다.

## 오답 포인트

- A: MySQL 기반 서비스는 PostgreSQL이 아닌 Data Store의 MySQL 서비스에 해당합니다.
- B: 인메모리 캐시 엔진 기반 서비스는 MemStore에 대한 설명입니다.
- D: 분산 NoSQL 데이터베이스는 Data Store의 구성 서비스에 해당하지 않습니다.
