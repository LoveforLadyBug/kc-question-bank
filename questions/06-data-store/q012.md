## 문제

카카오클라우드 Data Store의 MySQL 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 객체-관계형 데이터베이스(ORDBMS)를 기반으로 구축된 완전 관리형 서비스이다.
- B. 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)인 MySQL을 기반으로 구축된 완전 관리형 서비스이다.
- C. 서버의 운영 관리 부담을 줄여주는 완전 관리형 인메모리 데이터 스토어 서비스이다.
- D. 비관계형(NoSQL) 데이터베이스를 기반으로 고성능 캐싱을 제공하는 서비스이다.

## 정답

B

## 해설

카카오클라우드 MySQL 서비스는 오픈 소스로 제공되는 관계형 데이터베이스 관리 시스템(RDBMS)인 MySQL을 기반으로 구축된 완전 관리형 데이터베이스 서비스입니다.
사용자가 직접 서버를 운영하지 않아도 되는 완전 관리형 환경을 제공하며, 고성능과 고가용성을 보장합니다.

## 오답 포인트

- A: 객체-관계형 데이터베이스(ORDBMS) 기반은 MySQL이 아닌 PostgreSQL 서비스에 대한 설명입니다.
- C: 서버 운영 관리 부담을 줄여주는 인메모리 데이터 스토어는 MemStore 서비스에 대한 설명입니다.
- D: MySQL은 관계형 데이터베이스(RDBMS) 기반 서비스이며, NoSQL 기반 캐싱 서비스가 아닙니다.
