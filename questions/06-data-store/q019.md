## 문제

카카오클라우드 PostgreSQL 서비스가 ORDBMS 기반이라는 특성과 가장 관련이 깊은 설명은?

## 보기

- A. 데이터를 디스크가 아닌 메모리에 저장하여 초고속 처리를 지원한다.
- B. 관계형 모델에 객체 지향 개념을 결합하여 복잡한 데이터 구조를 표현할 수 있다.
- C. 단순한 키-값 형태의 데이터 저장에 특화된 구조를 갖는다.
- D. 수평적 샤딩을 기본으로 지원하여 무한 확장이 가능하다.

## 정답

B

## 해설

ORDBMS(객체-관계형 데이터베이스 관리 시스템)는 전통적인 관계형 모델에 객체 지향 개념(사용자 정의 타입, 상속, 복합 데이터 구조 등)을 통합한 데이터베이스 유형입니다.
PostgreSQL은 이러한 ORDBMS의 대표적인 오픈 소스 구현체로, 복잡한 데이터 타입이나 확장 기능이 필요한 시스템에 적합합니다.

## 오답 포인트

- A: 메모리 기반 초고속 처리는 ORDBMS 특성이 아닌 MemStore(인메모리 서비스)의 특성입니다.
- C: 키-값 형태의 데이터 저장 특화는 인메모리 데이터 스토어 또는 NoSQL 계열 서비스의 특성입니다.
- D: 수평적 샤딩을 통한 무한 확장은 일반적으로 NoSQL 또는 분산 데이터베이스의 특성으로, ORDBMS의 핵심 특성이 아닙니다.
