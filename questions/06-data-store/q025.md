## 문제

카카오클라우드 Data Store의 공통적인 제공 방식을 가장 잘 설명한 것은?

## 보기

- A. 고객이 직접 서버 OS를 설치하고 데이터베이스를 구성하는 자가 관리형 서비스
- B. 고성능과 고가용성을 보장하는 클라우드 기반 완전 관리형 데이터베이스 서비스
- C. 단일 노드로만 운영되며 백업 기능이 제공되지 않는 경량 데이터베이스 서비스
- D. 온프레미스 환경에 설치하여 사용하는 라이선스 기반 데이터베이스 서비스

## 정답

B

## 해설

카카오클라우드 Data Store는 고성능과 고가용성을 보장하는 클라우드 기반 완전 관리형(Managed) 데이터베이스 서비스 그룹입니다.
완전 관리형이란 서버 패치, 백업, 모니터링 등 운영 부담을 클라우드 사업자가 담당하여 고객이 애플리케이션 개발에 집중할 수 있도록 돕는 방식입니다.

## 오답 포인트

- A: 자가 관리형(Self-managed)은 Data Store와 반대 개념으로, IaaS 위에서 직접 DB를 운영하는 방식입니다.
- C: Data Store는 고가용성을 목표로 하며 단일 노드 전용 서비스가 아닙니다.
- D: Data Store는 클라우드 서비스로 제공되며, 온프레미스 설치형 라이선스 제품이 아닙니다.
