## 문제

카카오클라우드 Data Store에 포함된 관리형 데이터베이스 서비스를 모두 올바르게 나열한 것은?

## 보기

- A. MySQL, PostgreSQL, Redis
- B. MySQL, PostgreSQL, MemStore
- C. MySQL, MongoDB, MemStore
- D. PostgreSQL, MariaDB, MemStore

## 정답

B

## 해설

카카오클라우드 Data Store는 MySQL, PostgreSQL, MemStore 세 가지 관리형 데이터베이스 서비스로 구성됩니다.
각 서비스는 높은 성능과 고가용성을 제공하며, 서버 운영·관리 부담을 줄이는 완전 관리형(Fully Managed) 형태로 제공됩니다.

## 오답 포인트

- A: Redis는 카카오클라우드 Data Store의 구성 서비스가 아니며, 인메모리 서비스는 MemStore로 제공됩니다.
- C: MongoDB는 Data Store에 포함되지 않으며, 관계형 DB 서비스는 MySQL과 PostgreSQL입니다.
- D: MariaDB는 Data Store 구성 서비스가 아닙니다. MySQL과 PostgreSQL이 관계형 DB 서비스로 제공됩니다.
