## 문제

다음 중 카카오클라우드 Data Store MySQL을 선택하기에 가장 적합한 사용 사례는?

## 보기

- A. 초당 수십만 건의 캐시 조회가 필요한 세션 관리 시스템
- B. 오픈 소스 RDBMS 기반의 구조화된 데이터를 관계형 스키마로 관리하는 웹 서비스 백엔드
- C. 비정형 JSON 문서를 스키마 없이 저장하고 탐색하는 콘텐츠 플랫폼
- D. 대규모 바이너리 파일(이미지, 동영상)을 저장하고 CDN으로 배포하는 미디어 서비스

## 정답

B

## 해설

카카오클라우드 MySQL은 오픈 소스 RDBMS인 MySQL을 기반으로 하므로, 관계형 스키마를 활용한 구조화된 데이터 관리가 필요한 웹 서비스 백엔드에 가장 적합합니다.
RDBMS는 ACID 트랜잭션, 외래 키 제약, 복잡한 JOIN 쿼리 등을 지원하여 정형 데이터를 다루는 서비스에 최적화되어 있습니다.

## 오답 포인트

- A: 초당 수십만 건의 캐시 조회처럼 고속 인메모리 처리가 필요한 경우는 MemStore가 더 적합합니다.
- C: 비정형 JSON 문서를 스키마 없이 저장하는 요구는 NoSQL(도큐먼트 DB) 방식에 해당하며 MySQL의 주 용도가 아닙니다.
- D: 대규모 바이너리 파일 저장 및 CDN 배포는 Object Storage와 CDN 서비스의 역할입니다.
