## 문제

카카오클라우드 Data Store의 MemStore 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 디스크 기반의 관계형 데이터베이스를 제공하는 관리형 서비스이다.
- B. 서버 이중화와 운영 관리 부담을 줄여주는 완전 관리형 인메모리 데이터 스토어 서비스이다.
- C. 고객이 직접 서버를 프로비저닝하고 운영해야 하는 인메모리 서비스이다.
- D. 대용량 파일 저장 및 객체 스토리지 기능에 특화된 서비스이다.

## 정답

B

## 해설

MemStore는 서버의 이중화 및 운영 관리 부담을 줄여주고 데이터베이스를 쉽게 사용할 수 있도록 도와주는 완전 관리형 인메모리 데이터 스토어 서비스입니다.
관리형 서비스이기 때문에 고객은 서버를 직접 운영하지 않아도 되며, 인메모리 특성상 빠른 데이터 접근이 가능합니다.

## 오답 포인트

- A: MemStore는 디스크 기반이 아닌 인메모리(메모리 기반) 데이터 스토어 서비스입니다.
- C: MemStore는 완전 관리형 서비스로, 고객이 직접 서버를 프로비저닝하고 운영할 필요가 없습니다.
- D: 대용량 파일 저장 및 객체 스토리지는 Object Storage 등 별도 서비스의 역할이며, MemStore와 관련이 없습니다.
