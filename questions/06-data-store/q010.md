## 문제

스타트업 A사는 애플리케이션 서비스를 위해 카카오클라우드 Data Store 도입을 검토 중입니다. 서버 운영 인력이 부족하여 데이터베이스 인프라 관리 부담을 최소화하고 싶을 때, Data Store를 선택하는 가장 적절한 이유는?

## 보기

- A. 데이터베이스 쿼리 튜닝을 자동으로 대신 처리해 주기 때문이다.
- B. 서버 이중화, 패치 등 인프라 운영을 클라우드 제공사가 담당하는 완전 관리형 서비스이기 때문이다.
- C. 모든 종류의 데이터베이스 스키마를 자동으로 설계해 주기 때문이다.
- D. 애플리케이션 코드 배포와 데이터베이스 운영을 하나의 인터페이스에서 동시에 처리하기 때문이다.

## 정답

B

## 해설

카카오클라우드 Data Store는 완전 관리형 서비스로, 서버 이중화(HA), 운영 체제 패치, 백업 등의 인프라 운영 부담을 줄여줍니다. 이는 운영 인력이 부족한 스타트업 환경에 적합한 핵심 이유입니다.
관리형 서비스는 고객이 데이터베이스 본연의 활용에 집중할 수 있도록 돕는 것이 목적이며, 쿼리 튜닝이나 스키마 설계까지 대신해 주지는 않습니다.

## 오답 포인트

- A: Data Store는 인프라 운영을 관리해 주지만, SQL 쿼리 튜닝을 자동으로 수행해 주는 서비스는 아닙니다.
- C: 데이터베이스 스키마 설계는 고객의 비즈니스 로직에 따라 고객이 직접 수행해야 하는 영역입니다.
- D: 애플리케이션 코드 배포와 데이터베이스 운영을 통합 처리하는 기능은 Data Store의 제공 범위에 해당하지 않습니다.
