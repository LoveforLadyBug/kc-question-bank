## 문제

다음 중 카카오클라우드 Data Store MySQL을 사용할 때 고객이 직접 처리하지 않아도 되는 항목은?

## 보기

- A. 애플리케이션에서 사용할 SQL 쿼리 작성
- B. 데이터베이스 스키마 설계
- C. 서버 운영 체제 패치 및 이중화 관리
- D. 데이터베이스 사용자 계정 생성

## 정답

C

## 해설

Data Store MySQL은 완전 관리형 서비스로, 서버 운영 체제의 패치, 이중화(HA) 구성 등 인프라 운영 관리는 카카오클라우드가 담당합니다.
SQL 쿼리 작성, 스키마 설계, DB 계정 관리 등 데이터베이스 논리적 운영은 여전히 고객의 책임 영역입니다.

## 오답 포인트

- A: SQL 쿼리 작성은 애플리케이션 개발자가 직접 수행해야 하는 고객 책임 영역입니다.
- B: 데이터베이스 스키마 설계는 서비스 요구사항에 따라 고객이 직접 수행하는 영역입니다.
- D: 데이터베이스 사용자 계정 생성 및 권한 부여는 고객이 직접 관리하는 논리적 운영 범위에 해당합니다.
