## 문제

카카오클라우드 IAM에서 사용자를 **그룹에서 제거**할 때의 동작으로 옳은 것은?

## 보기

- A. 그룹에서 제거되면 해당 그룹을 통해 부여된 역할과 직접 부여된 역할이 모두 회수된다.
- B. 그룹에서 제거되면 해당 그룹을 통해 부여된 역할은 회수되고, 직접 부여된 역할은 유지되며, 해당 그룹을 통해 발급된 API 토큰은 즉시 만료된다.
- C. 그룹에서 제거되어도 기존 역할과 API 토큰은 24시간 동안 유효하게 유지된다.
- D. 그룹에서 제거되면 모든 API 토큰이 즉시 만료되고, 새 토큰을 발급받기 위해 재로그인이 필요하다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 그룹에서 사용자를 제거하거나 그룹을 삭제하면 해당 그룹을 통해 부여된 역할은 회수되지만, 사용자에게 직접 부여된 역할은 그대로 유지됩니다. 또한 그룹에서 제거되면 해당 그룹을 통해 발급된 API 토큰은 즉시 만료됩니다.

## 오답 포인트

- A: 그룹 제거 시 직접 부여된 역할까지 모두 회수되지는 않습니다. 직접 부여된 역할은 그룹 제거와 관계없이 유지됩니다.
- C: 그룹 제거 후 역할과 토큰이 24시간 동안 유효하다는 설명은 잘못되었습니다. 해당 그룹을 통해 발급된 API 토큰은 즉시 만료됩니다.
- D: 그룹 제거 시 모든 API 토큰이 만료되는 것이 아니라, 해당 그룹을 통해 발급된 토큰만 만료됩니다. 직접 부여된 역할로 발급된 토큰은 유지됩니다.
