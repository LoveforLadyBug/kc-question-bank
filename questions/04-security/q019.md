## 문제

카카오클라우드 IAM에서 **자원 삭제 시 본인 확인** 방법으로 새로 추가된 방식은?

## 보기

- A. 이메일 인증 코드 입력
- B. 비밀번호 재인증 방식
- C. SMS 문자 인증
- D. 관리자 승인 요청

## 정답

B

## 해설

카카오클라우드 공식 문서(2026.03.24 업데이트)에 따르면, 조직 내 사용자 및 프로젝트 서비스 계정 삭제 시 기존의 단순 확인 문구 입력 방식 외에 **비밀번호 재인증 방식**이 추가되었습니다. 중요 자원 삭제 시 본인 확인 절차를 강화하여 관리자의 조작 실수를 방지하고 보안성을 높였습니다.

## 오답 포인트

- A: 이메일 인증 코드 입력 방식 추가는 공식 문서에 명시된 내용이 아닙니다.
- C: SMS 문자 인증 방식 추가는 공식 문서에 명시된 내용이 아닙니다.
- D: 관리자 승인 요청 방식 추가는 공식 문서에 명시된 내용이 아닙니다.
