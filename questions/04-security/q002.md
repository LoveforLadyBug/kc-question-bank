## 문제

카카오클라우드 IAM **그룹(Group)**의 정책에 대한 설명으로 **옳지 않은** 것은?

## 보기

- A. 그룹 내에 다른 그룹을 중첩하여 포함할 수 없다.
- B. 그룹에는 사용자와 서비스 계정 모두 포함할 수 있다.
- C. 사용자는 여러 그룹에 동시에 속할 수 있다.
- D. 그룹에서 사용자가 제거되면 해당 그룹을 통해 발급된 API 토큰은 즉시 만료된다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, IAM 그룹에는 사용자만 포함할 수 있으며 서비스 계정은 그룹에 포함할 수 없습니다. 그룹 내 다른 그룹 포함(중첩)도 불가능합니다. 사용자는 여러 그룹에 동시에 속할 수 있으며, 그룹에서 제거되면 해당 그룹을 통해 발급된 토큰은 즉시 만료됩니다.

## 오답 포인트

- A: 그룹 내에 다른 그룹을 중첩하여 포함할 수 없다는 설명은 올바릅니다. IAM 그룹은 중첩을 지원하지 않습니다.
- C: 사용자는 여러 그룹에 동시에 속할 수 있다는 설명은 올바릅니다. 각 그룹마다 다른 역할을 가질 수 있습니다.
- D: 그룹에서 제거 시 해당 그룹을 통해 발급된 API 토큰이 즉시 만료된다는 설명은 올바릅니다.
