## 문제

카카오클라우드 **Cloud Trail** 이벤트에서 제공하는 **상세 정보**로 옳지 **않은** 것은?

## 보기

- A. 이벤트 발생 시간, 이벤트 이름
- B. 사용자 ID, 소스 IP
- C. 리소스 유형, 리소스 이름, 리소스 ID
- D. 이벤트 발생 당시의 CPU 및 메모리 사용량

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, Cloud Trail 이벤트에서 제공하는 상세 정보는 **이벤트 발생 시간, 이벤트 이름, 사용자 ID, 소스 IP, 리소스 유형, 리소스 이름, 리소스 ID 및 로그** 등입니다. **CPU 및 메모리 사용량**은 Cloud Trail 이벤트 정보에 포함되지 않으며, 이는 Monitoring 서비스에서 확인할 수 있습니다.

## 오답 포인트

- A: 이벤트 발생 시간과 이벤트 이름은 Cloud Trail에서 제공하는 상세 정보입니다.
- B: 사용자 ID와 소스 IP는 Cloud Trail에서 제공하는 상세 정보입니다.
- C: 리소스 유형, 이름, ID는 Cloud Trail에서 제공하는 상세 정보입니다.
