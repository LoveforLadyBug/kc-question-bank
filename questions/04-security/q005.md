## 문제

카카오클라우드 **Cloud Trail** 서비스에 대한 설명으로 옳은 것은?

## 보기

- A. Cloud Trail은 서버, 네트워크, 애플리케이션의 성능 지표를 수집하고 시각화하는 모니터링 서비스이다.
- B. Cloud Trail은 사용자의 활동을 자동으로 수집·기록하고 최근 90일간의 이벤트를 조회할 수 있는 서비스이다.
- C. Cloud Trail은 인스턴스의 로그를 실시간으로 분석하여 보안 위협을 차단하는 서비스이다.
- D. Cloud Trail은 별도의 요금이 부과되는 유료 서비스이다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Cloud Trail은 사용자의 활동(로그인, 리소스 생성/변경/삭제 등)을 자동으로 수집하고 기록하는 서비스입니다. 콘솔에서 프로젝트에서 발생한 최근 90일간의 이벤트를 조회할 수 있으며, 보안 정책 준수 감사 자료로 활용할 수 있습니다. Cloud Trail은 추가 비용 없이 무료로 제공됩니다.

## 오답 포인트

- A: 서버, 네트워크, 애플리케이션의 성능 지표를 수집하고 시각화하는 서비스는 Cloud Trail이 아닌 Monitoring 서비스입니다. Cloud Trail은 사용자 활동 추적에 초점을 맞춥니다.
- C: 실시간 보안 위협 차단은 Cloud Trail의 기능이 아닙니다. Cloud Trail은 이미 발생한 사용자 활동을 기록하고 조회하는 감사 서비스입니다.
- D: Cloud Trail은 추가 비용 없이 무료로 제공되는 서비스입니다.
