## 문제

카카오클라우드 IAM **서비스 에이전트 계정(Service Agent Account)**에 대한 설명으로 옳은 것은?

## 보기

- A. 서비스 에이전트 계정은 사용자가 직접 생성하여 자동화 작업에 활용하는 계정이다.
- B. 서비스 에이전트 계정은 카카오클라우드 서비스가 리소스를 조작할 때 자동으로 생성되는 시스템 전용 계정이다.
- C. 서비스 에이전트 계정은 사용자가 직접 삭제할 수 있으며, 삭제 후 재생성할 수 있다.
- D. 서비스 에이전트 계정은 IAM 사용자와 동일한 권한을 가지며, 콘솔 로그인도 가능하다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 서비스 에이전트 계정은 사용자가 직접 생성하지 않으며, 카카오클라우드 서비스가 리소스를 조작할 때 자동으로 생성되는 시스템 전용 계정입니다. 예를 들어 Kubernetes Engine에서 노드 풀을 추가하면 해당 요청을 수행하기 위해 전용 서비스 에이전트가 VM 인스턴스를 자동으로 생성합니다. 사용자가 직접 생성, 수정, 삭제할 수 없습니다.

## 오답 포인트

- A: 서비스 에이전트 계정은 사용자가 직접 생성하는 계정이 아닙니다. 사용자가 직접 생성하는 계정은 서비스 계정(Service Account)입니다. 서비스 에이전트 계정은 카카오클라우드 서비스가 자동으로 생성합니다.
- C: 서비스 에이전트 계정은 사용자가 직접 생성, 수정, 삭제할 수 없습니다. 단, 프로젝트를 삭제하면 해당 프로젝트의 모든 서비스 에이전트 계정도 함께 삭제됩니다.
- D: 서비스 에이전트 계정은 내부 전용 계정으로 콘솔 로그인 또는 API 호출 등 사용자가 직접 제어할 수 없습니다. IAM 사용자와 동일한 권한을 가지지 않습니다.
