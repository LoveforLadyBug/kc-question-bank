## 문제

카카오클라우드 Cloud Trail에서 **콘솔을 통해 조회 가능한 이벤트 보존 기간**으로 옳은 것은?

## 보기

- A. 최근 30일
- B. 최근 60일
- C. 최근 90일
- D. 제한 없이 영구 보존

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Cloud Trail 콘솔에서 프로젝트에서 발생한 최근 90일간의 이벤트를 조회할 수 있습니다. 이 기간의 이벤트를 통해 보안 정책 준수 여부를 확인하거나 권한 없는 사용자의 접근 여부 등을 감사 자료로 활용할 수 있습니다.

## 오답 포인트

- A: Cloud Trail의 이벤트 조회 가능 기간은 30일이 아닌 90일입니다.
- B: Cloud Trail의 이벤트 조회 가능 기간은 60일이 아닌 90일입니다.
- D: Cloud Trail 콘솔에서는 영구 보존이 아닌 최근 90일까지만 이벤트를 조회할 수 있습니다. 더 긴 기간의 보존이 필요한 경우 로그 저장 기능을 활용해야 합니다.
