## 문제

카카오클라우드 Cloud Trail에서 **프로젝트 이벤트**의 예시로 옳은 것은?

## 보기

- A. 프로젝트 생성, 프로젝트 삭제
- B. 빌링 조회, 결제 수단 변경
- C. 리소스 생성, 리소스 삭제
- D. 조직 멤버 초대, 조직 역할 변경

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 프로젝트 이벤트는 프로젝트 단위 서비스에서 발생한 이벤트로, 리소스 생성이나 리소스 삭제 등이 해당됩니다. 프로젝트 이벤트는 프로젝트 내 모든 사용자가 조회 가능합니다. 반면 프로젝트 생성/삭제, 빌링 조회 등은 조직 이벤트에 해당합니다.

## 오답 포인트

- A: 프로젝트 생성과 프로젝트 삭제는 조직 단위에서 발생하는 조직 이벤트입니다. 프로젝트 이벤트가 아닙니다.
- B: 빌링 조회는 조직 단위 서비스에서 발생하는 조직 이벤트입니다. 프로젝트 이벤트가 아닙니다.
- D: 조직 멤버 초대 및 조직 역할 변경은 조직 단위에서 발생하는 조직 이벤트입니다. 프로젝트 이벤트가 아닙니다.
