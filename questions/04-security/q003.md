## 문제

카카오클라우드 IAM **서비스 계정(Service Account)**에 대한 설명으로 옳은 것은?

## 보기

- A. 서비스 계정은 콘솔 로그인이 가능하며, 일반 사용자처럼 모든 UI 기능을 사용할 수 있다.
- B. 서비스 계정은 IAM 액세스 키와 S3 액세스 키를 각각 최대 2개까지 발급할 수 있으며, 프로젝트당 최대 100개 생성 가능하다.
- C. 서비스 계정에는 조직 역할도 부여할 수 있으며, 조직 단위의 작업을 자동화하는 데 사용된다.
- D. 서비스 계정은 IAM 사용자 목록에 포함되며, IAM 사용자 할당량에 영향을 준다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 서비스 계정은 콘솔 로그인 없이 인증 키를 사용해 API를 호출하는 비인터랙티브 계정입니다. IAM 액세스 키와 S3 액세스 키를 각각 최대 2개까지 발급할 수 있으며, 프로젝트당 최대 100개(비활성 포함)를 생성할 수 있습니다.

## 오답 포인트

- A: 서비스 계정은 콘솔 로그인이 불가능한 API 호출 전용 계정입니다. 콘솔 UI를 사용할 수 없습니다.
- C: 서비스 계정에는 프로젝트 단위 역할만 부여할 수 있습니다. 조직 역할은 서비스 계정에 적용되지 않습니다.
- D: 서비스 계정은 IAM 사용자 목록에 포함되지 않으며, IAM 사용자 할당량에도 영향을 주지 않습니다.
