## 문제

카카오클라우드 서비스 **릴리스 정책**의 버전 단계를 순서대로 나열한 것으로 옳은 것은?

## 보기

- A. Beta → Alpha → GA
- B. Alpha → Beta → GA
- C. GA → Beta → Alpha
- D. Alpha → GA → Beta

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 서비스 릴리스 정책의 버전 단계는 **Alpha → Beta → GA** 순서로 진행됩니다. Alpha(알파)는 POC 또는 테스트용 초기 버전으로 특정 고객에게만 제한적으로 공개되고, Beta(베타)는 기획/사업 요건에 일부만 충족되는 상태로 릴리즈되며, GA(General Availability)는 모든 요건을 완전히 충족한 최종 안정 버전입니다.

## 오답 포인트

- A: Beta → Alpha → GA는 순서가 틀립니다. Alpha가 가장 초기 단계입니다.
- C: GA → Beta → Alpha는 역순입니다.
- D: Alpha → GA → Beta는 Beta와 GA의 순서가 바뀌었습니다.
