## 문제

카카오클라우드 빠른 시작 가이드에서 안내하는 5단계 설정 순서로 올바른 것은?

## 보기

- A. 클라우드 계정 생성 → 사용자 등록 → 결제 수단 등록 → 프로젝트 생성 → VPC 생성
- B. 클라우드 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성
- C. 결제 수단 등록 → 클라우드 계정 생성 → 사용자 등록 → VPC 생성 → 프로젝트 생성
- D. 클라우드 계정 생성 → 프로젝트 생성 → 사용자 등록 → 결제 수단 등록 → VPC 생성

## 정답

B

## 해설

카카오클라우드 공식 문서의 빠른 시작 가이드에 따르면, 카카오클라우드 서비스를 시작하기 위한 5단계는 다음과 같은 순서로 진행됩니다. Step 1: 클라우드 계정 생성, Step 2: 결제 수단 등록, Step 3: 사용자 등록 및 조직 역할 추가, Step 4: 프로젝트 생성 및 프로젝트 역할 추가, Step 5: VPC 생성. 결제 수단 등록을 완료하지 않으면 다음 단계로 진행할 수 없습니다.

## 오답 포인트

- A: 결제 수단 등록은 사용자 등록보다 먼저 이루어집니다. 결제 수단을 등록하지 않으면 다음 단계로 진행할 수 없습니다.
- C: 클라우드 계정 생성이 먼저 이루어진 후에 결제 수단을 등록할 수 있습니다. 또한 VPC 생성은 프로젝트 생성 이후의 마지막 단계입니다.
- D: 프로젝트 생성은 사용자 등록 및 역할 추가 이후에 수행되며, 결제 수단 등록은 사용자 등록보다 앞선 단계입니다.
