## 문제

카카오클라우드 API를 사용하기 위해 발급하는 API 인증 토큰의 기본 만료 시간으로 올바른 것은?

## 보기

- A. 1시간
- B. 6시간
- C. 12시간
- D. 24시간

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰은 기본적으로 발급 후 12시간이 경과하면 만료됩니다. 단, 12시간 이내라도 프로젝트 역할 변경, IAM 액세스 키 삭제 등의 특정 상황에서는 조기에 변경되거나 만료될 수 있습니다.

## 오답 포인트

- A: 1시간은 일부 OAuth 서비스의 기본 만료 시간으로 혼동할 수 있지만, 카카오클라우드 API 인증 토큰의 기본 만료 시간은 12시간입니다.
- B: 6시간은 공식 문서에 언급되지 않는 임의의 시간입니다.
- D: 24시간은 기본 만료 시간보다 2배 긴 값으로, 보안상의 이유로 카카오클라우드는 더 짧은 12시간을 기본값으로 사용합니다.
