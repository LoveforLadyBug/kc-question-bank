## 문제

카카오클라우드의 모든 서비스가 기반으로 하는 권한 관리 방식은?

## 보기

- A. DAC(Discretionary Access Control, 임의적 접근 제어)
- B. MAC(Mandatory Access Control, 강제적 접근 제어)
- C. RBAC(Role Based Access Control, 역할 기반 접근 제어)
- D. ABAC(Attribute Based Access Control, 속성 기반 접근 제어)

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드의 모든 서비스는 역할 기반 접근 제어(RBAC, Role Based Access Control) 방식의 IAM(Identity and Access Management)을 기반으로 운영됩니다. 이를 통해 사용자에게 특정 역할을 부여하여 사용자/조직/프로젝트를 제어할 수 있는 권한을 설정합니다.

## 오답 포인트

- A: DAC는 자원 소유자가 직접 접근 권한을 결정하는 방식으로, 카카오클라우드가 채택한 방식이 아닙니다.
- B: MAC는 중앙에서 정해진 보안 정책에 따라 접근을 강제하는 방식으로, 주로 정부·군사 시스템에 사용됩니다.
- D: ABAC는 사용자·자원·환경의 속성을 조합해 접근을 결정하는 방식으로, 카카오클라우드 공식 문서에서 언급하는 방식이 아닙니다.
