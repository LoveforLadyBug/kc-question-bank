## 문제

카카오클라우드에서 IAM 액세스 키를 삭제했을 때 발생하는 결과로 옳은 것은?

## 보기

- A. 기존 발급된 API 인증 토큰은 만료 시간(12시간)이 지날 때까지 계속 유효하게 사용할 수 있다.
- B. IAM 액세스 키가 삭제되면 API 인증 토큰도 자동으로 만료되어 더 이상 사용할 수 없다.
- C. IAM 액세스 키 삭제 후 콘솔에서 신규 토큰을 발급받으면 기존 액세스 키 없이도 사용 가능하다.
- D. IAM 액세스 키 삭제는 API 인증 토큰에 영향을 주지 않으며, 토큰은 독립적으로 관리된다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, IAM 액세스 키를 삭제하면 해당 액세스 키를 기반으로 발급된 API 인증 토큰은 자동으로 만료되며 더 이상 사용할 수 없습니다. API 인증 토큰은 IAM 액세스 키를 기반으로 발급되므로, 액세스 키가 삭제되면 해당 토큰의 유효성도 즉시 종료됩니다. 따라서 새로운 API 인증 토큰을 사용하려면 새 IAM 액세스 키를 먼저 발급받아야 합니다.

## 오답 포인트

- A: IAM 액세스 키 삭제 시 API 인증 토큰은 즉시 만료됩니다. 기존 12시간의 만료 시간과 관계없이 액세스 키 삭제와 동시에 무효화됩니다.
- C: IAM 액세스 키 없이는 API 인증 토큰을 발급받을 수 없습니다. 토큰 발급을 위해서는 반드시 IAM 액세스 키(액세스 키 ID + 보안 액세스 키)가 필요합니다.
- D: API 인증 토큰은 IAM 액세스 키에 종속되어 있어, 액세스 키 삭제 시 함께 만료됩니다. 두 항목은 독립적으로 관리되지 않습니다.
