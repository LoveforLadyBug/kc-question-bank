## 문제

카카오클라우드 빠른 시작 가이드에서 **프로젝트 생성** 이전에 완료해야 하는 단계로 옳은 것은?

## 보기

- A. VPC 생성
- B. 사용자 등록 및 조직 역할 추가
- C. IAM 액세스 키 발급
- D. Kubernetes Engine 클러스터 생성

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 빠른 시작 가이드의 순서는 계정 생성 → 결제 수단 등록 → **사용자 등록 및 조직 역할 추가** → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성입니다. 따라서 프로젝트 생성 이전에 완료해야 하는 단계는 사용자 등록 및 조직 역할 추가입니다.

## 오답 포인트

- A: VPC 생성은 프로젝트 생성 이후의 단계입니다. 프로젝트 생성 전에 수행하는 단계가 아닙니다.
- C: IAM 액세스 키 발급은 빠른 시작 가이드의 기본 5단계에 포함되지 않으며, API 사용 준비를 위한 별도 작업입니다.
- D: Kubernetes Engine 클러스터 생성은 빠른 시작 가이드의 기본 5단계에 포함되지 않는 서비스 활용 단계입니다.
