## 문제

카카오클라우드에서 **IAM 보안 설정 변경 이력을 Cloud Trail로 추적**하고자 할 때, 2026년 3월 기능 강화로 새롭게 추적 가능해진 내용으로 옳은 것은?

## 보기

- A. VM 인스턴스의 CPU 사용률 변화 이력을 Cloud Trail에서 확인할 수 있게 되었다.
- B. 조직 및 사용자 보안 설정 변경 관련 신규 Cloud Trail 이벤트 17종이 추가되어 보안 관련 주요 변경 사항을 더 상세하게 추적할 수 있게 되었다.
- C. Cloud Trail에서 IAM 역할 부여 내역을 실시간 스트리밍으로 확인할 수 있게 되었다.
- D. 사용자의 API 호출 응답 본문(Response Body) 전체를 Cloud Trail에서 조회할 수 있게 되었다.

## 정답

B

## 해설

카카오클라우드 공식 문서(2026.03.24 기능 추가)에 따르면, **조직 및 사용자 보안 설정의 가시성을 확보하기 위해 신규 Cloud Trail 이벤트 17종을 추가**했습니다. 이를 통해 관리자는 비밀번호 변경, 세션 만료 옵션 설정, 자원 삭제 재인증 등 보안 관련 주요 변경 사항을 더욱 상세하게 추적하고 모니터링할 수 있게 되었습니다.

## 오답 포인트

- A: VM 인스턴스의 CPU 사용률은 Monitoring 서비스에서 확인하며, Cloud Trail과 관련이 없습니다.
- C: Cloud Trail은 실시간 스트리밍이 아닌 이벤트 로그 기록 방식으로 동작합니다.
- D: Cloud Trail은 API 호출의 응답 본문 전체를 저장하지 않으며, 주요 이벤트 메타데이터를 기록합니다.
