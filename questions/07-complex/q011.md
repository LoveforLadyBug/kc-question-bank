## 문제

카카오클라우드 빠른 시작 가이드에서 **VPC 생성**은 어느 단계에서 수행하는가?

## 보기

- A. 계정 생성 이전에 VPC를 먼저 생성한다.
- B. 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성 순으로 진행한다.
- C. 프로젝트 생성 전에 VPC를 먼저 생성해야 한다.
- D. VPC 생성은 빠른 시작 가이드에 포함되지 않는다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 빠른 시작 가이드의 단계는 다음과 같습니다: **계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성**. VPC 생성은 프로젝트 생성 후 마지막 단계에서 수행합니다.

## 오답 포인트

- A: VPC 생성은 계정 생성 이후의 마지막 단계입니다.
- C: VPC 생성은 프로젝트 생성 후에 수행합니다.
- D: VPC 생성은 빠른 시작 가이드의 마지막 단계로 명시적으로 포함됩니다.
