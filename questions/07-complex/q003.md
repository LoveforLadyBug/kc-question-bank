## 문제

카카오클라우드 API 인증 토큰 발급 후, 발급된 **토큰 값을 확인할 수 있는 위치**로 옳은 것은?

## 보기

- A. Response Body의 `token` 필드
- B. Response Body의 `access_token` 필드
- C. Response Header의 `X-Subject-Token` 항목
- D. Response Header의 `Authorization` 항목

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰 발급 요청에 대한 응답에서 발급된 토큰은 **Response Header의 `X-Subject-Token`** 항목에서 확인할 수 있습니다. 이는 방법 1(IAM 액세스 키 ID + 보안 액세스 키)과 방법 2(IAM 액세스 키 이름 + 보안 액세스 키 + 사용자 UUID) 모두에 공통으로 적용됩니다.

## 오답 포인트

- A: 토큰 값은 Response Body의 `token` 필드가 아닌 Response Header에서 확인합니다.
- B: 토큰 값은 Response Body의 `access_token` 필드가 아닌 Response Header의 `X-Subject-Token`에서 확인합니다.
- D: 발급된 토큰은 `Authorization` 헤더가 아닌 `X-Subject-Token` 헤더에서 확인합니다.
