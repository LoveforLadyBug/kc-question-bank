## 문제

카카오클라우드 API 인증 토큰 발급 **방법 2**에서 필요한 파라미터로 옳은 것은?

## 보기

- A. IAM 액세스 키 ID, 보안 액세스 키
- B. IAM 액세스 키 이름, 보안 액세스 키
- C. IAM 액세스 키 이름, 보안 액세스 키, 사용자 UUID
- D. IAM 액세스 키 ID, 보안 액세스 키, 사용자 UUID

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰 발급 방법은 두 가지입니다. 방법 1은 **IAM 액세스 키 ID**와 **보안 액세스 키**를 사용하고, 방법 2는 **IAM 액세스 키 이름(name)**, **보안 액세스 키**, **사용자 UUID** 세 가지를 사용합니다. 방법 2에서는 액세스 키 ID 대신 이름을 사용하며, 사용자를 식별하기 위한 UUID가 추가로 필요합니다.

## 오답 포인트

- A: IAM 액세스 키 ID와 보안 액세스 키는 방법 1에서 사용하는 파라미터입니다.
- B: 방법 2에서는 보안 액세스 키 외에 사용자 UUID도 추가로 필요합니다. 이름과 보안 액세스 키만으로는 불충분합니다.
- D: 방법 2에서는 IAM 액세스 키 ID가 아닌 IAM 액세스 키 이름(name)을 사용합니다. ID와 이름은 서로 다른 식별자입니다.
