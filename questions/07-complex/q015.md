## 문제

카카오클라우드 **3 Tier 아키텍처** 튜토리얼에서 외부 사용자의 트래픽을 **WEB 서버로 분산**하기 위해 사용하는 서비스로 옳은 것은?

## 보기

- A. Cloud Trail
- B. Beyond Networking Service(BNS)의 로드 밸런서(Load Balancer)
- C. Monitoring 서비스의 Alert Center
- D. Beyond Storage Service(BSS)의 Object Storage

## 정답

B

## 해설

카카오클라우드 **3 Tier 아키텍처**는 WEB, APP, DB 세 계층으로 구성됩니다. 외부 사용자의 트래픽을 WEB 서버로 분산하는 역할은 **BNS(Beyond Networking Service)의 로드 밸런서**가 담당합니다. 로드 밸런서는 리스너(Listener)로 트래픽을 수신하고, 타겟 그룹(Target Group)에 등록된 WEB 서버 인스턴스로 요청을 분산합니다. 이를 통해 고가용성과 트래픽 분산을 구현합니다.

## 오답 포인트

- A: Cloud Trail은 사용자 활동 및 API 호출 이벤트를 기록하는 감사 로그 서비스입니다.
- C: Alert Center는 알림 정책 설정 및 발송 관리 서비스로, 트래픽 분산과 무관합니다.
- D: Object Storage는 비정형 데이터 저장소로, 로드 밸런싱 기능을 제공하지 않습니다.
