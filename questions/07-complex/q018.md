## 문제

카카오클라우드 **서비스 계정의 S3 액세스 키**에 대한 설명으로 옳은 것은?

## 보기

- A. S3 액세스 키는 Kubernetes Engine API를 호출하기 위해 사용된다.
- B. S3 액세스 키는 Object Storage의 S3 API를 사용하기 위한 자격 증명이며, 서비스 계정당 최대 2개까지 발급할 수 있다.
- C. S3 액세스 키를 발급하면 IAM 액세스 키는 자동으로 비활성화된다.
- D. S3 액세스 키는 조직 단위 역할이 있는 사용자만 발급할 수 있다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **서비스 계정의 S3 액세스 키**는 **Object Storage의 S3 API를 사용하기 위한 자격 증명**입니다. 서비스 계정당 IAM 액세스 키와 S3 액세스 키 각각 **최대 2개**까지 발급이 가능합니다. S3 액세스 키 발급 기능은 2025년 4월에 서비스 계정 메뉴에 추가되었습니다.

## 오답 포인트

- A: S3 액세스 키는 Object Storage의 S3 API 전용이며, Kubernetes Engine API와는 관련이 없습니다.
- C: S3 액세스 키와 IAM 액세스 키는 독립적으로 발급 및 관리됩니다. 하나를 발급해도 다른 하나에 영향을 주지 않습니다.
- D: S3 액세스 키 발급은 조직 단위 역할 조건이 없습니다.
