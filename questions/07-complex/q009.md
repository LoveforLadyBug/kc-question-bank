## 문제

카카오클라우드 API 인증 토큰의 **권한이 즉시 만료(권한 만료)**되는 경우로 옳은 것을 모두 고른 것은?

## 보기

- A. 소속 프로젝트 역할이 관리자에서 멤버로 변경된 경우
- B. 프로젝트 역할이 완전히 삭제(프로젝트에서 내보내기)된 경우
- C. 사용자가 IAM 액세스 키를 직접 삭제한 경우
- D. B와 C 모두

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰의 권한이 즉시 만료되는 경우는 두 가지입니다. ① **프로젝트 역할이 삭제(프로젝트에서 내보내기)된 경우**, ② **사용자가 IAM 액세스 키를 직접 삭제한 경우**. 두 경우 모두 즉시 만료를 유발합니다. 반면, 프로젝트 역할이 변경(예: 관리자 → 멤버)된 경우는 즉시 만료가 아닌 권한 변경(낮아진 권한 상속)이 발생합니다.

## 오답 포인트

- A: 프로젝트 역할 변경(관리자 → 멤버)은 즉시 만료가 아닌 '권한 변경'에 해당합니다. 기존 토큰은 변경된 멤버 권한을 상속받아 계속 유효합니다.
- B만: 역할 삭제 시 토큰이 즉시 만료되는 것은 맞지만, 액세스 키 삭제 시에도 즉시 만료되므로 B만 선택하면 불완전합니다.
- C만: 액세스 키 삭제 시 토큰이 즉시 만료되는 것은 맞지만, 역할 삭제 시에도 즉시 만료되므로 C만 선택하면 불완전합니다.
