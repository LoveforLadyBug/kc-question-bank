## 문제

카카오클라우드 튜토리얼에서 제공하는 **3 Tier 아키텍처** 구성 요소로 옳은 것은?

## 보기

- A. WEB, Cache, DB
- B. WEB, APP, DB
- C. Frontend, Backend, Storage
- D. API Gateway, Microservice, Database

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드를 시작하는 가장 기본적인 시나리오로 **WEB, APP, DB**로 구성된 3 Tier 아키텍처를 구축하는 튜토리얼을 제공합니다. 이 3-tier 구조는 웹(프레젠테이션) 계층, 애플리케이션(비즈니스 로직) 계층, 데이터베이스(데이터) 계층으로 나뉘는 전통적인 클라우드 아키텍처 패턴입니다.

## 오답 포인트

- A: 3 Tier 아키텍처는 Cache가 아닌 APP(애플리케이션) 계층으로 구성됩니다.
- C: 카카오클라우드 공식 문서에서 제시하는 계층 명칭은 Frontend/Backend/Storage가 아닌 WEB/APP/DB입니다.
- D: API Gateway/Microservice/Database는 마이크로서비스 아키텍처 패턴으로, 카카오클라우드 기본 3 Tier 튜토리얼과는 다릅니다.
