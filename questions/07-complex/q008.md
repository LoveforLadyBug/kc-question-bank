## 문제

카카오클라우드 **API 인증 토큰 발급**의 선행 조건으로 옳은 것은?

## 보기

- A. VPC 생성이 완료되어 있어야 한다.
- B. IAM 액세스 키가 발급되어 있어야 한다.
- C. 프로젝트 관리자(Admin) 역할이 있어야 한다.
- D. Object Storage 버킷이 생성되어 있어야 한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰을 발급받으려면 **IAM 액세스 키 발급이 선행**되어야 합니다. IAM 액세스 키(IAM 액세스 키 ID와 보안 액세스 키)는 API 인증 토큰 발급을 위해 필요한 자격 증명 정보이며, IAM 액세스 키의 권한은 선택한 프로젝트의 IAM 역할에 따라 결정됩니다.

## 오답 포인트

- A: VPC 생성은 API 인증 토큰 발급의 선행 조건이 아닙니다. VPC는 네트워크 리소스 사용을 위해 필요하지만 토큰 발급 자체와는 무관합니다.
- C: 프로젝트 관리자 역할이 없어도 API 인증 토큰을 발급받을 수 있습니다. 프로젝트 멤버나 다른 역할을 가진 사용자도 IAM 액세스 키만 있으면 토큰을 발급받을 수 있습니다.
- D: Object Storage 버킷 생성은 API 인증 토큰 발급의 선행 조건이 아닙니다.
