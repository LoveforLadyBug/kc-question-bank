## 문제

카카오클라우드 API 인증 토큰 발급 요청 시, **Request Header에 반드시 포함해야 하는 항목**과 그 값으로 옳은 것은?

## 보기

- A. `Accept: application/json`
- B. `Content-Type: application/json`
- C. `Authorization: Bearer {token}`
- D. `X-Auth-Token: {access_key_id}`

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 2026년 4월 2일 22:00 이후부터 API 인증 토큰 발급 시 Request Header에 **`Content-Type: application/json`**을 반드시 설정해야 합니다. 이 항목이 포함되지 않은 요청은 인증 토큰이 발급되지 않습니다. `Content-Type`은 요청 본문(Body)에 담긴 데이터의 형식을 JSON으로 지정하는 역할을 합니다.

## 오답 포인트

- A: `Accept: application/json`은 응답 형식을 지정하는 헤더입니다. 토큰 발급 시 필수 요구 항목은 `Content-Type: application/json`입니다.
- C: `Authorization: Bearer {token}`은 이미 발급된 토큰을 사용하여 API를 호출할 때 사용하는 헤더입니다. 토큰 발급 요청 시에는 사용하지 않습니다.
- D: `X-Auth-Token`은 이미 발급된 API 인증 토큰으로 다른 API를 호출할 때 사용합니다.
