## 문제

카카오클라우드 **API 인증 토큰 발급** 요청 시 사용하는 **엔드포인트 URL**로 옳은 것은?

## 보기

- A. `https://api.kakaocloud.com/v3/auth/tokens`
- B. `https://iam.kakaocloud.com/identity/v3/auth/tokens`
- C. `https://iam.kakaoi.io/identity/v3/auth/tokens`
- D. `https://console.kakaocloud.com/api/tokens`

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰을 발급받기 위한 엔드포인트 URL은 **`https://iam.kakaocloud.com/identity/v3/auth/tokens`**입니다. HTTP 메서드는 POST를 사용하며, Request Header에 `Content-Type: application/json`을 반드시 포함해야 합니다. 기존 엔드포인트인 `https://iam.kakaoi.io`는 2024년 5월 이후 종료되었습니다.

## 오답 포인트

- A: `api.kakaocloud.com` 도메인은 IAM 인증 토큰 발급 엔드포인트가 아닙니다.
- C: `iam.kakaoi.io` 엔드포인트는 2024년 5월 1일 이후 종료된 기존 엔드포인트입니다.
- D: `console.kakaocloud.com`은 콘솔 UI 접속용 URL이며, API 토큰 발급에 사용하지 않습니다.
