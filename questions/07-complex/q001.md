## 문제

카카오클라우드 **빠른 시작 가이드(Quickstart)**의 단계 순서로 옳은 것은?

## 보기

- A. 계정 생성 → 프로젝트 생성 → 결제 수단 등록 → 사용자 등록 → VPC 생성
- B. 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성
- C. 계정 생성 → VPC 생성 → 결제 수단 등록 → 프로젝트 생성 → 사용자 등록
- D. 계정 생성 → 사용자 등록 → 결제 수단 등록 → VPC 생성 → 프로젝트 생성

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 빠른 시작 가이드의 단계는 다음 순서로 진행됩니다: ① 계정 생성 → ② 결제 수단 등록 → ③ 사용자 등록 및 조직 역할 추가 → ④ 프로젝트 생성 및 프로젝트 역할 추가 → ⑤ VPC 생성. 이 순서에 따라 기본 설정을 완료해야 카카오클라우드 서비스를 이용할 수 있습니다.

## 오답 포인트

- A: 결제 수단 등록은 프로젝트 생성보다 먼저 수행해야 합니다. 또한 사용자 등록 및 조직 역할 추가 단계가 누락되어 있습니다.
- C: VPC 생성은 마지막 단계입니다. 결제 수단 등록은 프로젝트 생성 전에 수행해야 합니다.
- D: 사용자 등록은 결제 수단 등록 이후에 수행하며, VPC 생성은 프로젝트 생성 이후에 수행합니다.
