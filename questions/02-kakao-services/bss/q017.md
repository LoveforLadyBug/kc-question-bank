## 문제

카카오클라우드 **Object Storage**의 **API 호환성**에 대한 설명으로 옳은 것은?

## 보기

- A. 자체 전용 API만 지원하며 타사 호환 API는 지원하지 않는다.
- B. AWS S3 호환 API를 지원한다.
- C. Google Cloud Storage API와 완전히 호환된다.
- D. REST API를 지원하지 않으며 SDK만 제공한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Object Storage는 **S3(Simple Storage Service) 호환 API를 지원**합니다. S3 API 관련 문서에서 CORS 관련 API(GetBucketCors, PutBucketCors, DeleteBucketCors 등)를 포함한 다양한 S3 API를 지원함을 확인할 수 있습니다. 이를 통해 기존 S3 기반 애플리케이션과의 호환이 가능합니다.

## 오답 포인트

- A: 카카오클라우드 Object Storage는 S3 호환 API를 지원합니다.
- C: Google Cloud Storage API와의 완전 호환은 공식 문서에 명시되지 않습니다.
- D: Object Storage는 REST API 기반의 S3 호환 API를 지원합니다.
