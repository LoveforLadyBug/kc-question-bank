## 문제

카카오클라우드 File Storage 서비스의 프로젝트 내 총 할당 가능 용량은?

## 보기

- A. 16TB
- B. 32TB
- C. 50TB
- D. 100TB

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage 서비스는 GA 버전으로 프로젝트 내 총 50TB까지 용량을 할당할 수 있습니다. 단일 인스턴스의 최대 용량은 16TB이며, 헬프데스크를 통해 증설 요청이 가능합니다.

## 오답 포인트

- A: 16TB는 단일 File Storage 인스턴스의 최대 크기(쿼터)입니다. 프로젝트 전체의 총 할당 가능 용량이 아닙니다.
- B: 32TB는 카카오클라우드 공식 문서에 언급되지 않는 수치입니다.
- D: 100TB는 실제 프로젝트 총 할당 가능 용량(50TB)의 2배이며, 공식 문서에서 언급되지 않는 수치입니다.
