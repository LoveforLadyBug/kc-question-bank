## 문제

카카오클라우드 File Storage의 Basic 인스턴스 유형에 대한 설명으로 옳은 것은?

## 보기

- A. 최소 100GB ~ 최대 1TB까지 할당 가능하며, 100GB 단위로 설정할 수 있다.
- B. 최소 1.0TB ~ 최대 16TB까지 할당 가능하며, 0.5TB 단위로 설정할 수 있다.
- C. 최소 1.0TB ~ 최대 50TB까지 할당 가능하며, 1TB 단위로 설정할 수 있다.
- D. 용량 제한 없이 자유롭게 설정할 수 있으며, 실제 사용량만큼 비용이 청구된다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, File Storage의 Basic 유형 인스턴스는 최소 1.0TB ~ 최대 16TB로 할당이 가능하며, 0.5TB 단위로 설정할 수 있습니다. Basic 유형은 가장 범용적인 사양의 인스턴스로 현재 File Storage에서 지원하는 유일한 유형입니다.

## 오답 포인트

- A: Basic 유형의 최소 용량은 100GB가 아닌 1.0TB이며, 최대 용량도 1TB가 아닌 16TB입니다. 단위도 100GB가 아닌 0.5TB입니다.
- C: 최대 50TB는 프로젝트 전체 File Storage의 최대 할당 용량이며, 단일 인스턴스의 최대 크기가 아닙니다. 단일 인스턴스 최대 용량은 16TB입니다.
- D: File Storage는 할당된 용량만큼 비용이 청구됩니다. 실제 사용량 기준이 아닌 할당 용량 기준으로 과금됩니다.
