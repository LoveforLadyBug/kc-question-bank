## 문제

카카오클라우드 **Object Storage**의 주요 활용 사례로 옳지 **않은** 것은?

## 보기

- A. 빅데이터, AI, IoT 데이터 저장 및 처리
- B. 백업 및 아카이빙 서비스
- C. 다수의 VM 인스턴스 간 실시간 파일 공유 및 협업
- D. 웹/모바일 애플리케이션 미디어 데이터 저장

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Object Storage의 주요 활용 사례는 **빅데이터, AI, IoT**, **백업 및 아카이빙**, **웹/모바일 애플리케이션** 등입니다. 반면 **다수의 서버 간 실시간 파일 공유 및 협업**은 **File Storage**의 주요 사용 목적입니다. Object Storage는 파일 공유 및 협업을 위한 별도의 파일 시스템이나 파일 공유 환경을 제공하지 않습니다.

## 오답 포인트

- A: 빅데이터, AI, IoT 데이터 저장은 Object Storage의 대표 활용 사례입니다.
- B: 백업 및 아카이빙은 Object Storage의 주요 활용 사례입니다.
- D: 웹/모바일 애플리케이션 미디어 저장은 Object Storage의 활용 사례입니다.
