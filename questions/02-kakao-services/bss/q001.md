## 문제

카카오클라우드 Object Storage에 대한 설명으로 옳은 것은?

## 보기

- A. 파일을 계층적 디렉터리 구조로 저장하며, 파일 공유 및 협업용으로 최적화된 서비스이다.
- B. 대용량 데이터를 객체(Key-value) 형태로 저장하며, 버킷 용량과 객체 수에 제한이 없다.
- C. 주로 데이터베이스와 같이 빠른 쓰기 성능이 필요한 워크로드에 특화된 블록 단위 스토리지이다.
- D. NFS 프로토콜을 사용하여 다수의 서버가 동시에 접근하는 공유 파일 시스템을 제공한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Object Storage는 대용량 데이터를 객체(Key-value) 형태로 저장 및 처리하는데 최적화된 확장성과 안정성을 갖춘 객체 기반 스토리지 서비스입니다. 버킷의 용량 및 객체 개수에 대한 제한이 없어 높은 확장성을 제공하며, 데이터는 여러 하드웨어에 분산되고 중복 저장되어 안정성을 보장합니다.

## 오답 포인트

- A: 계층적 디렉터리 구조와 파일 공유·협업에 특화된 서비스는 File Storage입니다. Object Storage는 실제로 Key-Value의 평면 구조로 데이터를 저장합니다.
- C: 데이터베이스와 같이 빠른 쓰기 성능이 필요한 블록 단위 스토리지는 Block Storage(볼륨 서비스)입니다. Object Storage는 대용량 비정형 데이터 저장에 적합합니다.
- D: NFS 프로토콜을 사용하는 공유 파일 시스템은 File Storage입니다. Object Storage는 파일 공유 환경을 별도로 제공하지 않습니다.
