## 문제

카카오클라우드 File Storage에서 제공하는 **백업** 기능에 대한 설명으로 옳은 것은?

## 보기

- A. Object Storage와 달리 File Storage는 백업 기능을 지원하지 않는다.
- B. File Storage는 Full 백업만 지원하며, Incremental 백업은 추후 지원 예정이다.
- C. File Storage는 Full 백업과 Incremental 백업을 모두 지원한다.
- D. File Storage 백업은 최대 10개까지만 생성할 수 있으며 증설이 불가하다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage는 안전한 데이터 보관을 위한 백업(Full/Incremental) 및 복원 기능을 지원합니다. 즉, Full 백업과 Incremental 백업 모두 지원됩니다. 또한 File Storage 백업 쿼터는 기본 100개이며, 헬프데스크를 통해 증설 요청이 가능합니다.

## 오답 포인트

- A: File Storage는 백업 기능을 지원합니다. Full/Incremental 백업 및 복원 기능이 모두 제공됩니다.
- B: File Storage는 Full 백업과 Incremental 백업을 모두 현재 지원합니다. Full만 지원한다는 설명은 잘못되었습니다.
- D: File Storage 백업의 기본 쿼터는 10개가 아닌 100개이며, 헬프데스크를 통해 증설 요청이 가능합니다.
