## 문제

카카오클라우드 Object Storage의 **버킷 라이프 사이클 정책**으로 수행할 수 있는 작업으로 옳은 것은?

## 보기

- A. 버킷 내 파일을 지정된 기간 후에 자동으로 삭제하거나 다른 유형의 버킷으로 이동시킬 수 있다.
- B. 버킷 내 파일의 중복을 자동으로 제거하여 저장 공간을 최적화한다.
- C. 파일의 액세스 패턴을 분석하여 자동으로 최적의 스토리지 클래스를 선택한다.
- D. 지정된 시간에 버킷 전체를 다른 리전으로 복제한다.

## 정답

A

## 해설

카카오클라우드 공식 문서에 따르면, Object Storage의 **버킷 라이프 사이클 정책**을 사용하면 **버킷 내의 파일을 지정된 기간 후에 자동으로 삭제**하거나 **다른 유형의 버킷으로 이동**시킬 수 있습니다. 이를 통해 데이터와 데이터 수명 주기를 효율적으로 관리하고 비용을 절감할 수 있습니다.

## 오답 포인트

- B: 중복 파일 자동 제거는 Object Storage 라이프 사이클 정책의 기능이 아닙니다.
- C: 액세스 패턴 분석을 통한 자동 스토리지 클래스 선택은 공식 문서에 명시된 기능이 아닙니다.
- D: 리전 간 자동 복제는 라이프 사이클 정책의 기능이 아닙니다.
