## 문제

카카오클라우드 File Storage가 원격 파일 공유와 분산 파일 시스템을 위해 사용하는 네트워크 프로토콜은?

## 보기

- A. SMB(Server Message Block)
- B. iSCSI(Internet Small Computer Systems Interface)
- C. NFS(Network File System)
- D. FTP(File Transfer Protocol)

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage는 원격 파일 공유와 분산 파일 시스템을 위한 네트워크 프로토콜인 NFS(Network File System) 프로토콜을 사용하여 다수의 서버와 애플리케이션 간에 데이터를 저장하고 공유합니다.

## 오답 포인트

- A: SMB(Server Message Block)는 주로 Windows 환경에서 파일 공유에 사용되는 프로토콜입니다. 카카오클라우드 File Storage는 NFS 프로토콜을 사용합니다.
- B: iSCSI는 블록 스토리지 접근에 사용되는 프로토콜입니다. File Storage의 파일 공유 프로토콜이 아닙니다.
- D: FTP(File Transfer Protocol)는 파일 전송 프로토콜이지만, 분산 파일 시스템이나 공유 파일 시스템 용도로 사용되지 않습니다. 카카오클라우드 File Storage는 NFS를 사용합니다.
