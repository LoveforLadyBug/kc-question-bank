## 문제

카카오클라우드 **File Storage** 서비스에서 **프로젝트 내 총 할당 가능한 용량**은?

## 보기

- A. 10TB
- B. 16TB
- C. 50TB
- D. 100TB

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage 서비스는 GA 버전으로 **프로젝트 내 총 50TB**까지 용량을 할당할 수 있습니다. 또한 개별 File Storage 인스턴스의 최대 크기는 16TB이며, 이는 쿼터 증설 요청이 가능합니다.

## 오답 포인트

- A: 10TB는 공식 문서에 명시된 프로젝트 총 할당 용량이 아닙니다.
- B: 16TB는 개별 File Storage 인스턴스(Instance 크기)의 최대값이며, 프로젝트 총량이 아닙니다.
- D: 100TB는 공식 문서에 명시된 수치가 아닙니다.
