## 문제

카카오클라우드 **File Storage**와 **Object Storage**의 차이점에 대한 설명으로 옳은 것은?

## 보기

- A. File Storage는 Key-value 형태의 객체를 저장하며, Object Storage는 계층적 파일 시스템을 제공한다.
- B. File Storage는 파일 공유·협업용으로 NFS 프로토콜을 사용하고, Object Storage는 대용량 비정형 데이터를 Key-value 형태로 저장한다.
- C. File Storage와 Object Storage는 모두 여러 사용자가 동시에 파일을 읽고 쓸 수 있는 파일 공유 환경을 제공한다.
- D. Object Storage는 Block Storage 대비 빠른 쓰기 성능을 제공하여 데이터베이스 워크로드에 적합하다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, File Storage는 NFS(Network File System) 프로토콜을 사용하여 다수의 서버와 애플리케이션 간에 데이터를 저장하고 공유하는 서비스로, 파일 공유 환경을 제공하는 공유 파일 시스템·협업용으로 사용됩니다. Object Storage는 주로 대용량 및 비정형 데이터를 Key-value 형태로 저장하는 서비스로, 파일 공유 환경은 별도로 제공하지 않습니다.

## 오답 포인트

- A: File Storage와 Object Storage의 특징이 뒤바뀐 오답입니다. Key-value는 Object Storage, 파일 시스템은 File Storage의 특징입니다.
- C: Object Storage는 파일 공유 및 협업을 위한 별도의 파일 공유 환경을 제공하지 않습니다. 공유 파일 시스템은 File Storage의 특징입니다.
- D: Object Storage는 데이터베이스와 같은 빠른 쓰기 성능이 필요한 워크로드가 아닌, 대용량 비정형 데이터 저장에 최적화된 서비스입니다.
