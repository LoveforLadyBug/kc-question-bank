## 문제

카카오클라우드 Object Storage에서 **2026년 3월 서비스 체계 표준화**로 변경된 내용으로 옳은 것은?

## 보기

- A. Cool Bucket이 Archive Storage Class로 변경되었다.
- B. Hot Bucket이 Standard Storage Class로 변경되었다.
- C. Standard Bucket이 Premium Storage Class로 변경되었다.
- D. Object Storage 서비스 전체가 Block Storage로 통합되었다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 2026년 3월 서비스 체계 표준화를 위해 Object Storage 서비스의 상품명 및 API 명칭이 변경되었습니다. **데이터 용량 상품명: Hot Bucket → Standard Storage Class**로 변경되었으며, API 상품명에는 기존 명칭 앞에 'Standard-' 접두사가 추가되었습니다(예: Standard-PUT, Standard-GET 등).

## 오답 포인트

- A: Cool Bucket의 변경 내용이 아닙니다. Hot Bucket이 Standard Storage Class로 변경되었습니다.
- C: Standard Bucket이 Premium Storage Class로 변경되었다는 내용은 없습니다.
- D: Object Storage가 Block Storage로 통합되었다는 내용은 없습니다.
