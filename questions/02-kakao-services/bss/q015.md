## 문제

카카오클라우드 **File Storage**에 연결할 수 있는 서비스 유형으로 옳은 것은?

## 보기

- A. VM(Virtual Machine)만 연결 가능하다.
- B. Kubernetes Engine만 연결 가능하다.
- C. 동일한 서브넷으로 접근이 가능한 경우 VM과 Kubernetes Engine 모두 연결 가능하다.
- D. 타 리전의 VM에서도 인터넷을 통해 직접 연결 가능하다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage는 **동일한 서브넷으로 접근이 가능한 경우**, **VM(Virtual Machine)뿐만 아니라 Kubernetes Engine과의 연결도 가능**합니다. 다양하게 구성된 환경에서의 연결을 지원하여 유연한 워크로드 구성이 가능합니다.

## 오답 포인트

- A: VM만 연결 가능한 것이 아니라 Kubernetes Engine도 연결 가능합니다.
- B: Kubernetes Engine만 지원하는 것이 아니라 VM도 지원합니다.
- D: 타 리전에서의 인터넷 직접 연결은 공식 문서에 명시된 기능이 아닙니다.
