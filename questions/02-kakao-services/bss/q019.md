## 문제

카카오클라우드 **File Storage** 인스턴스 하나의 **최대 크기**는?

## 보기

- A. 1TB
- B. 8TB
- C. 16TB
- D. 50TB

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, File Storage 인스턴스 크기의 기본 쿼터는 **16TB**입니다. 이 쿼터는 증설 요청이 가능합니다. 프로젝트 전체의 총 File Storage 용량은 50TB이지만, 개별 인스턴스의 최대 크기는 16TB입니다.

## 오답 포인트

- A: 1TB는 개별 File Storage 인스턴스의 최대 크기가 아닙니다.
- B: 8TB는 공식 문서에 명시된 인스턴스 최대 크기가 아닙니다.
- D: 50TB는 프로젝트 전체 총 할당 용량이며, 개별 인스턴스의 최대 크기가 아닙니다.
