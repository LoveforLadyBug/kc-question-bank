## 문제

카카오클라우드 Kubernetes Engine에 대한 설명으로 옳은 것은?

## 보기

- A. 사용자가 마스터 노드를 직접 구성하고 관리해야 하는 셀프 관리형 Kubernetes 서비스이다.
- B. VPC를 기반으로 손쉽게 Kubernetes를 실행할 수 있는 카카오클라우드의 관리형 Kubernetes 서비스이다.
- C. 컨테이너 이미지를 저장하고 관리하기 위한 비공개 이미지 저장소 서비스이다.
- D. 가상 머신을 사용하지 않고 물리 서버에서 직접 컨테이너를 실행하는 서비스이다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Kubernetes Engine은 VPC(Virtual Private Cloud)를 기반으로 손쉽게 Kubernetes를 실행할 수 있는 카카오클라우드의 관리형 Kubernetes 서비스입니다. 클러스터의 제어 영역(마스터 노드)은 카카오클라우드가 완전 관리하며, 사용자는 워커 노드와 애플리케이션에만 집중할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 관리형 서비스로, 사용자가 마스터 노드를 직접 구성하거나 관리할 필요가 없습니다. 제어 영역은 카카오클라우드가 별도의 VPC에서 완전 관리합니다.
- C: 컨테이너 이미지를 저장하고 관리하는 비공개 이미지 저장소는 Container Registry 서비스입니다. Kubernetes Engine과는 다른 서비스입니다.
- D: Kubernetes Engine의 노드는 Virtual Machine 서비스에서 제공하는 VM 인스턴스를 사용합니다. 물리 서버 직접 실행 서비스가 아닙니다.
