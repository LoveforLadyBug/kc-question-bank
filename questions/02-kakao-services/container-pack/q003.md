## 문제

카카오클라우드 Kubernetes Engine의 **노드 풀(Node Pool)**에 대한 설명으로 옳은 것은?

## 보기

- A. 하나의 노드 풀에는 서로 다른 인스턴스 유형의 노드를 혼합하여 배치할 수 있다.
- B. 노드 풀 내에서 특정 노드만 개별적으로 삭제할 수 있다.
- C. 하나의 노드 풀은 동일한 인스턴스 유형의 노드로 구성되며, 특정 노드만 삭제하는 것은 불가능하다.
- D. 노드 풀을 삭제해도 해당 노드 풀의 노드(인스턴스)는 자동으로 삭제되지 않는다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 노드 풀은 동일한 인스턴스 유형으로 구성된 노드(워커 노드)의 집합입니다. 노드 풀 내의 모든 노드는 동일한 인스턴스 유형을 사용하며, 노드 풀에서 특정 노드만 개별적으로 삭제하는 것은 불가능합니다. 노드를 줄이려면 노드 풀의 노드 수를 조정해야 합니다.

## 오답 포인트

- A: 하나의 노드 풀은 반드시 동일한 인스턴스 유형의 노드로만 구성됩니다. 서로 다른 인스턴스 유형을 혼합하려면 별도의 노드 풀을 생성해야 합니다.
- B: 노드 풀 내에서 특정 노드만 개별 삭제하는 것은 불가능합니다. 노드 수 조정은 노드 풀 단위로 이루어집니다.
- D: 노드 풀 삭제 시 해당 노드 풀의 노드(인스턴스)도 함께 삭제됩니다. 노드가 자동으로 삭제되지 않는다는 설명은 잘못되었습니다.
