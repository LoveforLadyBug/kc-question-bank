## 문제

카카오클라우드 Kubernetes Engine에서 **프로젝트 멤버** 역할로는 수행할 수 **없는** 작업은?

## 보기

- A. 클러스터 생성
- B. 노드 풀 추가
- C. 클러스터 삭제
- D. 프로젝트 멤버 관리

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, 프로젝트 멤버는 클러스터 생성, 노드 풀 추가, 클러스터 삭제 등 대부분의 Kubernetes Engine 작업을 수행할 수 있습니다. 그러나 프로젝트 멤버 관리(멤버 초대, 역할 변경, 멤버 삭제 등)는 프로젝트 관리자(Admin)만 수행할 수 있으며, 프로젝트 멤버에게는 허용되지 않습니다.

## 오답 포인트

- A: 클러스터 생성은 프로젝트 멤버도 수행 가능합니다. 클러스터 생성 권한은 프로젝트 관리자에게만 국한되지 않습니다.
- B: 노드 풀 추가는 프로젝트 멤버도 수행 가능합니다. 기존 클러스터에 노드 풀을 추가하는 작업도 멤버 권한으로 가능합니다.
- C: 클러스터 삭제는 프로젝트 멤버도 수행 가능합니다. 클러스터 삭제 권한도 프로젝트 관리자만의 전용 권한이 아닙니다.
