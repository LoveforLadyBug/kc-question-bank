## 문제

카카오클라우드 **Kubernetes Engine**에서 **Bare Metal Server 유형의 노드**에 대한 클러스터별 기본 쿼터는?

## 보기

- A. 1개
- B. 5개
- C. 10개
- D. 20개

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Kubernetes Engine에서 클러스터별 노드의 기본 쿼터는 100개이지만, **Bare Metal Server 유형의 노드는 클러스터별 5개**까지 생성 가능합니다. 일반 VM 노드와 BMS 노드의 쿼터가 다르게 적용됩니다.

## 오답 포인트

- A: 1개는 공식 문서에 명시된 BMS 노드 쿼터가 아닙니다.
- C: 10개는 프로젝트별 클러스터 쿼터입니다. BMS 노드 쿼터가 아닙니다.
- D: 20개는 클러스터별 노드 풀 쿼터입니다. BMS 노드 쿼터가 아닙니다.
