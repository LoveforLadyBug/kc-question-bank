## 문제

카카오클라우드 **Kubernetes Engine**의 기본 쿼터에서 **클러스터별 생성 가능한 최대 노드 풀(Node Pool) 수**는?

## 보기

- A. 5개
- B. 10개
- C. 20개
- D. 100개

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Kubernetes Engine의 기본 쿼터에서 **클러스터별 최대 노드 풀 수는 20개**입니다. 프로젝트별 클러스터는 최대 10개, 클러스터별 노드는 최대 100개이며, 단 Bare Metal Server 유형의 노드는 클러스터별 5개까지 생성 가능합니다. 모든 쿼터는 헬프데스크를 통해 증설 요청이 가능합니다.

## 오답 포인트

- A: 5개는 Bare Metal Server 유형의 노드 쿼터입니다. 노드 풀 쿼터가 아닙니다.
- B: 10개는 프로젝트별 클러스터 쿼터입니다. 노드 풀 쿼터가 아닙니다.
- D: 100개는 클러스터별 노드(Node) 쿼터입니다. 노드 풀 쿼터가 아닙니다.
