## 문제

카카오클라우드 **Container Registry**의 **이미지 이력 관리** 기능에 대한 설명으로 옳은 것은?

## 보기

- A. 이미지는 최신 버전만 유지되며 이전 버전은 자동으로 삭제된다.
- B. 태그를 사용하여 이미지 버전을 관리하고, 보존 정책을 통해 자동 삭제 및 보관 기간을 지정할 수 있다.
- C. 이미지 이력 관리는 외부 Git 저장소와만 연동하여 사용할 수 있다.
- D. 이미지 변경 이력은 30일 동안만 보관된다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Container Registry는 **태그(Tag)**를 사용하여 이미지 버전을 관리하며, 이미지의 변경 이력과 업데이트를 추적합니다. **이미지 이력 관리를 위한 보존 정책 설정**을 통해 자동 삭제 및 보관 기간을 지정할 수 있으며, 필요 시 변경 전 이미지 버전으로 복구가 가능합니다.

## 오답 포인트

- A: 이전 버전이 자동 삭제되는 것이 아닙니다. 보존 정책에 따라 관리할 수 있습니다.
- C: 외부 Git 저장소 연동만으로 사용하는 것이 아니라 자체 이력 관리 기능을 제공합니다.
- D: 30일 보관 기간 제한은 공식 문서에 명시된 내용이 아닙니다.
