## 문제

카카오클라우드 **Container Registry**의 **이미지 취약점 스캔** 기능에 대한 설명으로 옳은 것은?

## 보기

- A. 사용자가 수동으로 스캔을 요청해야 하며 자동 스캔은 지원하지 않는다.
- B. Trivy를 통해 이미지 업로드 또는 업데이트 시 취약점을 자동 스캔한다.
- C. 취약점 발견 시 해당 이미지를 자동으로 삭제한다.
- D. 오픈소스 취약점만 스캔하며 런타임 환경 검사는 지원하지 않는다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Container Registry는 **Trivy를 통한 이미지 취약점 스캔** 기능을 제공합니다. **이미지 업로드 또는 업데이트 시 취약점을 자동 스캔**하며, 패키지, 종속성, 런타임 환경을 검사합니다. 취약점 발견 시 경고 및 알림을 전송하여 신속한 조치가 가능하도록 합니다.

## 오답 포인트

- A: 이미지 업로드/업데이트 시 자동으로 스캔이 진행됩니다.
- C: 취약점 발견 시 자동 삭제가 아닌 경고 및 알림을 전송합니다.
- D: 런타임 환경 검사도 지원합니다.
