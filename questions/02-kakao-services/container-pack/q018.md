## 문제

카카오클라우드 **Kubernetes Engine**과 완벽하게 통합 및 연동되는 카카오클라우드 서비스로 옳은 것은?

## 보기

- A. Container Registry, Load Balancing, IAM, VPC
- B. Object Storage, File Storage, DNS, Transit Gateway
- C. BCS, Block Storage, Key Management Service
- D. CloudTrail, Config, Security Hub

## 정답

A

## 해설

카카오클라우드 공식 문서에 따르면, Kubernetes Engine은 애플리케이션의 확장성과 보안을 제공하기 위해 **Container Registry, Load Balancing, IAM, VPC 등** 다양한 카카오클라우드 서비스와 완벽하게 통합 및 연동되어 상호 시너지를 발휘할 수 있도록 설계되었습니다.

## 오답 포인트

- B: Object Storage, File Storage, DNS, Transit Gateway는 공식 문서에서 Kubernetes Engine과의 완벽 통합 서비스로 명시되지 않습니다.
- C: BCS, Block Storage, Key Management Service는 공식 문서에 명시된 통합 서비스 목록에 없습니다.
- D: CloudTrail, Config, Security Hub는 공식 문서에 명시된 통합 서비스가 아닙니다.
