## 문제

카카오클라우드 **DSRNLB(Direct Server Return Network Load Balancer)**를 선택해야 하는 상황으로 옳은 것은?

## 보기

- A. HTTP, HTTPS 트래픽을 처리하고 L7 규칙 기반의 세밀한 라우팅이 필요한 경우
- B. 클라이언트 IP 주소를 보존하여 백엔드 서버에 전달해야 하는 경우
- C. 대규모 트래픽에서 빠른 처리 속도가 필요하지만 클라이언트 IP는 필요 없는 경우
- D. SSL 인증서를 기반으로 암호화 및 복호화를 처리해야 하는 경우

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **DSRNLB(Direct Server Return NLB)**는 TCP, UDP 프로토콜을 지원하며 OSI L4로 동작합니다. **클라이언트 IP를 보존하여 전달해야 할 경우** 선택하는 로드 밸런서 유형입니다. NLB와 달리 Direct Server Return 방식으로 응답을 클라이언트에 직접 반환합니다.

## 오답 포인트

- A: HTTP/HTTPS 처리와 L7 세밀한 라우팅은 ALB(Application Load Balancer)의 특징입니다.
- C: 클라이언트 IP 없이 대규모 트래픽 처리만 필요한 경우는 NLB를 선택합니다.
- D: SSL 인증서 기반 암호화/복호화는 ALB 또는 NLB(TLS 지원)의 기능입니다.
