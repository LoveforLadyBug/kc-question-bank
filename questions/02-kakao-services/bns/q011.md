## 문제

카카오클라우드 **DNS(Domain Name System)** 서비스의 핵심 기능으로 옳은 것은?

## 보기

- A. 여러 서버에 트래픽을 분산하여 서비스 가용성을 높이는 기능
- B. 사람이 읽을 수 있는 도메인 이름을 머신이 읽을 수 있는 IP 주소로 변환하는 매핑 정보를 관리하는 기능
- C. VPC 내 서브넷 간 트래픽을 라우팅하는 기능
- D. 인스턴스의 퍼블릭 IP 주소를 자동으로 할당하는 기능

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **DNS(Domain Name System) 서비스**는 사람이 읽을 수 있는 도메인 이름을 머신이 읽을 수 있는 **IP 주소로 변환하는 매핑 정보를 관리**하여 컴퓨터가 서로 통신할 수 있도록 하는 서비스입니다. 사용자는 별도의 DNS 서버 운영 없이 DNS 영역과 레코드를 생성할 수 있습니다.

## 오답 포인트

- A: 트래픽 분산은 Load Balancing 서비스의 기능입니다.
- C: 서브넷 간 트래픽 라우팅은 라우팅 테이블의 역할입니다.
- D: 퍼블릭 IP 자동 할당은 퍼블릭 IP 서비스의 기능입니다.
