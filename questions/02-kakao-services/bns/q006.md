## 문제

카카오클라우드 VPC에서 **보안 그룹 생성** 권한이 있는 역할과, **VPC 생성** 권한이 있는 역할을 올바르게 짝지은 것은?

## 보기

- A. 보안 그룹 생성: 프로젝트 관리자만 / VPC 생성: 프로젝트 관리자, 프로젝트 멤버
- B. 보안 그룹 생성: 프로젝트 관리자, 프로젝트 멤버 / VPC 생성: 프로젝트 관리자만
- C. 보안 그룹 생성: 프로젝트 관리자, 프로젝트 멤버, 프로젝트 리더 / VPC 생성: 프로젝트 관리자만
- D. 보안 그룹 생성: 프로젝트 관리자만 / VPC 생성: 프로젝트 관리자만

## 정답

B

## 해설

카카오클라우드 공식 문서의 IAM 역할 관리 표에 따르면, VPC 생성은 프로젝트 관리자(Admin)만 수행할 수 있습니다. 반면 보안 그룹 생성은 프로젝트 관리자(Admin)와 프로젝트 멤버(Member) 모두 가능합니다. 보안 그룹 조회는 세 역할 모두 가능하지만, 생성·변경·삭제는 관리자와 멤버만 가능합니다.

## 오답 포인트

- A: VPC 생성은 관리자만 가능하나, 보안 그룹 생성도 관리자만 가능하다는 것은 잘못된 설명입니다. 보안 그룹은 프로젝트 멤버도 생성할 수 있습니다.
- C: 프로젝트 리더(Reader)는 보안 그룹 조회만 가능하며 생성 권한이 없습니다.
- D: VPC 생성은 관리자만 가능하지만, 보안 그룹 생성은 프로젝트 멤버도 할 수 있습니다. 두 항목 모두 관리자만 가능하다는 설명은 잘못되었습니다.
