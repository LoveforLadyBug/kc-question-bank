## 문제

카카오클라우드에서 VPC를 생성할 때 **자동으로 함께 생성되는** 리소스로 옳은 것은?

## 보기

- A. 기본 서브넷, 보안 그룹, 퍼블릭 IP
- B. 기본 서브넷, 인터넷 게이트웨이, 기본 라우팅 테이블
- C. 서브넷, NAT 게이트웨이, 보안 그룹
- D. 인터넷 게이트웨이, 로드 밸런서, 기본 라우팅 테이블

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드에서 생성되는 모든 VPC는 생성 시 기본 서브넷, 연결된 인터넷 게이트웨이, 모든 트래픽을 인터넷 게이트웨이로 보내는 기본 라우팅 테이블의 라우팅까지 함께 생성됩니다. 가용 영역에 생성된 기본 서브넷에 BCS 인스턴스를 생성하고 퍼블릭 IP를 연결하면 바로 인터넷에 액세스할 수 있습니다.

## 오답 포인트

- A: 퍼블릭 IP는 VPC 생성 시 자동으로 생성되지 않습니다. 퍼블릭 IP는 별도로 생성하여 네트워크 인터페이스에 연결해야 합니다.
- C: NAT 게이트웨이는 VPC 생성 시 자동으로 생성되지 않습니다. 프라이빗 서브넷에서 인터넷에 액세스하려면 NAT 인스턴스를 별도로 구성해야 합니다.
- D: 로드 밸런서는 VPC 생성 시 자동으로 생성되지 않습니다. 트래픽 분산이 필요한 경우 Load Balancing 서비스를 별도로 사용해야 합니다.
