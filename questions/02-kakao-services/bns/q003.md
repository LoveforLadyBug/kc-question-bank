## 문제

카카오클라우드 서브넷의 CIDR 블록에서 **예약되어 BCS 인스턴스 등의 리소스에 할당할 수 없는** IP 주소의 개수는?

## 보기

- A. 3개
- B. 5개
- C. 7개
- D. 10개

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 서브넷의 CIDR 블록에서 첫 6개의 IP 주소와 마지막 IP 주소, 총 7개의 주소는 예약되어 있으며 리소스에 할당할 수 없습니다. 예를 들어 10.0.0.0/24인 경우, 네트워크 주소(10.0.0.0), 기본 게이트웨이(10.0.0.1), 향후 사용 예약(10.0.0.2, 10.0.0.3), DHCP 용도(10.0.0.4, 10.0.0.5), 네트워크 브로드캐스트(10.0.0.255)가 예약됩니다.

## 오답 포인트

- A: 3개는 예약 IP 개수로 부족합니다. 실제로는 첫 번째부터 여섯 번째까지 6개와 마지막 1개, 총 7개가 예약됩니다.
- B: 5개는 예약 IP 개수보다 적습니다. 공식 문서에는 총 7개(첫 6개 + 마지막 1개)가 예약된다고 명시되어 있습니다.
- D: 10개는 실제 예약 IP 개수(7개)보다 많습니다. 예약 IP 개수를 과대 평가한 오답입니다.
