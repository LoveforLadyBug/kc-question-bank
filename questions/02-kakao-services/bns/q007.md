## 문제

카카오클라우드가 제공하는 로드 밸런서(Load Balancer) 유형을 모두 고른 것은?

## 보기

- A. ALB(Application Load Balancer), NLB(Network Load Balancer)
- B. ALB(Application Load Balancer), NLB(Network Load Balancer), DSRNLB(Direct Server Return Network Load Balancer)
- C. ALB(Application Load Balancer), NLB(Network Load Balancer), GLB(Global Load Balancer)
- D. NLB(Network Load Balancer), DSRNLB(Direct Server Return Network Load Balancer), ILB(Internal Load Balancer)

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드는 Application Load Balancer(ALB), Network Load Balancer(NLB), Direct Server Return Network Load Balancer(DSRNLB)의 3가지 로드 밸런서 유형을 제공합니다. ALB는 HTTP/HTTPS를 지원하는 OSI L7 로드 밸런서, NLB는 TCP/UDP/TLS를 지원하는 OSI L4 로드 밸런서, DSRNLB는 클라이언트 IP를 보존해야 할 경우 사용하는 OSI L4 로드 밸런서입니다.

## 오답 포인트

- A: 2가지 유형만 제시하고 있어 DSRNLB가 누락되었습니다. 카카오클라우드는 3가지 유형을 제공합니다.
- C: GLB(Global Load Balancer)는 카카오클라우드가 제공하는 로드 밸런서 유형에 포함되지 않습니다.
- D: ILB(Internal Load Balancer)는 카카오클라우드가 제공하는 로드 밸런서 유형에 포함되지 않습니다. ALB가 누락되어 있습니다.
