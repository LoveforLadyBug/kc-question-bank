## 문제

카카오클라우드 BCS VM의 **CPU 옵션 최적화** 기능에 대한 설명으로 옳은 것은?

## 보기

- A. CPU 코어 수를 실시간으로 자동 조정하여 비용을 최소화한다.
- B. 멀티스레딩을 비활성화하여 CPU 코어당 단일 스레드를 지정할 수 있다.
- C. GPU와 CPU를 병렬로 연결하여 연산 속도를 높인다.
- D. 인스턴스 생성 후에는 CPU 옵션을 변경할 수 없다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, BCS VM은 **고성능 컴퓨팅(HPC) 워크로드와 같은 특정 워크로드를 위한 멀티스레딩 비활성화 기능**을 제공합니다. **CPU 코어당 단일 스레드를 지정**하여 성능을 최적화할 수 있으며, 고객의 워크로드 요구사항에 가장 적합한 CPU 옵션으로 최적화가 가능합니다.

## 오답 포인트

- A: 실시간 자동 CPU 조정 기능은 공식 문서에 명시된 내용이 아닙니다.
- C: GPU와 CPU 병렬 연결은 CPU 옵션 최적화 기능의 설명이 아닙니다.
- D: CPU 옵션 변경 가능 여부에 대한 제한은 공식 문서에서 이렇게 명시하지 않습니다.
