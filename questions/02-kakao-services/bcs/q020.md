## 문제

카카오클라우드 BCS Virtual Machine의 **스토리지 연동** 특징에 대한 설명으로 옳은 것은?

## 보기

- A. 오직 Object Storage만 연동할 수 있다.
- B. Block Storage와 Object Storage 등 스토리지 서비스와 연동이 용이하며, Block Storage는 IOPS 요구 수준에 맞춰 설정할 수 있다.
- C. 스토리지는 인스턴스 생성 시에만 추가할 수 있으며, 이후에는 변경이 불가하다.
- D. Block Storage 연동은 지원하지만, 스냅샷 기능은 제공되지 않는다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, BCS Virtual Machine은 **Block Storage, Object Storage 등 스토리지 서비스와 연동이 용이**합니다. 또한 **Block Storage를 IOPS 요구 수준에 맞춰 설정**할 수 있으며, 안정적이고 편리한 서버 운영을 위한 **스냅샷 일정링 기능**도 제공합니다.

## 오답 포인트

- A: BCS VM은 Object Storage뿐만 아니라 Block Storage도 연동할 수 있습니다.
- C: 스토리지 연동은 생성 후에도 유연하게 추가·삭제할 수 있습니다.
- D: 카카오클라우드 공식 문서에 따르면 스냅샷 일정링 기능을 제공합니다.
