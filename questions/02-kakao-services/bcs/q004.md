## 문제

카카오클라우드 Virtual Machine 서비스에서 제공하는 이미지 유형에 대한 설명으로 옳은 것은?

## 보기

- A. 기본 이미지는 특정 프로젝트 전용으로만 사용 가능하며, 모든 사용자가 접근할 수 없다.
- B. 커스텀 이미지는 카카오클라우드에서 기본적으로 제공하는 이미지로 Linux 및 Windows를 지원한다.
- C. 기본 이미지는 카카오클라우드에서 제공하며 모든 사용자가 접근 가능하고, 커스텀 이미지는 사용자가 직접 생성한 이미지이다.
- D. Bare Metal Server와 달리 Virtual Machine은 커스텀 이미지 생성을 지원하지 않는다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, VM 서비스는 두 가지 이미지 유형을 제공합니다. 기본 이미지는 카카오클라우드에서 기본적으로 제공하는 이미지로 Linux 및 Windows 이미지를 포함하며, 기본적으로 모든 사용자가 접근할 수 있습니다. 커스텀 이미지는 사용자가 생성한 이미지로, 특정 프로젝트 또는 워크로드에 맞게 구성할 수 있습니다.

## 오답 포인트

- A: 기본 이미지는 카카오클라우드에서 제공하며 기본적으로 모든 사용자가 접근할 수 있습니다. 특정 프로젝트 전용이 아닙니다.
- B: 커스텀 이미지는 카카오클라우드에서 기본 제공하는 이미지가 아니라 사용자가 직접 생성하는 이미지입니다.
- D: Virtual Machine 서비스는 커스텀 이미지 생성을 지원합니다. 오히려 Bare Metal Server가 사용자 이미지 생성·관리를 지원하지 않으며 기본 이미지만 사용 가능합니다.
