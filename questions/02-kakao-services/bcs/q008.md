## 문제

카카오클라우드 VM의 퍼블릭 IP에 대한 설명으로 옳은 것은?

## 보기

- A. 퍼블릭 IP는 인스턴스 생성 시 자동으로 할당되며 별도 비용 없이 무료로 제공된다.
- B. 퍼블릭 IP를 할당받은 후 인스턴스에 연결하지 않아도 할당받는 동안 요금이 부과된다.
- C. 인스턴스가 Stopped 상태가 되면 퍼블릭 IP 연결이 자동으로 해제된다.
- D. 하나의 퍼블릭 IP는 특정 네트워크 인터페이스에 영구적으로 고정되어 다른 인터페이스로 이동할 수 없다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 퍼블릭 IP 주소는 사용자가 해제할 때까지 연결이 유지되며, 퍼블릭 IP 주소를 할당받은 동안 지속해서 요금이 부과됩니다. 즉, 인스턴스에 연결 여부와 관계없이 할당받은 시점부터 과금이 시작됩니다.

## 오답 포인트

- A: 퍼블릭 IP는 자동 할당이 아니며, 퍼블릭 IP 생성을 통해 IP 주소를 할당받은 뒤 네트워크 인터페이스에 연결해야 합니다. 또한 할당받는 동안 요금이 부과됩니다.
- C: 퍼블릭 IP 연결은 인스턴스가 중지(Stopped)되거나 종료(Shelved_offloaded)될 때까지만 유지된다고 문서에는 기재되어 있습니다. 그러나 할당받은 퍼블릭 IP 자체는 사용자가 직접 해제하기 전까지 유지되며 계속 과금됩니다.
- D: 할당받은 퍼블릭 IP는 이미 연결된 네트워크 인터페이스에서 연결을 해제한 후 다른 네트워크 인터페이스에 연결하여 재사용할 수 있습니다. 영구 고정이 아닙니다.
