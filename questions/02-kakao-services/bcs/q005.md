## 문제

카카오클라우드 Virtual Machine에서 커스텀 이미지 생성 시 안정적인 생성을 위해 권장되는 루트 볼륨 크기는?

## 보기

- A. 500GB 이하
- B. 1TB 이하
- C. 2TB 이하
- D. 제한 없음

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 커스텀 이미지 생성 시 연결된 루트 볼륨의 크기가 큰 경우 이미지 생성 작업이 실패할 수 있습니다. 안정적인 생성을 위해 루트 볼륨 크기는 1TB(=1,024GB) 이하로 구성하는 것을 권장합니다.

## 오답 포인트

- A: 500GB는 공식 권장 크기보다 작은 값입니다. 공식 문서에서는 1TB 이하를 권장합니다.
- C: 2TB는 공식 권장 크기(1TB)를 초과합니다. Windows 볼륨의 최대 크기(2,048GB)와 혼동할 수 있으나, 커스텀 이미지 생성 권장 크기는 1TB 이하입니다.
- D: 루트 볼륨 크기 제한이 없는 것이 아닙니다. 크기가 너무 클 경우 이미지 생성 작업이 실패할 수 있으므로 반드시 권장 크기를 확인해야 합니다.
