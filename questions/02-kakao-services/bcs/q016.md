## 문제

카카오클라우드 BCS에서 **보안 그룹(Security Group)**에 대한 설명으로 옳은 것은?

## 보기

- A. 인스턴스 간 트래픽을 암호화하는 서비스이다.
- B. IP 주소와 포트 번호를 기반으로 인스턴스로 전송된 패킷을 필터링한다.
- C. VPC와 외부 인터넷 사이의 게이트웨이 역할을 한다.
- D. 인스턴스의 CPU 사용량을 모니터링하고 자동으로 스케일링한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, BCS의 **보안 그룹**은 네트워크에 배포된 인스턴스의 보안을 강화하기 위해 **IP 주소와 포트 번호를 기반으로 인스턴스로 전송된 패킷을 필터링**하는 기능을 제공합니다. 이를 통해 강력한 보안 환경을 구성할 수 있습니다.

## 오답 포인트

- A: 트래픽 암호화는 보안 그룹의 기능이 아닙니다. 보안 그룹은 패킷 필터링에 집중합니다.
- C: VPC와 외부 인터넷 사이의 게이트웨이는 퍼블릭 IP 또는 NAT Gateway의 역할입니다.
- D: CPU 모니터링과 자동 스케일링은 모니터링 서비스나 Auto Scaling 기능의 역할입니다.
