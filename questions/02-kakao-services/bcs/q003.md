## 문제

카카오클라우드 BCS 인스턴스 명명 규칙에 따를 때, `c2a.xlarge` 인스턴스에 대한 설명으로 옳은 것은?

## 보기

- A. 메모리 최적화 유형의 2세대 Intel 프로세서 기반 엑스트라 라지 인스턴스
- B. 컴퓨팅 최적화 유형의 2세대 AMD 프로세서 기반 엑스트라 라지 인스턴스
- C. 범용 유형의 2세대 AMD 프로세서 기반 엑스트라 라지 인스턴스
- D. 가속 컴퓨팅 유형의 2세대 AMD 프로세서 기반 FPGA 최적화 엑스트라 라지 인스턴스

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, BCS 인스턴스 명명 규칙은 인스턴스 패밀리, 세대, 프로세서, 옵션, 크기를 조합합니다. c2a.xlarge를 분석하면 다음과 같습니다. 'c'는 컴퓨팅 최적화(Compute optimized) 패밀리, '2'는 2세대, 'a'는 AMD 프로세서, '.xlarge'는 인스턴스 크기를 의미합니다. 따라서 c2a.xlarge는 컴퓨팅 최적화 유형의 2세대 AMD 프로세서 기반 엑스트라 라지 인스턴스입니다.

## 오답 포인트

- A: 메모리 최적화 인스턴스 패밀리는 'r'로 시작합니다(예: r2a). 'c'는 컴퓨팅 최적화를 의미합니다.
- C: 범용 인스턴스 패밀리는 'm'으로 시작합니다(예: m2a, m3az). 'c'는 컴퓨팅 최적화를 의미합니다.
- D: 가속 컴퓨팅 중 FPGA 관련 인스턴스는 'gf'로 시작합니다(예: gf1i). 'c'는 컴퓨팅 최적화 패밀리를 의미합니다.
