## 문제

카카오클라우드 **MySQL** 서비스의 **실시간 모니터링**으로 확인할 수 있는 메트릭으로 옳은 것은?

## 보기

- A. 네트워크 패킷 손실률과 DNS 응답 시간만 확인 가능하다.
- B. CPU, Memory 사용량 등 시스템 상태부터 커넥션 수, 평균 응답 시간 등의 데이터베이스 메트릭까지 확인 가능하다.
- C. 디스크 I/O 메트릭만 제공한다.
- D. 실시간 모니터링 기능은 제공하지 않으며 일별 보고서만 지원한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, MySQL 서비스는 **CPU, Memory 사용량 등 시스템 상태**부터 **커넥션 수, 평균 응답 시간**과 같은 데이터베이스 메트릭까지 모니터링할 수 있는 메트릭을 제공합니다. 이를 통해 시스템과 데이터베이스 상태를 종합적으로 파악할 수 있습니다.

## 오답 포인트

- A: 네트워크 패킷 손실률과 DNS 응답 시간만 제공한다는 것은 공식 문서의 내용이 아닙니다.
- C: 디스크 I/O만 제공한다는 것은 공식 문서의 내용과 다릅니다. CPU, Memory, 커넥션 수 등 다양한 메트릭을 제공합니다.
- D: 실시간 모니터링 기능을 명시적으로 지원합니다.
