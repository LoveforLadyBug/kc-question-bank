## 문제

카카오클라우드 **MySQL** 서비스의 **고가용성 구현 방식**에 대한 설명으로 옳은 것은?

## 보기

- A. 여러 리전에 걸쳐 데이터를 동기화하는 멀티 리전 방식으로 고가용성을 구현한다.
- B. 메인 서버에서 장애가 발생하더라도 동기화된 대체 서버가 즉시 가동되는 Active-Standby 방식으로 서비스 지연을 최소화한다.
- C. 모든 서버가 동시에 쓰기 요청을 처리하는 Active-Active 방식으로 운영된다.
- D. 장애 발생 시 사용자가 수동으로 대체 서버를 가동해야 한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, MySQL 서비스는 **메인 서버에서 장애가 발생하더라도 동기화된 대체 서버가 즉시 가동**되며, **Active-Standby 방식으로 서비스 지연을 최소화**합니다. 복잡한 데이터베이스 이중화를 몇 번의 클릭만으로 구현할 수 있습니다.

## 오답 포인트

- A: 멀티 리전 동기화 방식은 공식 문서에 명시된 MySQL의 고가용성 구현 방식이 아닙니다.
- C: Active-Active 방식은 공식 문서에 명시된 MySQL의 구현 방식이 아닙니다.
- D: 대체 서버가 자동으로 즉시 가동되므로 수동 개입이 필요하지 않습니다.
