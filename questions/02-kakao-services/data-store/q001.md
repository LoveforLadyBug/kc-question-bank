## 문제

카카오클라우드 Data Store에 포함된 서비스를 모두 올바르게 나열한 것은?

## 보기

- A. MySQL, PostgreSQL, Redis, MongoDB
- B. MySQL, PostgreSQL, MemStore
- C. MySQL, MariaDB, MemStore
- D. MySQL, PostgreSQL, Cassandra, MemStore

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Data Store는 클라우드 기반 완전 관리형 데이터베이스 서비스 그룹으로 MySQL, PostgreSQL, MemStore 세 가지 서비스로 구성됩니다. MySQL과 PostgreSQL은 관계형 데이터베이스(RDBMS/ORDBMS)이며, MemStore는 Redis 기반의 완전 관리형 인 메모리 데이터 스토어 서비스입니다.

## 오답 포인트

- A: MongoDB는 카카오클라우드 Data Store에 포함되지 않습니다. Data Store는 MySQL, PostgreSQL, MemStore로 구성됩니다.
- C: MariaDB는 카카오클라우드 Data Store에 포함되지 않습니다. 관계형 데이터베이스 서비스로는 MySQL과 PostgreSQL만 제공됩니다.
- D: Cassandra는 카카오클라우드 Data Store에 포함되지 않습니다. Data Store의 구성 서비스는 MySQL, PostgreSQL, MemStore 세 가지입니다.
