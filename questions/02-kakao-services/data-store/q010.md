## 문제

카카오클라우드 MemStore에서 **클러스터 모드 사용 여부**에 따른 엔드포인트 제공 방식으로 옳은 것은?

## 보기

- A. 클러스터 모드 사용/미사용 모두 Primary 엔드포인트와 Read 엔드포인트를 동일하게 제공한다.
- B. 클러스터 모드 사용 시 Primary 엔드포인트와 Read 엔드포인트를 제공하고, 미사용 시 각 노드의 엔드포인트만 제공한다.
- C. 클러스터 모드 사용 시 Read 엔드포인트를 제공하지 않고 각 노드의 엔드포인트를 제공하며, 미사용 시 Primary 엔드포인트와 Read 엔드포인트를 제공한다.
- D. 클러스터 모드 사용/미사용 모두 각 노드의 엔드포인트만 제공하며, 통합 엔드포인트는 제공하지 않는다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 클러스터 모드를 사용하지 않을 경우 읽기/쓰기가 가능한 Primary 엔드포인트와 읽기 전용인 Read 엔드포인트가 제공됩니다. 반면 클러스터 모드를 사용할 경우 읽기 전용 Read 엔드포인트는 제공되지 않으며, 각 노드의 개별 엔드포인트가 제공됩니다.

## 오답 포인트

- A: 클러스터 모드 사용 여부에 따라 제공되는 엔드포인트가 다릅니다. 동일하게 제공된다는 설명은 잘못되었습니다.
- B: 클러스터 모드 사용/미사용의 설명이 반대로 되어 있습니다. 클러스터 모드 사용 시 Read 엔드포인트가 제공되지 않으며, 미사용 시 Primary와 Read 엔드포인트가 함께 제공됩니다.
- D: 클러스터 모드를 사용하지 않을 때는 각 노드의 개별 엔드포인트가 아닌 Primary 엔드포인트와 Read 엔드포인트가 제공됩니다. 각 노드 엔드포인트는 클러스터 모드 사용 시에 제공됩니다.
