## 문제

카카오클라우드 **PostgreSQL**이 MySQL보다 적합한 사용 사례로 옳은 것은?

## 보기

- A. 스타트업의 초기 서비스 프로토타입 개발
- B. 위치 기반 서비스, 금융 시스템, 의료 정보 시스템 등 고도화된 데이터 처리
- C. IoT 디바이스의 단순 데이터 수집 및 저장
- D. 게임 서버의 사용자 세션 관리

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **PostgreSQL**은 데이터 정합성과 확장성이 요구되는 다양한 분야에 사용됩니다. 특히 **위치 기반 서비스, 금융 시스템, 의료 정보 시스템** 등 고도화된 데이터 처리가 필요한 영역에서 활용됩니다. ACID 트랜잭션, JSON 지원, 확장 기능 등 강력한 기능이 이러한 복잡한 요구사항에 적합합니다.

## 오답 포인트

- A: 스타트업의 초기 프로토타입 개발은 공식 문서에서 MySQL의 사용 사례로 언급됩니다.
- C: IoT 데이터 수집 및 저장은 공식 문서에서 MySQL의 사용 사례로 언급됩니다.
- D: 게임 서버의 세션 관리는 공식 문서에서 MySQL의 사용 사례입니다.
