## 문제

카카오클라우드 **MySQL** 서비스의 기반으로 옳은 것은?

## 보기

- A. Oracle MySQL Enterprise Edition
- B. MySQL 오픈소스 커뮤니티 에디션(Community Edition)
- C. MariaDB 오픈소스 에디션
- D. 카카오클라우드 자체 개발 MySQL 포크(Fork)

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드의 MySQL 서비스는 **오픈소스 MySQL 커뮤니티 에디션을 기반**으로 제공되는 완전 관리형 관계형 데이터베이스 관리 시스템(RDBMS) 서비스입니다.

## 오답 포인트

- A: Oracle MySQL Enterprise Edition은 유료 상용 제품으로, 카카오클라우드 MySQL의 기반이 아닙니다.
- C: MariaDB는 별개의 오픈소스 DBMS입니다. 카카오클라우드 MySQL은 MySQL 커뮤니티 에디션 기반입니다.
- D: 자체 개발 포크가 아닌 오픈소스 MySQL 커뮤니티 에디션을 기반으로 합니다.
