## 문제

카카오클라우드 **MemStore**에 대한 설명으로 옳은 것은?

## 보기

- A. MySQL을 기반으로 한 완전 관리형 관계형 데이터베이스 서비스이다.
- B. 데이터를 디스크와 메모리에 함께 저장하여 영속성을 보장하는 데이터베이스 서비스이다.
- C. Redis OSS를 기반으로 하는 완전 관리형 인 메모리 데이터 스토어 서비스이다.
- D. 파일 형태로 데이터를 저장하는 완전 관리형 파일 데이터베이스 서비스이다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, MemStore는 서버의 운영 관리 부담을 덜어주고 데이터베이스를 쉽게 사용할 수 있도록 도와주는 완전 관리형 인 메모리 데이터 스토어(in-memory data store) 서비스입니다. MemStore가 제공하는 Redis는 오픈 소스 기반의 Redis OSS 버전으로, 메모리 내에 데이터를 저장하고 빠르게 데이터를 검색하는 NoSQL 데이터베이스 시스템입니다.

## 오답 포인트

- A: MySQL을 기반으로 한 관계형 데이터베이스 서비스는 MemStore가 아닌 MySQL 서비스입니다. MemStore는 Redis OSS 기반의 인 메모리 서비스입니다.
- B: MemStore는 Cache Only 방식으로 디스크를 사용하지 않고 모든 데이터를 메모리에만 저장합니다. 영속성(디스크 저장)을 보장하지 않습니다.
- D: MemStore는 파일 형태로 데이터를 저장하는 서비스가 아닙니다. 모든 데이터를 메모리에 저장하는 인 메모리 데이터 스토어 서비스입니다.
