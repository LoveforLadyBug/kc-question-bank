## 문제

카카오클라우드 DNS 서비스가 처리하는 핵심 기능은 무엇인가?

## 보기

- A. 네트워크 트래픽을 여러 인스턴스에 고르게 분산한다
- B. VPC 간 안전한 통신 경로를 설정하고 관리한다
- C. 사용자가 입력한 도메인을 해당 서버의 실제 IP 주소로 변환하여 응답한다
- D. 퍼블릭 인터넷과 프라이빗 네트워크 사이의 방화벽 정책을 적용한다

## 정답

C

## 해설

DNS 서비스는 사용자가 웹 브라우저 등에 도메인 이름을 입력했을 때, 해당 도메인을 호스팅하는 서버의 실제 IP 주소로 변환하여 응답하는 역할을 합니다.
이 과정을 통해 사용자는 복잡한 IP 주소를 기억하지 않아도 도메인 이름만으로 원하는 서비스에 접근할 수 있습니다.

## 오답 포인트

- A: 트래픽을 여러 인스턴스에 분산하는 것은 Load Balancing 서비스의 역할입니다.
- B: VPC 간 통신 경로 설정 및 관리는 Transit Gateway의 역할에 해당합니다.
- D: 방화벽 정책 적용은 Security Group 또는 Network ACL과 같은 보안 서비스의 역할입니다.
