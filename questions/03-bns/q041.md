## 문제

카카오클라우드 VPC(Virtual Private Cloud)에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 물리적으로 독립된 전용 서버 클러스터를 사용자에게 할당하는 서비스이다
- B. 사용자의 트래픽을 여러 서버에 자동으로 분산시켜 주는 서비스이다
- C. 사용자에게 논리적으로 격리된 가상 네트워크 공간을 제공하는 서비스이다
- D. 도메인 이름을 IP 주소로 변환하여 인터넷 접속을 가능하게 하는 서비스이다

## 정답

C

## 해설

VPC는 Virtual Private Cloud의 약자로, 논리적으로 격리된 가상 네트워크 공간을 사용자에게 제공하는 서비스입니다.
물리적 분리가 아닌 소프트웨어 기반의 논리적 격리를 통해 독립적인 네트워크 환경을 구성할 수 있으며, 이는 보안성과 유연성을 동시에 제공하는 핵심 특징입니다.

## 오답 포인트

- A: 물리적 전용 서버 할당은 베어메탈 서비스의 개념에 해당하며, VPC는 논리적 격리를 기반으로 합니다.
- B: 트래픽을 여러 서버에 분산시키는 것은 Load Balancing 서비스의 역할입니다.
- D: 도메인을 IP 주소로 변환하는 것은 DNS 서비스의 역할입니다.
