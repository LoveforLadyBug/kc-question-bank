## 문제

카카오클라우드 Load Balancing 서비스의 주요 목적으로 가장 적절한 것은?

## 보기

- A. 사용자 네트워크를 논리적으로 격리하여 보안을 강화하는 것
- B. 도메인 이름을 IP 주소로 변환하여 접속을 가능하게 하는 것
- C. 사용자 환경에 맞는 트래픽 분산 설정으로 효율적인 처리를 제공하는 것
- D. 여러 VPC를 단일 게이트웨이로 묶어 관리하는 것

## 정답

C

## 해설

Load Balancing은 사용자 환경에 맞는 트래픽 분산 설정을 통해 들어오는 요청을 여러 서버에 균등하게 나눠 처리 효율을 높이는 서비스입니다.
특정 서버에 부하가 집중되는 것을 방지함으로써 서비스의 안정성과 응답 속도를 개선하는 것이 핵심 역할입니다.

## 오답 포인트

- A: 네트워크의 논리적 격리는 VPC의 역할로, Load Balancing의 목적과 다릅니다.
- B: 도메인-IP 변환은 DNS 서비스의 기능입니다.
- D: 여러 VPC를 중앙에서 연결·관리하는 것은 Transit Gateway의 역할입니다.
