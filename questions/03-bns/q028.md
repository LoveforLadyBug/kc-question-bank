## 문제

사용자가 웹 브라우저에 "www.example.com"을 입력했을 때 DNS 서비스가 수행하는 동작으로 가장 적절한 것은?

## 보기

- A. 요청 트래픽을 가장 응답 속도가 빠른 서버로 자동 전환한다
- B. 입력된 도메인 이름에 매핑된 서버의 IP 주소를 조회하여 반환한다
- C. 도메인 접속 이력을 기반으로 사용자 인증 여부를 판단한다
- D. 여러 서버로 트래픽을 나누어 각 서버의 부하를 동일하게 유지한다

## 정답

B

## 해설

DNS 서비스는 사용자가 도메인 이름을 입력하면 해당 도메인에 매핑된 서버의 실제 IP 주소를 조회하여 반환(응답)합니다.
이 과정을 이름 해석(Name Resolution)이라 하며, 사용자의 브라우저는 반환된 IP 주소로 실제 서버에 접속하게 됩니다.

## 오답 포인트

- A: 응답 속도 기반의 서버 전환은 Load Balancing 또는 글로벌 트래픽 관리 서비스의 기능입니다.
- C: 사용자 인증 판단은 DNS의 역할이 아니며, 인증 및 권한 관리 서비스(IAM 등)에 해당합니다.
- D: 부하를 균등하게 분산하는 것은 Load Balancing 서비스의 역할입니다.
