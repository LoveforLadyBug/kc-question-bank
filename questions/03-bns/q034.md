## 문제

카카오클라우드 DNS 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 여러 서버로 들어오는 요청 트래픽을 균형 있게 나누어 처리한다
- B. 네트워크 간 라우팅 정보를 중앙에서 관리하여 경로를 최적화한다
- C. 사용자가 입력한 도메인을 해당 서버의 실제 IP 주소로 응답한다
- D. 사용자별로 독립된 가상 네트워크 공간을 생성하고 격리한다

## 정답

C

## 해설

DNS 서비스는 사용자가 사이트에 접속하기 위해 입력한 도메인을 해당 사이트를 호스팅하는 컴퓨터의 실제 IP 주소로 응답하는 서비스입니다.
이를 통해 사용자는 복잡한 IP 주소를 외우지 않고 도메인 이름만으로 서비스에 접근할 수 있습니다.

## 오답 포인트

- A: 트래픽을 균형 있게 분산하는 것은 Load Balancing의 기능입니다.
- B: 라우팅 경로를 중앙에서 관리하는 것은 Transit Gateway의 역할입니다.
- D: 독립된 가상 네트워크 공간 생성은 VPC의 역할에 해당합니다.
