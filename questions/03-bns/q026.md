## 문제

VPC(Virtual Private Cloud)의 '논리적 격리'가 의미하는 바로 가장 적절한 것은?

## 보기

- A. 사용자마다 별도의 물리 서버를 구매하여 완전히 독립된 환경을 구성하는 것
- B. 동일한 물리 인프라를 공유하되, 소프트웨어적으로 네트워크를 분리하여 독립성을 확보하는 것
- C. 인터넷 접근을 차단하고 내부 네트워크만 허용하도록 방화벽을 설정하는 것
- D. 서로 다른 리전의 VPC를 물리적 전용선으로 연결하여 통합 운영하는 것

## 정답

B

## 해설

VPC의 논리적 격리는 동일한 물리 인프라를 여러 사용자가 공유하면서도, 소프트웨어 정의 네트워킹(SDN) 기술로 각 사용자의 네트워크를 독립적으로 분리하는 개념입니다.
물리적 분리 없이도 보안성과 독립성을 확보할 수 있어 클라우드 환경의 핵심 특성으로 꼽힙니다.

## 오답 포인트

- A: 물리 서버를 별도 구매하는 방식은 베어메탈 서버 개념으로, VPC의 논리적 격리와 다릅니다.
- C: 방화벽 설정은 보안 그룹(Security Group) 등의 기능이며, 논리적 격리의 정의가 아닙니다.
- D: 리전 간 전용선 연결은 Transit Gateway 또는 Direct Connect 개념에 해당합니다.
