## 문제

카카오클라우드 VPC가 제공하는 네트워크 환경의 특성으로 가장 적절한 것은?

## 보기

- A. 모든 사용자가 동일한 공용 네트워크를 공유하여 사용한다
- B. 사용자별로 논리적으로 격리된 독립적인 네트워크 공간이 제공된다
- C. 네트워크 설정 변경 시 서비스 재시작이 반드시 필요하다
- D. 단일 가용 영역(AZ)에서만 동작하도록 설계되어 있다

## 정답

B

## 해설

VPC는 각 사용자에게 논리적으로 격리된 독립적인 가상 네트워크 공간을 제공하여, 다른 사용자의 네트워크와 완전히 분리된 환경에서 리소스를 운영할 수 있게 합니다.
이러한 격리 구조는 보안성을 높이고, 사용자가 IP 대역·서브넷·라우팅 등을 자유롭게 구성할 수 있는 유연성을 제공합니다.

## 오답 포인트

- A: VPC의 핵심은 격리된 사설 네트워크 공간 제공으로, 공용 네트워크 공유 방식과는 정반대의 개념입니다.
- C: 논리적 네트워크 구성인 VPC는 설정 변경 시 서비스 재시작 없이 동적으로 적용하는 것이 클라우드의 장점입니다.
- D: 카카오클라우드는 다중 AZ 환경을 지원하며, VPC도 이를 기반으로 고가용성 구성이 가능합니다.
