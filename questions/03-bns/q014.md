## 문제

카카오클라우드 DNS 서비스가 처리하는 작업으로 가장 적절한 것은?

## 보기

- A. 사용자 트래픽을 여러 서버에 균형 있게 분배한다
- B. 사용자가 입력한 도메인을 해당 서버의 실제 IP 주소로 응답한다
- C. VPC 간 네트워크 연결을 중앙에서 관리하고 라우팅한다
- D. 사용자별로 격리된 독립적인 가상 네트워크 공간을 생성한다

## 정답

B

## 해설

DNS 서비스는 사용자가 사이트에 접속하기 위해 입력한 도메인을 해당 사이트를 호스팅하는 서버의 실제 IP 주소로 변환하여 응답합니다.
이 이름 해석(Name Resolution) 과정이 없으면 브라우저는 도메인만으로 목적지 서버를 찾을 수 없어 인터넷 접속이 불가능합니다.

## 오답 포인트

- A: 트래픽을 여러 서버에 분배하는 것은 Load Balancing의 역할입니다.
- C: VPC 간 네트워크를 중앙에서 관리하는 것은 Transit Gateway의 기능입니다.
- D: 격리된 가상 네트워크 공간 생성은 VPC의 역할입니다.
