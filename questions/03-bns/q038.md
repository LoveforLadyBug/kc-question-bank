## 문제

사용자가 웹 브라우저에 특정 사이트 주소(도메인)를 입력했을 때, DNS 서비스가 수행하는 동작으로 가장 정확한 것은?

## 보기

- A. 사용자 요청을 부하가 가장 낮은 서버로 자동 전달한다
- B. 입력된 도메인에 해당하는 서버의 실제 IP 주소를 찾아 응답한다
- C. 도메인 접속 이력을 기록하고 보안 위협 여부를 검사한다
- D. 여러 VPC에 걸쳐 있는 라우팅 테이블을 자동으로 갱신한다

## 정답

B

## 해설

DNS 서비스는 사용자가 입력한 도메인 이름을 해당 사이트를 호스팅하는 서버의 실제 IP 주소로 변환하여 응답합니다.
이 과정을 통해 사용자는 숫자로 이루어진 IP 주소를 직접 알지 않아도 원하는 서비스에 접속할 수 있습니다.

## 오답 포인트

- A: 부하가 낮은 서버로 요청을 전달하는 것은 Load Balancing의 역할이며 DNS와는 다릅니다.
- C: 보안 위협 검사는 DNS의 기본 동작 원리가 아니며 별도 보안 서비스의 기능에 해당합니다.
- D: 라우팅 테이블 갱신은 Transit Gateway 또는 VPC 라우팅 관련 기능으로 DNS와 무관합니다.
