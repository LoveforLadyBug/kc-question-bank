## 문제

카카오클라우드의 Networking 서비스 그룹에 포함된 서비스 조합으로 올바른 것은?

## 보기

- A. VPC, Object Storage, Load Balancing
- B. VPC, Transit Gateway, Load Balancing
- C. Transit Gateway, Block Storage, DNS
- D. Load Balancing, Kubernetes, VPC

## 정답

B

## 해설

카카오클라우드의 Networking 서비스 그룹에는 VPC, Transit Gateway, Load Balancing 등이 포함되며, 각 서비스는 특화된 네트워크 기술을 보완하며 구성됩니다.
Object Storage, Block Storage, Kubernetes는 각각 스토리지·컨테이너 서비스 범주에 해당하여 Networking 그룹에 포함되지 않습니다.

## 오답 포인트

- A: Object Storage는 네트워킹 서비스가 아닌 스토리지 서비스 범주에 속합니다.
- C: Block Storage는 스토리지 서비스이며, Networking 그룹에 포함되지 않습니다.
- D: Kubernetes(컨테이너 오케스트레이션)는 컨테이너 서비스 범주로, Networking 그룹 외부에 위치합니다.
