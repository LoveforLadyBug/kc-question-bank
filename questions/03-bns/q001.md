## 문제

카카오클라우드 VPC(Virtual Private Cloud)의 핵심 역할로 가장 적절한 것은?

## 보기

- A. 물리적으로 분리된 전용 하드웨어 서버를 제공하는 서비스
- B. 논리적으로 격리된 가상 네트워크 공간을 제공하는 서비스
- C. 사용자 도메인을 IP 주소로 변환하는 DNS 전용 서비스
- D. 다수의 서버에 트래픽을 분산시키는 부하 분산 서비스

## 정답

B

## 해설

VPC는 클라우드 환경에서 논리적으로 격리된 가상 네트워크 공간을 제공하는 서비스로, 사용자가 자신만의 네트워크 환경을 구성할 수 있도록 합니다.
물리적 분리가 아닌 소프트웨어 기반의 논리적 격리를 통해 보안성과 유연성을 동시에 확보하는 것이 VPC의 핵심 개념입니다.

## 오답 포인트

- A: VPC는 물리적 하드웨어 분리가 아닌 논리적(소프트웨어) 격리를 제공합니다. 물리 전용 서버는 베어메탈 서비스에 해당합니다.
- C: 도메인을 IP 주소로 변환하는 것은 DNS(Domain Name System) 서비스의 역할이며, VPC의 주요 기능이 아닙니다.
- D: 트래픽 분산은 Load Balancing 서비스의 역할로, VPC와는 다른 서비스입니다.
