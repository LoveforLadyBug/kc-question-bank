## 문제

사용자가 웹 브라우저에 특정 도메인 주소를 입력했을 때, 카카오클라우드 DNS 서비스가 수행하는 동작으로 올바른 것은?

## 보기

- A. 사용자의 요청을 가장 가까운 서버로 물리적으로 라우팅한다
- B. 입력된 도메인을 해당 사이트를 호스팅하는 서버의 실제 IP 주소로 응답한다
- C. 도메인 요청을 암호화하여 보안 터널을 생성한다
- D. 여러 서버 중 부하가 가장 낮은 서버로 자동 연결한다

## 정답

B

## 해설

DNS 서비스는 사용자가 입력한 도메인 이름을 해당 사이트를 호스팅하는 컴퓨터의 실제 IP 주소로 변환하여 응답하는 역할을 담당합니다.
이를 통해 사용자는 복잡한 IP 주소를 외울 필요 없이 도메인 이름만으로 원하는 서버에 접근할 수 있습니다.

## 오답 포인트

- A: 물리적 라우팅은 네트워크 라우터의 역할이며, DNS는 이름 해석(Name Resolution)에 집중합니다.
- C: 암호화 터널 생성은 VPN 서비스의 기능으로, DNS의 역할과 구분됩니다.
- D: 부하 기반 자동 연결은 Load Balancing 서비스의 기능이며, 일반적인 DNS의 주요 역할은 아닙니다.
