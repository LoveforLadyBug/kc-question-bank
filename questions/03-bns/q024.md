## 문제

카카오클라우드 DNS 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 여러 서버 간의 트래픽을 균등하게 분산하여 과부하를 방지하는 서비스
- B. 사용자가 입력한 도메인에 대응하는 실제 IP 주소를 응답해 주는 서비스
- C. 다수의 VPC와 온프레미스 네트워크를 단일 네트워크로 연결하는 서비스
- D. 사용자별 격리된 가상 네트워크를 할당하여 보안을 확보하는 서비스

## 정답

B

## 해설

DNS(Domain Name System) 서비스는 사용자가 브라우저에 도메인을 입력하면, 해당 사이트를 호스팅하는 서버의 실제 IP 주소를 응답하는 이름 해석 서비스입니다.
이를 통해 사용자는 복잡한 IP 주소를 기억하지 않고도 도메인 이름만으로 서비스에 접근할 수 있습니다.

## 오답 포인트

- A: 서버 간 트래픽 분산은 Load Balancing 서비스의 역할입니다.
- C: VPC와 온프레미스 네트워크 연결은 Transit Gateway의 기능입니다.
- D: 격리된 가상 네트워크 제공은 VPC의 역할입니다.
