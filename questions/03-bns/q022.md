## 문제

카카오클라우드 Load Balancing 서비스의 핵심 역할로 가장 적절한 것은?

## 보기

- A. 도메인 이름을 IP 주소로 변환하여 서비스 접근을 가능하게 한다
- B. 사용자 환경에 적합한 트래픽 분산 설정을 통해 효율적으로 처리한다
- C. 여러 VPC를 하나의 허브 네트워크로 묶어 중앙 관리한다
- D. 사용자별로 격리된 가상 네트워크 공간을 제공한다

## 정답

B

## 해설

Load Balancing은 사용자 환경에 적합한 트래픽 분산 설정을 통해 인입 트래픽을 여러 서버에 효율적으로 분배하는 서비스입니다.
이를 통해 특정 서버에 트래픽이 집중되는 것을 방지하고 서비스 안정성과 응답 속도를 향상시킵니다.

## 오답 포인트

- A: 도메인을 IP로 변환하는 것은 DNS 서비스의 역할입니다.
- C: 여러 VPC를 허브 네트워크로 연결하는 것은 Transit Gateway의 역할입니다.
- D: 격리된 가상 네트워크 공간 제공은 VPC의 역할입니다.
