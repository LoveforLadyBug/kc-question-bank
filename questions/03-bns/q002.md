## 문제

카카오클라우드 네트워킹 서비스 중 사용자가 입력한 도메인 이름을 해당 서버의 실제 IP 주소로 응답해 주는 역할을 담당하는 서비스는?

## 보기

- A. VPC (Virtual Private Cloud)
- B. Transit Gateway
- C. DNS (Domain Name System)
- D. Load Balancing

## 정답

C

## 해설

DNS 서비스는 사용자가 브라우저에 도메인 이름을 입력했을 때, 해당 도메인을 실제로 호스팅하는 컴퓨터의 IP 주소로 변환하여 응답하는 역할을 합니다.
이는 인터넷의 전화번호부와 같은 역할로, 사람이 읽기 쉬운 도메인과 컴퓨터가 통신에 사용하는 IP 주소를 연결하는 핵심 인프라입니다.

## 오답 포인트

- A: VPC는 격리된 가상 네트워크 공간을 제공하는 서비스로, 도메인-IP 변환 기능을 담당하지 않습니다.
- B: Transit Gateway는 여러 VPC와 온프레미스 네트워크를 연결하는 중앙 허브 역할을 하며, 도메인 변환과는 무관합니다.
- D: Load Balancing은 트래픽을 여러 서버에 분산시키는 서비스로, DNS의 도메인 해석 기능과는 다른 역할을 합니다.
