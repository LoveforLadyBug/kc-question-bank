## 문제

카카오클라우드 **IAM(Identity and Access Management) 서비스**의 요금에 대한 설명으로 옳은 것은?

## 보기

- A. IAM 서비스는 사용자 수에 비례하여 월정액 요금이 부과된다.
- B. IAM 서비스는 API 호출 횟수에 따라 요금이 부과된다.
- C. IAM 서비스는 현재 기본적으로 제공되며 별도의 과금이 발생하지 않는다.
- D. IAM 서비스는 리전별로 별도로 과금되는 리전 서비스이다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, IAM 서비스는 모든 리전에서 공통적으로 제공되는 글로벌 서비스로, 현재 기본적으로 제공되며 별도의 과금은 발생하지 않습니다. 다만, 향후 API별 과금 항목이 추가될 수 있다고 안내되어 있습니다.

## 오답 포인트

- A: IAM 서비스는 사용자 수에 따른 월정액 과금이 발생하지 않습니다. 현재 별도의 과금이 없습니다.
- B: IAM 서비스는 현재 API 호출 횟수에 따른 과금이 없습니다. 향후 API별 과금 항목이 추가될 수 있다고 언급되어 있지만, 현재는 무료입니다.
- D: IAM은 리전별로 별도 과금되는 리전 서비스가 아니라, 모든 리전에서 공통으로 제공되는 글로벌 서비스입니다.
