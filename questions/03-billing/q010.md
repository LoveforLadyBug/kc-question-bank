## 문제

카카오클라우드 Kubernetes Engine의 쿼터 관련 설명으로 **옳지 않은** 것은?

## 보기

- A. 프로젝트별 클러스터 기본 쿼터는 10개이며, 증설 요청이 가능하다.
- B. 클러스터별 노드 풀 기본 쿼터는 20개이며, 증설 요청이 가능하다.
- C. 클러스터별 노드 기본 쿼터는 100개이며, BMS 노드는 클러스터별 5개까지 생성 가능하다.
- D. 자동 생성된 인스턴스, 볼륨, 보안 그룹의 프로젝트 쿼터 초과 여부와 무관하게 Kubernetes Engine 리소스를 생성할 수 있다.

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, 자동 생성된 리소스(인스턴스, 볼륨, 보안 그룹)의 프로젝트 쿼터가 초과된 경우 Kubernetes Engine의 리소스(클러스터, 노드 풀, 노드)를 신규 생성할 수 없습니다. 즉, 연관 서비스의 쿼터 한도도 Kubernetes Engine 리소스 생성에 영향을 미칩니다.

## 오답 포인트

- A: 프로젝트별 클러스터 기본 쿼터는 10개이며, 헬프데스크를 통해 증설 요청이 가능합니다. 올바른 설명입니다.
- B: 클러스터별 노드 풀 기본 쿼터는 20개이며, 헬프데스크를 통해 증설 요청이 가능합니다. 올바른 설명입니다.
- C: 클러스터별 노드 기본 쿼터는 100개이며, BMS 노드는 클러스터별 5개까지 생성 가능합니다. 올바른 설명입니다.
