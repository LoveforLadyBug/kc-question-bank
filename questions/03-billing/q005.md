## 문제

카카오클라우드 Load Balancing 서비스의 **프로젝트당 기본 로드 밸런서 쿼터**와 **증설 가능 여부**로 옳은 것은?

## 보기

- A. 10개, 증설 가능
- B. 30개, 증설 가능
- C. 50개, 증설 불가능
- D. 100개, 증설 가능

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Load Balancing 서비스의 기본 쿼터 중 로드 밸런서는 프로젝트당 50개이며, 증설 요청이 불가능합니다. 리스너는 프로젝트당 150개, 대상 그룹은 750개로 제한되며 이들도 모두 증설 불가입니다.

## 오답 포인트

- A: 10개는 Kubernetes Engine의 프로젝트당 클러스터 기본 쿼터입니다. 로드 밸런서 기본 쿼터는 50개이며 증설이 불가능합니다.
- B: 30개는 VPC의 프로젝트당 퍼블릭 IP 쿼터 또는 서브넷 쿼터입니다. 로드 밸런서 기본 쿼터는 50개입니다.
- D: 로드 밸런서 기본 쿼터는 100개가 아닌 50개이며, 증설이 불가능합니다. 증설이 가능하다는 설명도 잘못되었습니다.
