## 문제

카카오클라우드 Virtual Machine 서비스의 **프로젝트당 기본 인스턴스 생성 가능 개수**로 옳은 것은?

## 보기

- A. 100개
- B. 300개
- C. 500개
- D. 1,000개

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Virtual Machine의 기본 쿼터 중 프로젝트별로 생성 가능한 최대 인스턴스 개수는 500개입니다. 이 쿼터는 헬프데스크를 통해 증설 요청이 가능합니다. 또한 vCPU는 4,000개, 메모리는 8,000GiB까지 기본으로 할당됩니다.

## 오답 포인트

- A: 100개는 Kubernetes Engine 클러스터당 최대 노드 수입니다. VM의 프로젝트당 기본 인스턴스 쿼터가 아닙니다.
- B: 300개는 카카오클라우드 공식 문서에서 언급되지 않는 수치입니다. VM 프로젝트당 기본 인스턴스 쿼터는 500개입니다.
- D: 1,000개는 VM 프로젝트당 기본 볼륨 개수 쿼터입니다. 인스턴스 기본 쿼터는 500개입니다.
