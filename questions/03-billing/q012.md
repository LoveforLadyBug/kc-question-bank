## 문제

카카오클라우드 Virtual Machine에서 **기본 이미지로 제공되는 Linux 배포판**으로 옳지 **않은** 것은?

## 보기

- A. Ubuntu 22.04
- B. CentOS Stream 9
- C. Debian 12
- D. Rocky Linux 9.4

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, VM 서비스에서 제공하는 Linux 기본 이미지는 **Ubuntu 20.04/22.04/24.04**, **CentOS Stream 9**, **Rocky Linux 8.10/9.4**, **Alma Linux 8.10/9.4**입니다. **Debian** 배포판은 공식 문서에서 지원하는 기본 이미지 목록에 포함되어 있지 않습니다.

## 오답 포인트

- A: Ubuntu 22.04는 제공되는 Linux 기본 이미지입니다.
- B: CentOS Stream 9는 제공되는 Linux 기본 이미지입니다.
- D: Rocky Linux 9.4는 제공되는 Linux 기본 이미지입니다.
