## 문제

카카오클라우드 VPC 서비스의 **퍼블릭 IP 기본 쿼터**와 **증설 가능 여부**로 옳은 것은?

## 보기

- A. 프로젝트별 10개, 증설 가능
- B. 프로젝트별 30개, 증설 불가능
- C. 프로젝트별 50개, 증설 가능
- D. 프로젝트별 100개, 증설 불가능

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, VPC 서비스에서 퍼블릭 IP의 기본 쿼터는 프로젝트별 30개이며, 증설 요청이 불가능합니다. 참고로 VPC 자체는 3개/프로젝트(증설 가능), 서브넷은 30개/프로젝트(증설 가능), 보안 그룹은 180개/프로젝트(증설 불가)입니다.

## 오답 포인트

- A: 퍼블릭 IP 기본 쿼터는 10개가 아닌 30개이며, 증설이 불가능합니다.
- C: 50개는 Load Balancing 서비스의 로드 밸런서 기본 쿼터입니다. 퍼블릭 IP 기본 쿼터는 30개입니다.
- D: 퍼블릭 IP 기본 쿼터는 100개가 아닌 30개입니다. 증설 불가능하다는 점은 맞지만, 수량이 틀렸습니다.
