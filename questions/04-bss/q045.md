## 문제

카카오클라우드의 Object Storage와 File Storage를 비교한 설명으로 가장 적절한 것은?

## 보기

- A. Object Storage는 NFS 프로토콜을 사용하고, File Storage는 키-값 구조로 저장한다.
- B. Object Storage는 키-값 구조로 대용량 데이터를 저장하고, File Storage는 NFS로 파일을 공유한다.
- C. 두 서비스 모두 블록 단위 입출력을 기반으로 동작한다.
- D. File Storage는 단일 서버에서만 접근할 수 있고, Object Storage는 다중 서버에서 접근할 수 없다.

## 정답

B

## 해설

Object Storage는 키-값(Key-value) 형태로 대용량 데이터를 저장·처리하며, File Storage는 NFS 프로토콜을 통해 여러 서버와 애플리케이션 간 파일 공유를 담당합니다.
두 서비스는 저장 방식과 접근 프로토콜이 명확히 다르며, 각각의 사용 목적에 맞게 설계되어 있습니다.

## 오답 포인트

- A: Object Storage와 File Storage의 특성이 서로 뒤바뀐 설명으로, NFS는 File Storage가 사용하는 프로토콜입니다.
- C: 블록 단위 입출력은 Block Storage의 특성이며, Object Storage와 File Storage 모두에 해당하지 않습니다.
- D: File Storage는 다수의 서버에서 동시에 접근하는 것이 핵심이며, Object Storage도 다중 서버 접근이 가능합니다.
