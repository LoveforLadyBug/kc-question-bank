## 문제

카카오클라우드 Object Storage의 특성을 가장 올바르게 설명한 것은 무엇입니까?

## 보기

- A. 낮은 지연 시간이 요구되는 실시간 트랜잭션 처리에 최적화되어 있다.
- B. 대용량 데이터를 키-값 형태로 저장하며 높은 확장성과 안정성을 제공한다.
- C. NFS 프로토콜을 사용하여 여러 서버가 동시에 동일 파일에 접근할 수 있다.
- D. 파일 시스템 계층 구조를 기반으로 디렉터리와 파일을 관리한다.

## 정답

B

## 해설

Object Storage는 대용량 데이터를 키-값(Key-value) 방식으로 저장하는 데 최적화된 서비스로, 높은 확장성과 안정성이 핵심 특성입니다.
실시간 트랜잭션 처리나 NFS 프로토콜 공유는 각각 Block Storage, File Storage의 특성에 해당합니다.

## 오답 포인트

- A: 낮은 지연 시간과 실시간 트랜잭션 처리는 Block Storage의 특성입니다.
- C: NFS 프로토콜을 통한 다중 서버 공유는 File Storage의 핵심 특성입니다.
- D: 파일 시스템의 디렉터리-파일 계층 구조는 File Storage에 해당하며, Object Storage는 계층 없이 객체를 저장합니다.
