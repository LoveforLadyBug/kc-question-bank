## 문제

카카오클라우드 Storage 서비스 그룹의 설명으로 가장 적절한 것은?

## 보기

- A. 낮은 확장성 대신 극단적으로 낮은 지연 시간을 제공하는 로컬 디스크 관리 서비스이다.
- B. 높은 확장성과 성능, 안정성을 기반으로 데이터를 보관 및 처리하는 서비스 그룹이다.
- C. 단일 인스턴스 전용으로 데이터를 블록 단위로 처리하는 서비스이다.
- D. 관계형 데이터베이스 연동을 위한 전용 스토리지 솔루션이다.

## 정답

B

## 해설

카카오클라우드 Storage 서비스 그룹은 높은 확장성과 성능, 안정성을 기반으로 데이터를 보관 및 처리하는 서비스들의 집합입니다.
Object Storage와 File Storage 등 여러 서비스를 포함하며, 다양한 워크로드의 데이터 저장 요구를 충족합니다.

## 오답 포인트

- A: 낮은 확장성이나 로컬 디스크 관리는 카카오클라우드 Storage 서비스 그룹의 특징과 다릅니다.
- C: 단일 인스턴스 전용 블록 단위 처리는 블록 스토리지의 특징으로, Storage 전체 그룹을 설명하지 못합니다.
- D: 관계형 데이터베이스 연동 전용 솔루션은 스토리지 서비스 그룹의 목적이 아닙니다.
