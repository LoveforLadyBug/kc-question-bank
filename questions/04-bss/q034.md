## 문제

카카오클라우드의 Object Storage와 File Storage를 비교한 내용으로 가장 적절한 것은?

## 보기

- A. Object Storage는 NFS 프로토콜로 파일을 공유하고, File Storage는 키-값 구조로 데이터를 저장한다.
- B. Object Storage는 키-값 구조로 대용량 데이터를 저장하고, File Storage는 NFS를 통해 여러 서버 간 파일을 공유한다.
- C. Object Storage와 File Storage 모두 NFS 프로토콜을 기반으로 동작한다.
- D. Object Storage는 단일 서버 전용이고, File Storage는 블록 단위로 데이터를 처리한다.

## 정답

B

## 해설

Object Storage는 키-값(Key-value) 형태로 대용량 데이터를 저장·처리하며, File Storage는 NFS 프로토콜을 사용하여 여러 서버와 애플리케이션 간에 파일을 공유합니다.
두 서비스는 저장 방식과 접근 프로토콜이 명확히 구분되며, 사용 사례도 다릅니다.

## 오답 포인트

- A: Object Storage와 File Storage의 특징이 서로 바뀌어 설명된 오답입니다.
- C: NFS 프로토콜은 File Storage에서만 사용하며, Object Storage는 NFS를 사용하지 않습니다.
- D: Object Storage는 단일 서버 전용이 아니며, File Storage는 파일 공유 서비스로 블록 단위 처리와 무관합니다.
