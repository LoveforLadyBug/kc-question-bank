## 문제

카카오클라우드 File Storage를 사용하기에 가장 적합한 시나리오는?

## 보기

- A. 단일 가상 머신에서만 접근하는 OS 부트 디스크 저장
- B. 여러 웹 서버 인스턴스가 동일한 첨부 파일 디렉터리를 공유
- C. IoT 센서에서 수집한 원시 데이터를 객체 형태로 장기 보관
- D. 관계형 데이터를 테이블 구조로 저장하고 SQL로 조회

## 정답

B

## 해설

File Storage는 NFS 프로토콜을 사용하여 다수의 서버와 애플리케이션이 동일한 파일 시스템을 공유하는 환경에 적합합니다.
여러 웹 서버가 동일한 첨부 파일 디렉터리에 접근해야 하는 경우, File Storage를 마운트하여 일관된 데이터 공유가 가능합니다.

## 오답 포인트

- A: OS 부트 디스크처럼 단일 인스턴스 전용의 고성능 블록 접근은 블록 스토리지에 적합합니다.
- C: 객체 형태의 대용량 원시 데이터 장기 보관은 Object Storage의 사용 사례에 해당합니다.
- D: SQL 기반의 관계형 데이터 조회는 데이터베이스 서비스(RDB)의 역할입니다.
