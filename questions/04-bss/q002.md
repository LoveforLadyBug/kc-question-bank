## 문제

카카오클라우드 Object Storage가 최적화되어 있는 주요 사용 목적으로 가장 적절한 것은?

## 보기

- A. 소용량 파일의 빠른 블록 단위 입출력
- B. 대용량 데이터의 저장 및 처리
- C. 단일 서버 내 로컬 디스크 관리
- D. 실시간 스트리밍 트랜잭션 처리

## 정답

B

## 해설

Object Storage는 대용량 데이터를 객체 형태로 저장하고 처리하는 데 최적화된 서비스입니다.
높은 확장성과 안정성을 기반으로 설계되어, 이미지·동영상·로그 등 비정형 대용량 데이터 관리에 적합합니다.

## 오답 포인트

- A: 블록 단위 빠른 입출력은 블록 스토리지의 특성이며, Object Storage의 주된 용도가 아닙니다.
- C: 단일 서버 내 로컬 디스크 관리는 Object Storage의 범위 밖이며, 이는 인스턴스 로컬 스토리지에 해당합니다.
- D: 실시간 트랜잭션 처리는 데이터베이스 서비스의 역할로, Object Storage의 설계 목적과 다릅니다.
