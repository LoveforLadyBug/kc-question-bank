## 문제

카카오클라우드 Object Storage의 특징으로 가장 적절한 것은?

## 보기

- A. 블록 기반 아키텍처로 낮은 지연 시간과 높은 IOPS를 제공한다.
- B. 확장성과 안정성을 갖추고 대용량 데이터 저장·처리에 최적화되어 있다.
- C. 파일 시스템 계층 구조를 통해 디렉터리 단위로 파일을 관리한다.
- D. 단일 가상 머신에 연결하여 OS 디스크로 주로 활용된다.

## 정답

B

## 해설

카카오클라우드 Object Storage는 높은 확장성과 안정성을 기반으로 대용량 데이터를 객체 형태로 저장·처리하는 데 최적화된 서비스입니다.
필요에 따라 저장 용량을 유연하게 확장할 수 있어 대용량 비정형 데이터 관리에 적합합니다.

## 오답 포인트

- A: 낮은 지연 시간과 높은 IOPS는 블록 스토리지의 특징이며 Object Storage와 다릅니다.
- C: 디렉터리 기반 파일 계층 구조 관리는 파일 스토리지(File Storage)의 특징입니다.
- D: 단일 VM에 연결하여 OS 디스크로 활용하는 것은 블록 스토리지의 대표적 사용 사례입니다.
