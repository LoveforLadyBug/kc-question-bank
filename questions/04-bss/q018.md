## 문제

카카오클라우드 File Storage에서 NFS 프로토콜을 사용하는 주된 이유로 가장 적절한 것은 무엇입니까?

## 보기

- A. 데이터를 블록 단위로 분할하여 단일 서버의 읽기/쓰기 속도를 극대화하기 위해
- B. 여러 서버와 애플리케이션이 네트워크를 통해 동일한 파일 공간을 공유하기 위해
- C. 파일을 객체로 변환하여 메타데이터 검색 성능을 높이기 위해
- D. 외부 인터넷과의 파일 전송 보안성을 강화하기 위해

## 정답

B

## 해설

NFS(Network File System)는 분산 파일 시스템 환경에서 여러 서버와 애플리케이션이 네트워크를 통해 동일한 파일 디렉터리를 마운트하고 공유할 수 있도록 설계된 프로토콜입니다.
카카오클라우드 File Storage는 바로 이 목적을 달성하기 위해 NFS를 채택하였습니다.

## 오답 포인트

- A: 블록 단위 I/O 속도 극대화는 Block Storage의 목적이며, NFS 프로토콜과는 관계없습니다.
- C: 파일을 객체로 변환하는 것은 Object Storage의 개념이며, NFS의 역할이 아닙니다.
- D: NFS는 LAN 환경의 파일 공유 프로토콜로, 외부 인터넷 보안 강화를 주목적으로 하지 않습니다.
