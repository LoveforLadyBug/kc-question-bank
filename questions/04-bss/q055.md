## 문제

다음 시나리오에서 카카오클라우드 스토리지 서비스를 올바르게 선택한 것은 무엇입니까?

> **시나리오 A**: 동영상 플랫폼에서 수백만 개의 영상 파일을 HTTP API로 업로드·다운로드하며 저장한다.  
> **시나리오 B**: 여러 웹 애플리케이션 서버가 공통 설정 파일과 로그 디렉터리를 실시간으로 함께 읽고 쓴다.

## 보기

- A. A: File Storage, B: Object Storage
- B. A: Object Storage, B: Object Storage
- C. A: Object Storage, B: File Storage
- D. A: File Storage, B: File Storage

## 정답

C

## 해설

시나리오 A는 대용량 비정형 데이터를 HTTP API 기반으로 저장·조회하는 구조로, 키-값 객체 방식의 Object Storage에 적합합니다.
시나리오 B는 여러 서버가 동일한 디렉터리를 동시에 마운트하여 파일을 공유하는 구조이므로, NFS 기반의 File Storage를 사용하는 것이 적절합니다.

## 오답 포인트

- A: File Storage는 HTTP API 기반의 대용량 객체 저장에 최적화되어 있지 않으며, Object Storage는 다중 서버의 실시간 파일 공유(마운트)를 지원하지 않습니다.
- B: Object Storage는 NFS 마운트를 통한 실시간 파일 공유를 제공하지 않으므로 시나리오 B에 적합하지 않습니다.
- D: File Storage는 HTTP API 기반의 대규모 비정형 객체 저장보다 네트워크 파일 시스템 공유에 특화되어 있어 시나리오 A에 부적합합니다.
