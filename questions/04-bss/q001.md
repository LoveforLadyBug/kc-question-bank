## 문제

카카오클라우드 Object Storage에서 데이터를 저장하고 처리하는 기본 단위 형태로 올바른 것은?

## 보기

- A. 블록(Block) 형태
- B. 테이블(Table) 형태
- C. 키-값(Key-value) 형태
- D. 트리(Tree) 형태

## 정답

C

## 해설

카카오클라우드 Object Storage는 데이터를 키-값(Key-value) 형태의 객체(Object)로 저장하고 처리하는 서비스입니다.
이 구조 덕분에 대용량 데이터를 효율적으로 관리할 수 있으며, 높은 확장성과 안정성을 제공합니다.

## 오답 포인트

- A: 블록 형태는 블록 스토리지(Block Storage)에 해당하는 저장 방식으로, Object Storage의 저장 방식이 아닙니다.
- B: 테이블 형태는 관계형 데이터베이스(RDB)의 저장 구조이며, Object Storage와는 다릅니다.
- D: 트리 형태는 파일 시스템의 디렉터리 계층 구조에 해당하며, Object Storage의 저장 방식이 아닙니다.
