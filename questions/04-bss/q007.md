## 문제

카카오클라우드 Object Storage와 File Storage를 비교한 설명으로 가장 옳은 것은?

## 보기

- A. Object Storage는 NFS 프로토콜을 사용하고, File Storage는 키-값 구조로 저장한다.
- B. Object Storage는 키-값 구조의 객체 저장에, File Storage는 NFS 기반 파일 공유에 최적화되어 있다.
- C. 두 서비스 모두 NFS 프로토콜을 기반으로 동작한다.
- D. 두 서비스 모두 키-값 구조를 사용하며 저장 방식에 차이가 없다.

## 정답

B

## 해설

Object Storage는 키-값(Key-value) 형태의 객체 저장에 최적화되어 있고, File Storage는 NFS 프로토콜 기반의 파일 공유에 최적화되어 있습니다.
두 서비스는 서로 다른 기술 기반과 사용 목적을 가지므로, 워크로드 특성에 맞게 선택하는 것이 중요합니다.

## 오답 포인트

- A: Object Storage와 File Storage의 특성이 서로 뒤바뀐 설명으로, 각 서비스의 프로토콜과 저장 방식이 반대로 기술되어 있습니다.
- C: NFS 프로토콜은 File Storage만의 특성이며, Object Storage는 NFS를 사용하지 않습니다.
- D: 두 서비스는 저장 구조와 프로토콜이 상이하며, 같은 키-값 방식을 공유하지 않습니다.
