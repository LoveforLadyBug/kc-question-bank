## 문제

카카오클라우드 Object Storage의 확장성(Scalability)이 실제 운영 환경에서 의미하는 바로 가장 적절한 것은?

## 보기

- A. 고정된 용량 내에서 데이터를 빠르게 압축하여 저장 공간을 절약한다.
- B. 저장 용량을 필요에 따라 유연하게 늘릴 수 있어 대용량 데이터를 수용한다.
- C. 단일 서버에서 처리 속도를 수직으로 확장하여 성능을 높인다.
- D. 특정 가용 영역에 고정 배치되어 일관된 응답 속도를 보장한다.

## 정답

B

## 해설

Object Storage의 확장성은 저장 용량을 사전에 고정하지 않고, 데이터 증가에 따라 유연하게 수용할 수 있음을 의미합니다.
이는 대용량 비정형 데이터를 다루는 환경에서 특히 중요하며, 사용량에 따라 동적으로 스케일아웃이 가능한 구조를 뒷받침합니다.

## 오답 포인트

- A: 데이터 압축을 통한 공간 절약은 확장성의 개념이 아니라 압축 기술에 해당합니다.
- C: 단일 서버 내 수직 확장(Scale-up)은 Object Storage의 확장성 개념과 다르며, 분산 구조에 반합니다.
- D: 특정 가용 영역 고정 배치는 확장성이 아닌 가용성 구성 개념에 해당합니다.
