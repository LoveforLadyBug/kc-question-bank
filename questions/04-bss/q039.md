## 문제

카카오클라우드 File Storage에서 NFS 프로토콜을 사용하는 이유로 가장 적절한 것은?

## 보기

- A. 블록 수준의 낮은 지연 시간과 높은 처리량을 확보하기 위해서이다.
- B. 분산 파일 시스템 환경에서 여러 서버가 네트워크를 통해 동일한 파일에 접근할 수 있게 하기 위해서이다.
- C. 데이터를 키-값 형태로 변환하여 빠른 검색 성능을 제공하기 위해서이다.
- D. 인터넷을 통해 불특정 다수에게 파일을 공개 배포하기 위해서이다.

## 정답

B

## 해설

NFS는 네트워크 파일 시스템 프로토콜로, 분산 파일 시스템 환경에서 여러 서버와 애플리케이션이 동일한 파일 시스템에 동시에 접근할 수 있도록 지원합니다.
카카오클라우드 File Storage는 이 프로토콜을 채택하여 다수의 서버 간 파일 공유 및 협업 워크로드를 지원합니다.

## 오답 포인트

- A: 블록 수준 저지연·고처리량은 NFS가 아닌 블록 스토리지 기술의 목적입니다.
- C: 키-값 변환 및 검색 성능은 Object Storage의 특성이며 NFS와 관련이 없습니다.
- D: 불특정 다수에게 공개 배포하는 것은 CDN이나 Object Storage의 사용 사례에 해당합니다.
