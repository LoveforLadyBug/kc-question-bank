## 문제

카카오클라우드 스토리지 서비스 중 대용량 데이터를 객체 단위로 저장하는 데 최적화된 서비스는?

## 보기

- A. File Storage
- B. Block Storage
- C. Object Storage
- D. Archive Storage

## 정답

C

## 해설

Object Storage는 대용량 데이터를 키-값(Key-value) 형태의 객체로 저장하고 처리하는 데 최적화된 서비스입니다.
높은 확장성과 안정성을 제공하여 이미지, 동영상, 로그 파일 등 다양한 비정형 대용량 데이터 보관에 적합합니다.

## 오답 포인트

- A: File Storage는 NFS 프로토콜 기반의 파일 공유 서비스이며, 객체 단위 저장 최적화 서비스가 아닙니다.
- B: Block Storage는 블록 단위 고속 I/O를 제공하는 서비스로, 객체 기반 저장 방식과 다릅니다.
- D: Archive Storage는 장기 보관용 콜드 스토리지 개념으로, 본 레퍼런스 문서에서 언급된 서비스와 다릅니다.
