## 문제

카카오클라우드 Object Storage에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 블록 단위로 데이터를 분할하여 낮은 지연 시간의 입출력을 제공한다.
- B. NFS 프로토콜을 기반으로 여러 서버 간 파일을 동기화한다.
- C. 대용량 데이터를 객체(Key-value) 형태로 저장 및 처리하는 데 최적화된 서비스이다.
- D. 관계형 데이터베이스처럼 테이블 구조로 데이터를 관리한다.

## 정답

C

## 해설

카카오클라우드 Object Storage는 대용량 데이터를 객체(Key-value) 형태로 저장·처리하는 데 최적화된 확장성과 안정성을 갖춘 서비스입니다.
블록 스토리지나 파일 스토리지와 달리 키-값 구조를 채택하여 대용량 비정형 데이터 보관에 적합합니다.

## 오답 포인트

- A: 블록 단위 저장 및 낮은 지연 시간은 블록 스토리지(Block Storage)의 특징이며 Object Storage와는 다릅니다.
- B: NFS 프로토콜 기반 파일 공유는 File Storage의 특징입니다.
- D: 테이블 구조의 데이터 관리는 관계형 데이터베이스(RDBMS)의 특징으로, Object Storage와 무관합니다.
