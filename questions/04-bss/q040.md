## 문제

카카오클라우드의 스토리지 서비스를 선택할 때, 각 서비스의 핵심 차별점을 올바르게 설명한 것은?

## 보기

- A. Object Storage는 네트워크 파일 공유에, File Storage는 키-값 객체 저장에 특화되어 있다.
- B. Object Storage와 File Storage는 동일한 프로토콜을 사용하며 용량 차이만 존재한다.
- C. Object Storage는 대용량 데이터의 객체 저장·처리에, File Storage는 NFS를 통한 다중 서버 파일 공유에 특화되어 있다.
- D. Object Storage는 트랜잭션 처리에 강하고, File Storage는 장기 백업 보관에 강하다.

## 정답

C

## 해설

Object Storage는 키-값 기반의 확장성 높은 객체 저장소로 대용량 데이터 처리에 최적화되어 있고, File Storage는 NFS 프로토콜을 기반으로 여러 서버가 파일을 공유하는 데 특화되어 있습니다.
두 서비스의 저장 방식, 접근 프로토콜, 최적 사용 사례를 구분하는 것이 올바른 서비스 선택의 핵심입니다.

## 오답 포인트

- A: Object Storage와 File Storage의 특화 영역이 서로 바뀐 오답입니다.
- B: 두 서비스는 프로토콜(키-값 vs NFS)과 목적이 명확히 다르며, 단순한 용량 차이가 아닙니다.
- D: Object Storage는 트랜잭션 처리 전용이 아니며, File Storage는 장기 백업 전용 서비스가 아닙니다.
