## 문제

카카오클라우드 File Storage가 분산 파일 공유를 위해 사용하는 네트워크 프로토콜은 무엇입니까?

## 보기

- A. HTTP(HyperText Transfer Protocol)
- B. iSCSI(Internet Small Computer Systems Interface)
- C. NFS(Network File System)
- D. SFTP(SSH File Transfer Protocol)

## 정답

C

## 해설

카카오클라우드 File Storage는 분산 파일 시스템에 적합한 네트워크 프로토콜인 NFS(Network File System)를 사용하여 여러 서버와 애플리케이션 간에 파일을 저장하고 공유합니다.
NFS는 유닉스/리눅스 환경에서 널리 사용되는 파일 공유 표준 프로토콜입니다.

## 오답 포인트

- A: HTTP는 웹 데이터 전송 프로토콜로, 파일 시스템 공유 목적으로 사용되지 않습니다.
- B: iSCSI는 블록 스토리지를 네트워크로 연결할 때 사용하는 프로토콜입니다.
- D: SFTP는 SSH 기반의 파일 전송 프로토콜로, 분산 파일 시스템 마운트 방식이 아닙니다.
