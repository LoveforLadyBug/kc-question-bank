## 문제

카카오클라우드 Object Storage의 특성으로 가장 적절한 것은?

## 보기

- A. 파일 잠금(File Locking) 기능을 통해 동시 편집 충돌을 방지하는 데 최적화되어 있다.
- B. 높은 확장성과 안정성을 바탕으로 대용량 데이터를 객체 기반으로 저장·처리한다.
- C. 단일 인스턴스에 전용으로 연결되어 빠른 블록 수준 I/O를 보장하는 구조이다.
- D. 계층적 디렉터리 구조를 반드시 사용해야 데이터를 저장할 수 있다.

## 정답

B

## 해설

카카오클라우드 Object Storage는 확장성과 안정성을 기반으로 대용량 데이터를 객체 형태로 저장·처리하는 데 최적화된 서비스입니다.
고정된 용량 제한 없이 데이터 증가에 유연하게 대응할 수 있으며, 비정형 데이터(이미지, 동영상, 백업 등)를 대규모로 보관하는 환경에 적합합니다.

## 오답 포인트

- A: 파일 잠금 기능은 파일 시스템 기반의 File Storage에서 다루는 특성이며, Object Storage의 핵심 특징이 아닙니다.
- C: 단일 인스턴스 전용 블록 수준 I/O는 Block Storage의 특성입니다.
- D: Object Storage는 계층적 디렉터리 구조가 아닌 키-값 기반으로 데이터를 저장하므로, 디렉터리 구조가 필수 조건이 아닙니다.
