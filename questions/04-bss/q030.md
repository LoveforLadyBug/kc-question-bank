## 문제

카카오클라우드 File Storage의 활용 사례로 가장 적합한 것은?

## 보기

- A. 수백만 건의 로그 파일을 장기 아카이빙하고 외부 URL로 다운로드 제공한다.
- B. 오토스케일링 환경에서 여러 WAS 인스턴스가 동일한 업로드 디렉터리를 공유한다.
- C. 단일 데이터베이스 서버의 트랜잭션 로그를 고속으로 기록하기 위해 사용한다.
- D. 머신러닝 모델 학습용 데이터셋을 키-값 형태로 변환하여 저장한다.

## 정답

B

## 해설

File Storage는 NFS 프로토콜을 통해 여러 서버와 애플리케이션이 동일한 파일을 공유할 수 있도록 설계된 서비스입니다. 오토스케일링 환경에서 다수의 WAS 인스턴스가 업로드 디렉터리를 함께 사용하는 시나리오는 File Storage의 대표적인 활용 사례입니다.
이처럼 다중 인스턴스 간 파일 공유가 필요한 분산 환경에서 File Storage의 네트워크 파일 시스템 기능이 핵심적인 역할을 합니다.

## 오답 포인트

- A: 대용량 로그 파일의 장기 아카이빙과 URL 기반 다운로드 제공은 Object Storage에 더 적합한 사용 사례입니다.
- C: 단일 데이터베이스 서버의 트랜잭션 로그를 고속으로 기록하는 것은 낮은 지연 시간과 높은 IOPS가 필요하므로 Block Storage가 적합합니다.
- D: 데이터셋을 키-값 형태로 변환하여 저장하는 방식은 Object Storage의 접근 방식이며, File Storage의 특성이 아닙니다.
