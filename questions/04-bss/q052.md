## 문제

카카오클라우드 File Storage가 다중 서버 간 파일 공유를 위해 채택한 네트워크 프로토콜은 무엇입니까?

## 보기

- A. iSCSI
- B. NFS(Network File System)
- C. HTTP/HTTPS
- D. SFTP(SSH File Transfer Protocol)

## 정답

B

## 해설

카카오클라우드 File Storage는 분산 파일 시스템을 위한 네트워크 프로토콜인 NFS(Network File System)를 사용하여 여러 서버와 애플리케이션 간에 동일한 파일을 공유합니다.
NFS는 네트워크를 통해 원격 디렉터리를 로컬처럼 마운트할 수 있어, 다중 인스턴스 환경에서 공유 스토리지를 구성하는 데 적합합니다.

## 오답 포인트

- A: iSCSI는 블록 스토리지를 네트워크로 연결하는 프로토콜로, 파일 공유가 아닌 블록 수준의 접근에 사용됩니다.
- C: HTTP/HTTPS는 웹 통신 프로토콜로, Object Storage API 접근 등에 활용되지만 File Storage의 마운트 프로토콜이 아닙니다.
- D: SFTP는 SSH 기반의 파일 전송 프로토콜로, 분산 파일 시스템의 공유 마운트 방식과는 다릅니다.
