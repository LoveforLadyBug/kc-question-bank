## 문제

카카오클라우드 File Storage의 활용 사례로 가장 적합한 것은 무엇입니까?

## 보기

- A. 단일 데이터베이스 서버가 전용 고속 디스크로 트랜잭션 로그를 기록하는 경우
- B. 여러 WAS(Web Application Server) 인스턴스가 공통 설정 파일과 로그를 함께 읽고 쓰는 경우
- C. 수천만 개의 이미지 파일을 URL로 외부에 공개 서비스하는 경우
- D. 머신러닝 모델 학습 결과를 키-값 형태로 저장하여 빠르게 조회하는 경우

## 정답

B

## 해설

File Storage는 NFS 프로토콜을 통해 여러 서버와 애플리케이션이 동일한 파일 공간을 공유하는 데 최적화되어 있습니다.
여러 WAS 인스턴스가 공통 설정 파일이나 로그 디렉터리를 함께 사용하는 구성이 File Storage의 전형적인 활용 시나리오입니다.

## 오답 포인트

- A: 전용 고속 디스크를 단일 서버가 독점 사용하는 형태는 Block Storage의 활용 사례에 해당합니다.
- C: 수천만 개의 파일을 URL로 외부에 공개하는 대용량 서비스는 Object Storage에 적합한 사례입니다.
- D: 키-값 형태로 데이터를 저장하고 빠르게 조회하는 것은 Object Storage 또는 NoSQL 데이터베이스의 활용 사례입니다.
