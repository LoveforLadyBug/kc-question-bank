## 문제

카카오클라우드 File Storage가 다수의 서버와 애플리케이션 간 파일 공유를 위해 사용하는 네트워크 프로토콜은 무엇인가?

## 보기

- A. iSCSI(Internet Small Computer Systems Interface)
- B. NFS(Network File System)
- C. SFTP(SSH File Transfer Protocol)
- D. HTTP(HyperText Transfer Protocol)

## 정답

B

## 해설

카카오클라우드 File Storage는 분산 파일 시스템을 위한 네트워크 프로토콜인 NFS를 사용하여 여러 서버와 애플리케이션 간 데이터를 저장하고 공유합니다.
NFS는 원격 호스트의 파일 시스템을 로컬처럼 마운트하여 동시 접근이 가능하도록 지원하므로, 다중 인스턴스 환경에서의 파일 공유에 적합합니다.

## 오답 포인트

- A: iSCSI는 블록 스토리지를 네트워크로 연결하는 프로토콜로, 파일 공유가 아닌 블록 수준 접근에 사용됩니다.
- C: SFTP는 SSH 기반의 파일 전송 프로토콜로, 분산 파일 시스템 마운트 목적으로는 사용되지 않습니다.
- D: HTTP는 웹 데이터 전송을 위한 프로토콜로, 파일 시스템 공유 프로토콜이 아닙니다.
