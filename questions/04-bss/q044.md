## 문제

카카오클라우드 File Storage의 주요 사용 목적으로 가장 적절한 것은?

## 보기

- A. 대용량 오브젝트를 키-값 형태로 분산 보관한다.
- B. 단일 인스턴스의 OS 부팅 볼륨으로 사용한다.
- C. 여러 서버와 애플리케이션 간에 파일을 저장하고 공유한다.
- D. 관계형 데이터베이스의 트랜잭션 로그를 기록한다.

## 정답

C

## 해설

File Storage는 네트워크 프로토콜(NFS)을 활용하여 다수의 서버와 애플리케이션이 동일한 파일에 접근하고 공유할 수 있도록 하는 서비스입니다.
단일 인스턴스에 종속되지 않고, 여러 클라이언트가 동시에 동일한 스토리지를 사용할 수 있다는 점이 핵심 특징입니다.

## 오답 포인트

- A: 키-값 형태의 분산 보관은 Object Storage의 특성이며, File Storage와는 구별됩니다.
- B: OS 부팅 볼륨은 일반적으로 Block Storage에 해당하며, File Storage의 주요 용도가 아닙니다.
- D: 데이터베이스 트랜잭션 로그 기록은 Block Storage 또는 전용 데이터베이스 스토리지의 역할에 가깝습니다.
