## 문제

카카오클라우드 File Storage가 분산 파일 시스템 구현을 위해 사용하는 네트워크 프로토콜은?

## 보기

- A. SMB(Server Message Block)
- B. FTP(File Transfer Protocol)
- C. NFS(Network File System)
- D. iSCSI(Internet Small Computer Systems Interface)

## 정답

C

## 해설

카카오클라우드 File Storage는 NFS(Network File System) 프로토콜을 사용하여 분산 파일 시스템 환경을 구현합니다.
NFS는 네트워크를 통해 여러 서버와 애플리케이션이 동일한 파일 시스템을 공유할 수 있게 해주는 표준 프로토콜입니다.

## 오답 포인트

- A: SMB는 주로 Windows 환경에서 파일 공유에 사용되는 프로토콜로, 카카오클라우드 File Storage의 프로토콜이 아닙니다.
- B: FTP는 파일 전송에 사용되는 프로토콜이며, 분산 파일 시스템용 공유 프로토콜로는 적합하지 않습니다.
- D: iSCSI는 블록 스토리지를 네트워크로 연결하는 프로토콜로, 파일 공유 목적과 다릅니다.
