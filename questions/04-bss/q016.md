## 문제

카카오클라우드 Object Storage가 속하는 스토리지 서비스 그룹(카테고리)은 무엇입니까?

## 보기

- A. Compute Service
- B. Network Service
- C. Beyond Storage Service
- D. Database Service

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면 Object Storage는 File Storage와 함께 'Beyond Storage Service' 그룹에 속하며, 높은 확장성과 성능, 안정성을 기반으로 데이터를 저장 및 처리합니다.
Beyond Storage Service는 카카오클라우드의 스토리지 서비스 그룹으로, 다양한 스토리지 요구사항을 충족하기 위한 서비스를 포함합니다.

## 오답 포인트

- A: Compute Service는 가상 머신, 컨테이너 등 연산 자원을 제공하는 서비스 그룹입니다.
- B: Network Service는 VPC, 로드밸런서 등 네트워킹 관련 서비스를 포함하는 그룹입니다.
- D: Database Service는 관계형 또는 NoSQL 데이터베이스를 제공하는 서비스 그룹입니다.
