## 문제

카카오클라우드 File Storage가 네트워크를 통한 파일 공유에 사용하는 프로토콜은?

## 보기

- A. iSCSI(Internet Small Computer System Interface)
- B. NFS(Network File System)
- C. HTTP(HyperText Transfer Protocol)
- D. SFTP(SSH File Transfer Protocol)

## 정답

B

## 해설

카카오클라우드 File Storage는 분산 파일 시스템을 위한 네트워크 프로토콜인 NFS(Network File System)를 사용합니다.
NFS를 통해 여러 서버와 애플리케이션이 동일한 스토리지에 동시에 접근하고 파일을 공유할 수 있습니다.

## 오답 포인트

- A: iSCSI는 블록 스토리지를 네트워크로 연결하는 프로토콜로, File Storage의 파일 공유 방식과 다릅니다.
- C: HTTP는 웹 콘텐츠 전송 프로토콜이며, 분산 파일 시스템용 공유 프로토콜로 사용되지 않습니다.
- D: SFTP는 SSH 기반 파일 전송 프로토콜로, File Storage의 공유 프로토콜과는 다릅니다.
