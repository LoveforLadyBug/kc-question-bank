## 문제

카카오클라우드 File Storage의 주된 목적으로 가장 적절한 것은 무엇입니까?

## 보기

- A. 단일 인스턴스 전용 블록 디스크 제공
- B. 대용량 비정형 데이터를 객체 단위로 보관
- C. 여러 서버와 애플리케이션 간 네트워크 파일 공유
- D. 인메모리 기반의 고속 캐시 데이터 저장

## 정답

C

## 해설

File Storage는 NFS 프로토콜을 활용하여 다수의 서버와 애플리케이션이 동일한 파일 시스템에 동시에 접근하고 데이터를 공유할 수 있도록 설계된 서비스입니다.
단일 인스턴스 전용이 아니라 여러 클라이언트가 동시에 사용할 수 있다는 점이 Block Storage와의 핵심 차이입니다.

## 오답 포인트

- A: 단일 인스턴스 전용 블록 디스크 제공은 Block Storage의 특징이며, File Storage는 다중 접근이 핵심입니다.
- B: 객체 단위 대용량 데이터 보관은 Object Storage의 역할로, File Storage의 디렉터리 기반 파일 공유와 다릅니다.
- D: 인메모리 고속 캐시는 Redis 등 캐시 서비스의 기능으로, 스토리지 서비스와는 구분됩니다.
