## 문제

다음 중 카카오클라우드 Object Storage의 특성을 가장 잘 나타내는 키워드 조합은?

## 보기

- A. 저지연 · 블록 I/O · 단일 인스턴스
- B. 확장성 · 안정성 · 대용량 객체 저장
- C. NFS · 파일 계층 · 다중 서버 공유
- D. 트랜잭션 · 정합성 · 관계형 구조

## 정답

B

## 해설

Object Storage의 핵심 특성은 높은 확장성과 안정성을 기반으로 대용량 데이터를 객체 단위로 저장·처리하는 것입니다.
이는 블록 I/O 중심의 단일 인스턴스 접근이나 NFS 기반 파일 공유와는 다른 방향성으로 설계된 서비스입니다.

## 오답 포인트

- A: 저지연과 블록 I/O는 Block Storage의 특성을 나타내는 키워드입니다.
- C: NFS와 파일 계층 구조를 통한 다중 서버 공유는 File Storage의 특성에 해당합니다.
- D: 트랜잭션, 정합성, 관계형 구조는 RDBMS(관계형 데이터베이스)의 특성이며, Object Storage와 무관합니다.
