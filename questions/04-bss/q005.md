## 문제

카카오클라우드 File Storage의 주요 사용 목적으로 가장 적절한 것은?

## 보기

- A. 단일 인스턴스에서 블록 수준의 고속 데이터 접근
- B. 여러 서버와 애플리케이션 간에 데이터를 저장하고 공유
- C. 키-값 방식의 대용량 비정형 데이터 보관
- D. 관계형 데이터베이스의 트랜잭션 로그 관리

## 정답

B

## 해설

File Storage는 다수의 서버와 애플리케이션 사이에서 데이터를 저장하고 공유하기 위한 서비스입니다.
NFS 프로토콜을 기반으로 동작하여 여러 클라이언트가 동시에 동일한 파일 시스템에 접근할 수 있습니다.

## 오답 포인트

- A: 블록 수준의 고속 데이터 접근은 블록 스토리지(Block Storage)의 특성이며, File Storage의 주목적이 아닙니다.
- C: 키-값 방식의 대용량 비정형 데이터 보관은 Object Storage의 특성에 해당합니다.
- D: 관계형 DB 트랜잭션 로그 관리는 데이터베이스 서비스의 역할이며, File Storage의 목적과 다릅니다.
