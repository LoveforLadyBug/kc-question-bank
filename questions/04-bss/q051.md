## 문제

카카오클라우드 Object Storage에서 데이터를 저장할 때 사용하는 기본 구조 단위는 무엇입니까?

## 보기

- A. 블록(Block)
- B. 파일(File)
- C. 객체(Object)
- D. 레코드(Record)

## 정답

C

## 해설

Object Storage는 데이터를 '객체(Object)' 단위로 저장하며, 각 객체는 데이터 본문과 메타데이터, 고유 식별자(Key)로 구성됩니다.
이 구조는 키-값(Key-value) 방식으로 대용량 데이터의 저장 및 검색에 최적화되어 있으며, 블록이나 파일 시스템 계층 구조와는 근본적으로 다릅니다.

## 오답 포인트

- A: 블록(Block)은 Block Storage에서 사용하는 저장 단위로, 낮은 지연 시간과 높은 IOPS가 필요한 환경에 적합합니다.
- B: 파일(File) 단위 저장은 File Storage의 방식이며, 디렉터리 계층 구조를 기반으로 합니다.
- D: 레코드(Record)는 관계형 데이터베이스에서 사용하는 단위로, 스토리지 서비스와는 무관합니다.
