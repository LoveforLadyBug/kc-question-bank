## 문제

다음 중 카카오클라우드 File Storage의 사용 사례로 가장 적절한 것은?

## 보기

- A. 가상 머신의 부트 디스크로 사용하여 OS를 빠르게 로드한다.
- B. 멀티미디어 파일을 키-값 구조로 저장하여 전 세계에 배포한다.
- C. 복수의 애플리케이션 서버가 동일한 설정 파일 디렉터리를 공유한다.
- D. 블록 단위 I/O로 데이터베이스 트랜잭션 성능을 극대화한다.

## 정답

C

## 해설

File Storage는 NFS 프로토콜을 통해 여러 서버와 애플리케이션이 동일한 파일 시스템을 공유할 수 있게 해주므로, 복수의 서버가 공통 설정 파일이나 공유 데이터를 함께 사용하는 시나리오에 적합합니다.
분산 환경에서 파일 공유가 필요한 워크로드가 File Storage의 대표적인 사용 사례입니다.

## 오답 포인트

- A: 가상 머신 부트 디스크는 블록 스토리지의 전형적인 사용 사례입니다.
- B: 키-값 구조의 멀티미디어 파일 저장 및 배포는 Object Storage의 사용 사례입니다.
- D: 블록 단위 I/O를 통한 데이터베이스 성능 최적화는 블록 스토리지의 역할입니다.
