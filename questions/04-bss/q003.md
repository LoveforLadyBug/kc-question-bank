## 문제

카카오클라우드 Object Storage의 핵심 설계 원칙으로 가장 적절하게 설명한 것은?

## 보기

- A. 낮은 지연 시간과 높은 IOPS를 위한 블록 기반 아키텍처
- B. NFS 프로토콜을 활용한 다중 서버 간 파일 동기화
- C. 확장성과 안정성을 기반으로 한 객체 기반 데이터 저장
- D. 단일 가용 영역 내 고성능 로컬 캐시 제공

## 정답

C

## 해설

Object Storage는 확장성(Scalability)과 안정성(Reliability)을 핵심 설계 원칙으로 삼아, 대용량 데이터를 객체 형태로 저장하는 서비스입니다.
이 두 가지 특성은 서비스 소개 문서에서 명시적으로 강조되며, 비정형 데이터를 대규모로 운영하는 환경에서 특히 유용합니다.

## 오답 포인트

- A: 낮은 지연 시간과 높은 IOPS는 블록 스토리지의 설계 목표이며, Object Storage의 핵심 원칙이 아닙니다.
- B: NFS 프로토콜을 이용한 다중 서버 파일 공유는 File Storage의 특징입니다.
- D: 단일 가용 영역 내 로컬 캐시 제공은 Object Storage의 설계 개념과 다릅니다.
