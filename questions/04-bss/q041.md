## 문제

카카오클라우드 Object Storage는 데이터를 어떤 방식으로 저장하고 처리하는 서비스인가?

## 보기

- A. 디렉터리 계층 구조를 기반으로 파일을 저장한다.
- B. 고정된 블록 단위로 데이터를 분할하여 저장한다.
- C. 객체(Key-value) 형태로 데이터를 저장하고 처리한다.
- D. 관계형 테이블 구조로 데이터를 정규화하여 저장한다.

## 정답

C

## 해설

카카오클라우드 Object Storage는 데이터를 객체(Key-value) 형태로 저장하고 처리하는 데 최적화된 서비스입니다.
이 방식은 대용량 데이터를 유연하게 관리하기 유리하며, 파일 시스템의 계층 구조나 블록 단위 분할과는 구별되는 핵심 특성입니다.

## 오답 포인트

- A: 디렉터리 계층 구조는 파일 시스템(File Storage)의 특성으로, Object Storage와는 다른 저장 방식입니다.
- B: 블록 단위 저장은 Block Storage의 특성이며, Object Storage는 블록이 아닌 객체 단위로 데이터를 다룹니다.
- D: 관계형 테이블 구조는 데이터베이스(RDBMS)의 저장 방식으로, Object Storage와 무관합니다.
