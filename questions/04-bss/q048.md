## 문제

카카오클라우드 스토리지 서비스에서 "분산 파일 시스템을 위한 네트워크 프로토콜"을 기반으로 동작하는 서비스와 "객체 기반 저장 방식"을 사용하는 서비스를 순서대로 바르게 짝지은 것은?

## 보기

- A. Object Storage — File Storage
- B. File Storage — Object Storage
- C. Block Storage — File Storage
- D. File Storage — Block Storage

## 정답

B

## 해설

분산 파일 시스템을 위한 네트워크 프로토콜(NFS)을 기반으로 동작하는 것은 File Storage이고, 객체(Key-value) 기반 저장 방식을 사용하는 것은 Object Storage입니다.
두 서비스는 저장 구조와 접근 방식이 명확히 다르며, 각각의 특성에 맞는 워크로드에 사용해야 합니다.

## 오답 포인트

- A: Object Storage와 File Storage의 특성이 서로 뒤바뀐 선택지입니다.
- C: Block Storage는 NFS 프로토콜 기반 서비스가 아니며, 블록 단위 I/O를 제공하는 별도 서비스입니다.
- D: Block Storage는 객체 기반 저장 방식을 사용하지 않으며, File Storage와의 조합도 잘못된 설명입니다.
