## 문제

카카오클라우드 File Storage를 선택해야 하는 시나리오로 가장 적절한 것은?

## 보기

- A. 머신러닝 모델 학습용 대용량 이미지 파일을 키-값 형태로 보관할 때
- B. 가상 머신의 루트 볼륨으로 사용하여 OS를 부팅해야 할 때
- C. 여러 WAS(웹 애플리케이션 서버) 인스턴스가 동일한 설정 파일을 공유해야 할 때
- D. 초당 수만 건의 트랜잭션을 처리하는 데이터베이스 스토리지가 필요할 때

## 정답

C

## 해설

File Storage는 NFS 프로토콜을 통해 여러 서버와 애플리케이션이 동일한 파일에 동시에 접근할 수 있도록 지원하므로, 다수의 WAS 인스턴스가 공통 파일을 공유하는 시나리오에 가장 적합합니다.
파일 공유가 필요한 분산 환경에서 File Storage는 단일 스토리지 엔드포인트로 다중 클라이언트를 지원합니다.

## 오답 포인트

- A: 키-값 형태의 대용량 파일 보관은 Object Storage의 적합한 사용 사례입니다.
- B: OS 부팅을 위한 루트 볼륨은 Block Storage의 전형적인 사용 시나리오입니다.
- D: 고성능 트랜잭션 처리를 위한 데이터베이스 스토리지는 Block Storage 또는 전용 DB 스토리지가 적합합니다.
