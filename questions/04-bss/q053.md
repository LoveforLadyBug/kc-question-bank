## 문제

카카오클라우드의 스토리지 서비스 중 **확장성과 안정성**을 바탕으로 대용량 데이터를 키-값(Key-value) 형태로 저장하는 서비스는 무엇입니까?

## 보기

- A. File Storage
- B. Block Storage
- C. Object Storage
- D. Archive Storage

## 정답

C

## 해설

Object Storage는 높은 확장성과 안정성을 기반으로 대용량 데이터를 키-값(Key-value) 구조의 객체 단위로 저장하고 처리하는 데 최적화된 서비스입니다.
File Storage는 NFS 기반의 파일 공유, Block Storage는 블록 단위 입출력에 특화되어 있어 Object Storage와 용도가 다릅니다.

## 오답 포인트

- A: File Storage는 NFS 프로토콜을 사용하여 여러 서버 간 파일을 공유하는 서비스로, 키-값 구조와는 무관합니다.
- B: Block Storage는 블록 단위 입출력이 필요한 OS 디스크나 DB 환경에 적합하며, 키-값 기반 객체 저장 방식을 사용하지 않습니다.
- D: Archive Storage는 기존 문제 보기에 없는 서비스이며, 카카오클라우드 공식 문서에서 Object Storage와 구분되는 대표 서비스가 아닙니다.
