## 문제

대규모 미디어 스트리밍 플랫폼을 운영 중인 팀이 두 가지 요구사항을 검토하고 있습니다.  
① 수백만 개의 동영상 파일을 장기 보관하며 외부에서 접근  
② 인코딩 작업 중 여러 워커 인스턴스가 동시에 임시 작업 파일을 읽고 쓰기  

각 요구사항에 적합한 카카오클라우드 스토리지 서비스를 순서대로 올바르게 짝지은 것은?

## 보기

- A. ① File Storage, ② Object Storage
- B. ① Object Storage, ② File Storage
- C. ① Object Storage, ② Object Storage
- D. ① File Storage, ② File Storage

## 정답

B

## 해설

① 대량의 동영상 파일을 장기 보관하고 외부에서 접근하는 용도는 확장성과 안정성을 갖춘 Object Storage가 적합합니다.
② 여러 워커 인스턴스가 동시에 임시 작업 파일을 읽고 쓰는 공유 파일 시스템 환경은 NFS 기반의 File Storage가 적합합니다.

## 오답 포인트

- A: 두 서비스의 적합한 역할이 서로 바뀐 선택으로, File Storage는 대규모 외부 객체 보관에 최적화되어 있지 않습니다.
- C: Object Storage는 다수 인스턴스가 파일 시스템처럼 동시에 읽고 쓰는 공유 작업 환경에 최적화되어 있지 않습니다.
- D: File Storage만으로는 수백만 개의 비정형 대용량 객체를 장기 보관하는 시나리오를 효율적으로 처리하기 어렵습니다.
