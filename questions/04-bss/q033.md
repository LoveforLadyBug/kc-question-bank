## 문제

카카오클라우드 File Storage의 주요 목적으로 가장 적절한 것은?

## 보기

- A. 단일 서버에서 운영체제 부트 볼륨을 고속으로 마운트한다.
- B. 여러 서버와 애플리케이션이 네트워크를 통해 데이터를 저장하고 공유한다.
- C. 데이터를 키-값 형태로 저장하여 대용량 비정형 파일을 관리한다.
- D. 블록 단위 I/O를 통해 데이터베이스의 높은 성능을 보장한다.

## 정답

B

## 해설

카카오클라우드 File Storage는 NFS 프로토콜을 활용해 다수의 서버와 애플리케이션 간 파일을 저장하고 공유하기 위한 서비스입니다.
단일 서버 전용이 아닌 분산 환경에서 파일 시스템을 공유하는 것이 핵심 목적입니다.

## 오답 포인트

- A: 단일 서버 부트 볼륨 마운트는 블록 스토리지의 사용 사례에 해당합니다.
- C: 키-값 형태의 대용량 비정형 파일 관리는 Object Storage의 특징입니다.
- D: 블록 단위 고성능 I/O는 블록 스토리지(Block Storage)의 목적입니다.
