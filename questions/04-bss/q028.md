## 문제

카카오클라우드 File Storage에서 NFS 프로토콜을 사용하는 이유로 가장 적절한 것은?

## 보기

- A. 데이터를 암호화하여 외부 침입으로부터 파일을 보호하기 위해서이다.
- B. 여러 서버와 애플리케이션이 네트워크를 통해 동일한 파일 시스템에 접근하도록 하기 위해서이다.
- C. 파일을 키-값 형태로 변환하여 검색 성능을 향상시키기 위해서이다.
- D. 블록 단위로 데이터를 분할하여 전송 속도를 최대화하기 위해서이다.

## 정답

B

## 해설

NFS(Network File System)는 분산 파일 시스템을 위한 네트워크 프로토콜로, 원격의 여러 서버와 애플리케이션이 동일한 파일 시스템을 공유·접근할 수 있게 해줍니다.
카카오클라우드 File Storage는 이 특성을 활용하여 다중 인스턴스 환경에서 파일 데이터를 공동으로 저장하고 공유하는 기능을 제공합니다.

## 오답 포인트

- A: 암호화는 NFS의 주요 목적이 아니며, 보안 암호화는 별도의 보안 레이어에서 처리됩니다.
- C: 파일을 키-값으로 변환하여 검색 성능을 높이는 것은 Object Storage의 접근 방식과 관련된 개념이며, NFS의 사용 목적이 아닙니다.
- D: 블록 단위 분할을 통한 전송 속도 최대화는 블록 스토리지 또는 iSCSI와 관련된 개념으로, NFS의 주요 사용 목적이 아닙니다.
