## 문제

카카오클라우드 버스터블 인스턴스(t1i)에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 전용 물리 서버에서만 실행되며 다른 고객과 자원을 공유하지 않는다.
- B. CPU 크레딧을 축적하여 필요한 시점에 높은 성능을 발휘할 수 있는 인스턴스이다.
- C. GPU 연산에 특화된 고성능 컴퓨팅 인스턴스 유형이다.
- D. 컨테이너 이미지를 직접 실행하는 서버리스 방식으로 동작한다.

## 정답

B

## 해설

버스터블 인스턴스(t1i)는 평소에 CPU 크레딧을 축적하고, 높은 처리 성능이 필요한 순간에 해당 크레딧을 사용하여 일시적으로 성능을 높일 수 있는 동작 방식을 가집니다.
이 방식은 평균 CPU 사용률이 낮지만 간헐적으로 높은 성능이 필요한 워크로드에 비용 효율적으로 적합합니다.

## 오답 포인트

- A: 버스터블 인스턴스는 전용 물리 서버 방식이 아니라 가상화 환경에서 동작하며, 자원 공유 방식으로 운영됩니다.
- C: GPU 특화 인스턴스는 별도의 GPU 인스턴스 유형에 해당하며, t1i는 범용 버스터블 인스턴스입니다.
- D: 서버리스 방식으로 컨테이너를 실행하는 것은 버스터블 인스턴스가 아닌 별도의 컨테이너 서비스에 해당합니다.
