## 문제

카카오클라우드에서 강력하고 유연한 클라우드 컴퓨팅 자원을 제공하는 서비스로, 다양한 워크로드에 맞춰 인스턴스 유형을 선택할 수 있는 서비스는?

## 보기

- A. Kubernetes Engine
- B. Data Catalog
- C. BCS 인스턴스
- D. Object Storage

## 정답

C

## 해설

BCS 인스턴스는 카카오클라우드의 핵심 컴퓨팅 서비스로, 강력하고 유연한 클라우드 컴퓨팅 자원을 제공합니다.
사용자는 워크로드 요구사항에 따라 다양한 인스턴스 유형(예: 버스터블 인스턴스 t1i 등)을 선택하여 사용할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 컨테이너 오케스트레이션을 위한 관리형 쿠버네티스 서비스로, 범용 컴퓨팅 자원 제공 서비스가 아닙니다.
- B: Data Catalog는 데이터 메타데이터를 통합 관리하는 데이터 관리 서비스입니다.
- D: Object Storage는 코드 기반 API를 통해 데이터를 저장하고 관리하는 스토리지 서비스입니다.
