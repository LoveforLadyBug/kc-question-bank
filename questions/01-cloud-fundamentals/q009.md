## 문제

카카오클라우드 Data Catalog 서비스에서 지원하는 주요 기능으로 옳은 것은?

## 보기

- A. 실시간 스트리밍 데이터를 처리하는 파이프라인 자동 구성
- B. Apache Iceberg 테이블 등록 및 메타데이터 통합 관리
- C. SQL 쿼리 기반의 데이터 웨어하우스 자동 프로비저닝
- D. 머신러닝 모델 학습을 위한 GPU 클러스터 자동 배포

## 정답

B

## 해설

카카오클라우드 Data Catalog는 Apache Iceberg 테이블을 손쉽게 등록하고 메타데이터를 통합하여 관리할 수 있는 서비스입니다.
이를 통해 다양한 데이터 소스의 메타데이터를 중앙에서 탐색하고 관리하는 데이터 거버넌스 기능을 지원합니다.

## 오답 포인트

- A: 실시간 스트리밍 파이프라인 자동 구성은 Data Catalog의 기능이 아니며, Kafka 등 스트리밍 서비스에 해당합니다.
- C: SQL 기반 데이터 웨어하우스 자동 프로비저닝은 별도의 데이터 웨어하우스 서비스의 기능입니다.
- D: GPU 클러스터 자동 배포는 AI/ML 플랫폼(Kubeflow 등) 관련 서비스의 기능입니다.
