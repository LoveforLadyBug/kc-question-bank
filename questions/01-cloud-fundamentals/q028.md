## 문제

카카오클라우드 Object Storage의 접근 방식에 대한 설명으로 올바른 것은?

## 보기

- A. iSCSI 프로토콜을 통해 블록 디바이스로만 마운트하여 사용한다.
- B. NFS 프로토콜 기반의 파일시스템으로만 접근 가능하다.
- C. 코드 기반의 API를 통해 프로그래밍 방식으로 접근하고 관리할 수 있다.
- D. SSH 터널링을 통해서만 데이터를 업로드하고 다운로드할 수 있다.

## 정답

C

## 해설

카카오클라우드 Object Storage는 코드 기반의 API를 제공하여 사용자가 프로그래밍을 통해 데이터에 접근하고 관리할 수 있도록 지원합니다.
이를 통해 애플리케이션과의 손쉬운 연동이 가능하며, HTTP 기반의 RESTful API를 활용한 유연한 데이터 관리가 가능합니다.

## 오답 포인트

- A: iSCSI 기반 블록 디바이스 마운트는 Block Storage의 접근 방식이며, Object Storage는 이와 다른 방식으로 동작합니다.
- B: NFS 프로토콜 기반 파일시스템 접근은 File Storage에 해당하는 방식입니다.
- D: SSH 터널링 전용 방식은 Object Storage의 공식 접근 방법이 아닙니다.
