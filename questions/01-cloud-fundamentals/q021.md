## 문제

카카오클라우드를 처음 시작할 때, VPC를 생성하기 직전에 반드시 완료해야 하는 단계로 올바른 것은?

## 보기

- A. 스냅샷 생성 및 이미지 등록
- B. 프로젝트 생성 및 프로젝트 역할 추가
- C. 보안 그룹 규칙 설정 및 인바운드 허용
- D. Kubernetes Engine 클러스터 생성

## 정답

B

## 해설

카카오클라우드 시작 절차는 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성 순서로 진행됩니다.
따라서 VPC 생성 직전 단계는 프로젝트 생성 및 프로젝트 역할 추가이며, 이 단계가 완료되어야 VPC를 생성할 수 있습니다.

## 오답 포인트

- A: 스냅샷 생성 및 이미지 등록은 인스턴스가 이미 운영 중인 상태에서 수행하는 작업으로, 초기 시작 절차에 포함되지 않습니다.
- C: 보안 그룹 규칙 설정은 VPC 및 인스턴스 생성 이후에 진행하는 네트워크 보안 설정 단계입니다.
- D: Kubernetes Engine 클러스터 생성은 VPC가 이미 구성된 이후에 가능한 작업입니다.
