## 문제

카카오클라우드 버스터블 인스턴스(t1i)의 특징으로 가장 적절한 것은?

## 보기

- A. 항상 최대 CPU 성능을 보장하며 크레딧 제도를 사용하지 않는다.
- B. CPU 크레딧을 사용하여 필요 시 성능을 높일 수 있는 동작 방식을 가진다.
- C. GPU 연산에 특화된 인스턴스 유형으로 딥러닝 학습에 주로 사용된다.
- D. 스토리지 집약적 워크로드를 위해 고성능 SSD만을 로컬 디스크로 제공한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 버스터블 인스턴스(t1i)는 CPU 크레딧 기반의 동작 방식을 가지며, 평소에 크레딧을 적립하고 필요 시 버스트하여 성능을 높일 수 있습니다.
이 방식은 평균 CPU 사용률이 낮으면서 간헐적으로 높은 성능이 필요한 워크로드에 비용 효율적으로 적합합니다.

## 오답 포인트

- A: 버스터블 인스턴스는 항상 최대 성능을 보장하지 않으며, 크레딧 제도가 핵심 동작 원리입니다.
- C: t1i는 GPU 특화 인스턴스가 아니며, GPU 연산은 별도의 GPU 인스턴스 유형에서 지원됩니다.
- D: 버스터블 인스턴스는 스토리지 집약 워크로드 전용 인스턴스가 아니며, 일반 컴퓨팅 용도로 사용됩니다.
