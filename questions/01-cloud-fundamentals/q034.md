## 문제

카카오클라우드 Object Storage에 대한 설명으로 가장 적절한 것은 무엇인가요?

## 보기

- A. 전용 파일시스템 마운트가 필수이며 NFS 프로토콜로만 접근할 수 있다.
- B. 코드 기반의 API를 제공하여 프로그래밍을 통해 접근하고 관리할 수 있다.
- C. 블록 디바이스 형태로만 제공되며 인스턴스에 직접 연결해야 한다.
- D. 단일 가용 영역에만 데이터를 저장하므로 영역 장애 시 데이터가 손실된다.

## 정답

B

## 해설

카카오클라우드 Object Storage는 코드 기반의 API를 제공하여 사용자가 프로그래밍을 통해 데이터를 저장, 조회, 삭제 등의 작업을 수행할 수 있는 오브젝트 스토리지 서비스입니다.
HTTP 기반 RESTful API 방식으로 접근하므로, 별도의 파일시스템 마운트나 블록 디바이스 연결 없이도 다양한 애플리케이션에서 유연하게 활용할 수 있습니다.

## 오답 포인트

- A: Object Storage는 NFS 전용이 아니며, API를 통한 HTTP 방식으로 접근합니다.
- C: Object Storage는 블록 디바이스 방식이 아닌 오브젝트 방식으로 데이터를 저장합니다.
- D: Object Storage는 내구성과 가용성을 위해 다중 복제 저장 방식을 채택하고 있습니다.
