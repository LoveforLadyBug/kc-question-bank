## 문제

카카오클라우드 Object Storage에 대한 설명으로 옳은 것은?

## 보기

- A. 블록 디바이스 방식으로만 데이터에 접근할 수 있다.
- B. NFS 프로토콜을 통해서만 파일을 읽고 쓸 수 있다.
- C. 코드 기반의 API를 통해 프로그래밍으로 접근하고 관리할 수 있다.
- D. 단일 인스턴스에만 마운트 가능한 로컬 스토리지 서비스이다.

## 정답

C

## 해설

카카오클라우드 Object Storage는 코드 기반의 API를 제공하여, 사용자가 프로그래밍을 통해 데이터에 접근하고 관리할 수 있습니다.
이를 통해 애플리케이션에서 자동화된 방식으로 파일 업로드, 다운로드, 삭제 등 다양한 스토리지 작업을 수행할 수 있습니다.

## 오답 포인트

- A: Object Storage는 블록 디바이스 방식이 아닌 객체(Object) 단위로 데이터를 관리합니다.
- B: NFS 전용이 아니라 HTTP 기반 REST API를 통해 접근하는 방식을 사용합니다.
- D: Object Storage는 특정 인스턴스에 마운트하는 로컬 스토리지가 아닌 범용 분산 스토리지입니다.
