## 문제

카카오클라우드 Data Catalog 서비스에서 지원하는 주요 기능으로 올바른 것은?

## 보기

- A. 가상 머신의 CPU·메모리 사용률을 실시간 대시보드로 시각화한다.
- B. Apache Iceberg 테이블을 등록하고 메타데이터를 통합 관리한다.
- C. 블록 스토리지 볼륨을 인스턴스에 동적으로 연결하고 해제한다.
- D. 쿠버네티스 클러스터의 노드 오토스케일링을 자동으로 수행한다.

## 정답

B

## 해설

카카오클라우드 Data Catalog는 Apache Iceberg 테이블을 손쉽게 등록하고 메타데이터를 통합 관리할 수 있는 데이터 관리 서비스입니다.
이를 통해 데이터 자산의 검색·분류·관리가 용이해지며, 다양한 데이터 소스와의 연계를 지원합니다.

## 오답 포인트

- A: CPU·메모리 사용률 실시간 시각화는 Monitoring 서비스의 기능으로, Data Catalog와는 다른 서비스입니다.
- C: 블록 스토리지 볼륨 관리는 스토리지 서비스(Block Storage)에 해당하는 기능입니다.
- D: 쿠버네티스 노드 오토스케일링은 Kubernetes Engine 서비스의 기능입니다.
