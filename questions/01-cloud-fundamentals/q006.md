## 문제

카카오클라우드 Monitoring에서 서드파티 모니터링 도구로 메트릭 데이터를 전송하기 위해 활용하는 기능은?

## 보기

- A. VPC 피어링 엔드포인트
- B. Metric Export 엔드포인트
- C. Object Storage 버킷 이벤트
- D. IAM 역할 기반 토큰 발급

## 정답

B

## 해설

카카오클라우드 Monitoring은 Metric Export 엔드포인트를 제공하여 외부 또는 서드파티 모니터링 도구에서 메트릭 데이터를 조회하고 활용할 수 있도록 지원합니다.
이를 통해 Prometheus, Grafana 등 외부 시스템과의 연동이 가능합니다.

## 오답 포인트

- A: VPC 피어링 엔드포인트는 네트워크 연결을 위한 개념으로, 메트릭 데이터 전송과는 관련이 없습니다.
- C: Object Storage 버킷 이벤트는 스토리지 오브젝트 변경 감지에 사용되며 메트릭 익스포트 기능이 아닙니다.
- D: IAM 역할 기반 토큰 발급은 인증·인가 관련 기능이며 모니터링 데이터 전송 기능이 아닙니다.
