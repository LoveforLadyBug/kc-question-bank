## 문제

카카오클라우드 Data Catalog 서비스의 주요 기능으로 가장 적절한 것은?

## 보기

- A. 가상 머신 인스턴스의 CPU 및 메모리 사용량을 실시간으로 모니터링한다.
- B. VPC 내 네트워크 트래픽 흐름을 제어하고 방화벽 정책을 관리한다.
- C. Apache Iceberg 테이블을 등록하고 메타데이터를 통합 관리한다.
- D. Kubernetes 클러스터의 노드 그룹을 자동으로 스케일링한다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, Data Catalog 서비스는 Apache Iceberg 테이블을 쉽게 등록하고 메타데이터를 통합 관리하는 기능을 제공합니다.
Data Catalog는 다양한 데이터 소스의 메타데이터를 한 곳에서 관리하여 데이터 검색과 활용 효율성을 높이는 역할을 합니다.

## 오답 포인트

- A: 실시간 리소스 모니터링은 Monitoring 서비스의 기능이며, Data Catalog와는 다릅니다.
- B: 네트워크 트래픽 제어 및 방화벽 정책 관리는 VPC/보안 그룹 관련 서비스의 기능입니다.
- D: Kubernetes 클러스터의 노드 자동 스케일링은 Kubernetes Engine의 기능에 해당합니다.
