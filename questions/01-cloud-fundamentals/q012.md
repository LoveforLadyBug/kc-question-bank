## 문제

카카오클라우드에서 제공하는 클라우드 컴퓨트 자원으로, 강력하고 유연한 가상 서버 인스턴스를 제공하는 서비스의 명칭은?

## 보기

- A. Kubernetes Engine
- B. BCS 인스턴스
- C. Object Storage
- D. Data Catalog

## 정답

B

## 해설

카카오클라우드 공식 문서에서 BCS 인스턴스는 강력하고 유연한 클라우드 컴퓨트 자원으로 소개됩니다.
BCS(Basic Cloud Server) 인스턴스는 카카오클라우드의 핵심 컴퓨팅 서비스로, 다양한 워크로드에 맞게 인스턴스 유형을 선택하여 사용할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 관리형 쿠버네티스 서비스로, 컨테이너 오케스트레이션을 담당하며 범용 컴퓨트 자원과는 다릅니다.
- C: Object Storage는 코드 기반 API를 통해 데이터를 저장하고 관리하는 스토리지 서비스입니다.
- D: Data Catalog는 Apache Iceberg 테이블 등록 및 메타데이터를 통합 관리하는 데이터 관리 서비스입니다.
