## 문제

카카오클라우드 Monitoring 서비스에서 서드파티 모니터링 도구(예: Grafana)에 메트릭 데이터를 전달하기 위해 사용하는 기능은 무엇인가요?

## 보기

- A. VPC 피어링 연결을 통한 데이터 전송
- B. Metric Export 엔드포인트를 이용한 외부 전송
- C. Object Storage 버킷에 메트릭 파일 저장
- D. IAM 정책을 통한 데이터 공유

## 정답

B

## 해설

카카오클라우드 Monitoring은 Metric Export 엔드포인트를 제공하여 수집된 메트릭 데이터를 Grafana 등의 외부 서드파티 모니터링 도구에서 조회하고 시각화할 수 있도록 지원합니다.
이를 통해 카카오클라우드 콘솔 외부에서도 동일한 모니터링 데이터를 활용한 통합 대시보드 구성이 가능합니다.

## 오답 포인트

- A: VPC 피어링은 네트워크 연결을 위한 기능으로, 모니터링 데이터 외부 전달과는 관련이 없습니다.
- C: Object Storage 버킷 저장은 Monitoring의 메트릭 외부 전달 방식이 아닙니다.
- D: IAM 정책은 접근 권한을 제어하는 기능이며, 외부 모니터링 도구로 데이터를 전송하는 수단이 아닙니다.
