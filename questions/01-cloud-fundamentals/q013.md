## 문제

카카오클라우드 Kubernetes Engine에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 베어메탈 서버 위에서 직접 쿠버네티스를 설치하고 운영하는 방식이다.
- B. VPC 기반에서 쉽게 쿠버네티스를 실행할 수 있는 관리형 쿠버네티스 서비스이다.
- C. 사용자가 직접 마스터 노드와 워커 노드를 모두 관리해야 하는 서비스이다.
- D. Hadoop Eco 클러스터 위에서 동작하는 컨테이너 오케스트레이션 서비스이다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Kubernetes Engine은 VPC 기반 환경에서 쉽게 쿠버네티스를 실행할 수 있는 관리형(Managed) 쿠버네티스 서비스입니다.
관리형 서비스이므로 마스터 노드 운영 등의 복잡한 관리 부담이 줄어들며, 사용자는 애플리케이션 배포에 집중할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 베어메탈 직접 설치 방식이 아니라 VPC 기반의 관리형 서비스입니다.
- C: 관리형 서비스이므로 사용자가 마스터 노드를 직접 관리할 필요가 없습니다.
- D: Kubernetes Engine은 Hadoop Eco 클러스터와 관련이 없으며, 독립적인 쿠버네티스 관리형 서비스입니다.
