## 문제

카카오클라우드의 버스터블 인스턴스(t1i)에 대한 설명으로 가장 적절한 것은 무엇인가요?

## 보기

- A. 고정된 최대 CPU 성능을 항시 제공하며 별도의 크레딧 메커니즘 없이 동작한다.
- B. CPU 크레딧을 기반으로 평소에는 기준 성능을 유지하고 필요 시 성능을 높일 수 있다.
- C. GPU 가속 연산에 특화되어 딥러닝 학습 워크로드에 최적화된 인스턴스이다.
- D. 컨테이너 기반 서버리스 환경에서만 실행되는 특수 목적 인스턴스이다.

## 정답

B

## 해설

버스터블 인스턴스(t1i)는 CPU 크레딧 제도를 활용하여, 평소에는 기준 성능 수준으로 동작하다가 크레딧이 충분할 때 CPU 성능을 일시적으로 높이는(버스트) 방식으로 동작합니다.
이 방식은 트래픽이 일정하지 않은 워크로드나 개발·테스트 환경처럼 간헐적으로 높은 성능이 필요한 경우에 비용 효율적으로 활용할 수 있습니다.

## 오답 포인트

- A: 버스터블 인스턴스는 항상 최대 성능을 보장하지 않으며, CPU 크레딧 제도가 핵심 동작 원리입니다.
- C: 버스터블 인스턴스는 GPU 가속과 관련이 없으며, 범용 컴퓨팅 워크로드를 대상으로 합니다.
- D: 버스터블 인스턴스는 서버리스 환경이 아닌 일반 VM 기반 인스턴스 유형입니다.
