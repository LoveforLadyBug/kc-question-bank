## 문제

카카오클라우드를 처음 시작할 때 수행해야 하는 기본 설정 단계를 순서대로 나열한 것으로 가장 적절한 것은?

## 보기

- A. 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 역할 추가 → VPC 생성
- B. 프로젝트 생성 → 계정 생성 → VPC 생성 → 결제 수단 등록 → 조직 역할 추가
- C. VPC 생성 → 계정 생성 → 사용자 등록 → 프로젝트 생성 → 결제 수단 등록
- D. 계정 생성 → 프로젝트 생성 → VPC 생성 → 결제 수단 등록 → 조직 역할 추가

## 정답

A

## 해설

카카오클라우드 공식 문서의 빠른 시작 가이드에 따르면, 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성 순서로 기본 설정을 완료합니다.
결제 수단 등록은 계정 생성 직후에 이루어지며, 서비스 사용을 위한 전제 조건이 되므로 프로젝트나 VPC 생성보다 먼저 수행해야 합니다.

## 오답 포인트

- B: 프로젝트 생성이 계정 생성보다 앞에 위치하며, 결제 수단 등록 순서가 잘못되었습니다.
- C: VPC 생성이 가장 먼저 수행되는 것으로 잘못 기술되어 있으며, 결제 수단 등록이 마지막에 위치합니다.
- D: 결제 수단 등록이 프로젝트 생성 이후에 수행되는 것으로 순서가 뒤바뀌어 있습니다.
