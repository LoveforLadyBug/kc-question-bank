## 문제

카카오클라우드 시작 튜토리얼에서 가장 기본적인 시작 시나리오로 소개하는 아키텍처 구성은?

## 보기

- A. Single Server 단일 인스턴스 아키텍처
- B. WEB, APP, DB로 구성된 3 Tier 아키텍처
- C. 마이크로서비스 기반의 서비스 메시 아키텍처
- D. Active-Active 이중화 멀티 리전 아키텍처

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 카카오클라우드를 시작하는 가장 기본적인 시나리오로 WEB, APP, DB로 구성된 3 Tier 아키텍처 구축을 소개합니다.
3 Tier 아키텍처는 프레젠테이션·비즈니스 로직·데이터 계층을 분리하는 전통적이고 보편적인 설계로, 클라우드 환경에서 기본 네트워크 및 서버 구성을 학습하는 데 적합합니다.

## 오답 포인트

- A: 단일 인스턴스 아키텍처는 공식 시작 시나리오로 소개되지 않으며, 확장성과 가용성이 부족합니다.
- C: 서비스 메시 아키텍처는 고급 마이크로서비스 환경에 해당하며, 기초 시작 시나리오로는 적합하지 않습니다.
- D: 멀티 리전 이중화 아키텍처는 기본 시작 수준을 초과하는 고급 구성으로 공식 문서의 기초 시나리오에 해당하지 않습니다.
