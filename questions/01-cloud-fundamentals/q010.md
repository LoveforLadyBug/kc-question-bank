## 문제

카카오클라우드에서 각 가용 영역에 VM과 함께 NAT 인스턴스를 구축하는 주된 목적은?

## 보기

- A. 인스턴스 간 블록 스토리지를 공유하기 위해
- B. 프라이빗 서브넷의 인스턴스가 외부 인터넷에 접근할 수 있도록 하기 위해
- C. 퍼블릭 IP 없이 인스턴스 간 암호화 통신을 수행하기 위해
- D. 데이터베이스 인스턴스의 읽기 성능을 향상시키기 위해

## 정답

B

## 해설

NAT(Network Address Translation) 인스턴스는 프라이빗 서브넷에 위치한 인스턴스가 퍼블릭 IP 없이도 외부 인터넷에 접근할 수 있도록 트래픽을 변환·전달하는 역할을 합니다.
카카오클라우드 튜토리얼에서는 각 가용 영역의 VM에 NAT 인스턴스를 구성하는 방법을 소개하며, 이는 보안을 유지하면서 외부 통신이 필요한 아키텍처에서 핵심 구성 요소입니다.

## 오답 포인트

- A: 블록 스토리지 공유는 NAT 인스턴스가 아닌 공유 스토리지 서비스(예: NFS, SAN)를 통해 구현합니다.
- C: 인스턴스 간 암호화 통신은 VPN 또는 TLS 설정으로 구현하며 NAT의 주 목적이 아닙니다.
- D: 데이터베이스 읽기 성능 향상은 읽기 전용 복제본(Read Replica) 등을 통해 달성하며 NAT와 무관합니다.
