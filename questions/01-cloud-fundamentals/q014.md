## 문제

카카오클라우드 Object Storage에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 블록 스토리지 방식으로 데이터를 저장하며 파일시스템 마운트가 필수이다.
- B. 코드 기반의 API를 제공하여 프로그래밍으로 접근하고 관리할 수 있다.
- C. Apache Iceberg 테이블 형식으로만 데이터를 저장하는 스토리지 서비스이다.
- D. 오직 콘솔(UI)을 통해서만 데이터를 업로드하고 관리할 수 있다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, Object Storage는 코드 기반의 API를 제공하여 사용자가 프로그래밍을 통해 데이터를 저장하고 관리할 수 있도록 지원합니다.
이를 통해 애플리케이션에서 자동화된 방식으로 오브젝트 데이터를 처리할 수 있으며, SDK 및 REST API 방식의 접근이 가능합니다.

## 오답 포인트

- A: Object Storage는 블록 스토리지가 아닌 오브젝트 기반 스토리지 방식으로, 파일시스템 마운트가 필수가 아닙니다.
- C: Apache Iceberg 테이블 관련 기능은 Data Catalog 서비스의 특징이며, Object Storage와는 다릅니다.
- D: 콘솔뿐 아니라 API를 통해서도 데이터를 관리할 수 있으므로 오직 콘솔만 가능하다는 설명은 잘못되었습니다.
