## 문제

카카오클라우드 Kubernetes Engine에 대한 설명으로 옳은 것은?

## 보기

- A. 베어메탈 서버 위에서만 동작하는 전용 쿠버네티스 환경이다.
- B. VPC 기반 위에서 쿠버네티스를 손쉽게 실행할 수 있는 관리형 서비스이다.
- C. 오직 단일 가용 영역에서만 클러스터를 구성할 수 있는 서비스이다.
- D. 사용자가 직접 마스터 노드를 설치하고 운영해야 하는 자체 관리형 서비스이다.

## 정답

B

## 해설

카카오클라우드 Kubernetes Engine은 VPC 기반 위에서 쿠버네티스를 손쉽게 실행할 수 있도록 지원하는 관리형(Managed) 쿠버네티스 서비스입니다.
관리형 서비스이므로 마스터 노드의 운영·관리 부담 없이 클러스터를 생성하고 애플리케이션을 배포할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 베어메탈 전용이 아닌 VPC 기반의 가상화 환경에서 동작합니다.
- C: 단일 가용 영역 제한은 Kubernetes Engine의 특징이 아닙니다.
- D: 관리형 서비스이므로 사용자가 마스터 노드를 직접 설치하거나 운영할 필요가 없습니다.
