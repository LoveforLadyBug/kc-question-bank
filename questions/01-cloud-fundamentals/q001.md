## 문제

카카오클라우드를 처음 시작할 때 수행하는 기본 설정 절차를 올바른 순서로 나열한 것은?

## 보기

- A. 계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성
- B. VPC 생성 → 계정 생성 → 결제 수단 등록 → 사용자 등록 → 프로젝트 생성
- C. 프로젝트 생성 → 계정 생성 → 결제 수단 등록 → VPC 생성 → 사용자 등록
- D. 계정 생성 → 프로젝트 생성 → VPC 생성 → 결제 수단 등록 → 사용자 등록

## 정답

A

## 해설

카카오클라우드 공식 시작 가이드는 '계정 생성 → 결제 수단 등록 → 사용자 등록 및 조직 역할 추가 → 프로젝트 생성 및 프로젝트 역할 추가 → VPC 생성' 순서로 기본 설정을 안내합니다.
결제 수단 등록이 사용자·프로젝트 설정보다 먼저 이루어져야 서비스를 정상적으로 사용할 수 있으며, 네트워크(VPC) 설정은 가장 마지막 단계입니다.

## 오답 포인트

- B: VPC 생성은 계정·결제 설정 이후에 수행해야 하므로 첫 번째 단계가 될 수 없습니다.
- C: 프로젝트 생성은 계정 및 결제 수단 등록 이후에 가능하며, 첫 번째 단계가 아닙니다.
- D: 결제 수단 등록은 프로젝트 생성이나 VPC 생성보다 앞서 이루어져야 합니다.
