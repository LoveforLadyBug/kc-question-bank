## 문제

카카오클라우드 Data Catalog 서비스가 지원하는 기능으로 가장 적절한 것은 무엇인가요?

## 보기

- A. 실시간 스트리밍 데이터를 수집하여 파이프라인을 자동으로 구성한다.
- B. Apache Iceberg 테이블을 등록하고 메타데이터를 통합 관리한다.
- C. 가상 머신의 CPU·메모리 사용량을 실시간으로 모니터링한다.
- D. 데이터베이스 인스턴스의 바이너리 로그를 자동으로 백업한다.

## 정답

B

## 해설

카카오클라우드 Data Catalog는 Apache Iceberg 테이블을 쉽게 등록하고 메타데이터를 통합 관리할 수 있는 데이터 관리 서비스입니다.
다양한 데이터 소스의 메타데이터를 한 곳에서 관리함으로써 데이터 검색과 거버넌스를 효율화하는 역할을 담당합니다.

## 오답 포인트

- A: 실시간 스트리밍 파이프라인 구성은 Data Catalog가 아닌 별도의 스트리밍 처리 서비스 영역에 해당합니다.
- C: CPU·메모리 실시간 모니터링은 Monitoring 서비스의 기능으로, Data Catalog와는 관련이 없습니다.
- D: 바이너리 로그 자동 백업은 데이터베이스 복구 서비스의 기능이며 Data Catalog의 역할이 아닙니다.
