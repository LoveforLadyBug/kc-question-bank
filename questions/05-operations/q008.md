## 문제

카카오클라우드 Monitoring의 **대시보드 지원 서비스** 목록에 포함되지 **않는** 것은?

## 보기

- A. Virtual Machine, Bare Metal Server
- B. Kubernetes Engine(기본), MySQL, PostgreSQL, MemStore
- C. Load Balancing, Pub/Sub
- D. Object Storage, File Storage

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, Monitoring 서비스의 대시보드 지원 서비스에는 Beyond Compute Service(Virtual Machine, Bare Metal Server, GPU), Networking(Load Balancing), Container Pack(Kubernetes Engine 기본), Data Store(MySQL, PostgreSQL, MemStore), Analytics(Pub/Sub)가 포함됩니다. Object Storage와 File Storage는 대시보드 지원 서비스 목록에 포함되지 않습니다.

## 오답 포인트

- A: Virtual Machine과 Bare Metal Server는 Monitoring 대시보드 지원 서비스에 포함됩니다.
- B: Kubernetes Engine(기본), MySQL, PostgreSQL, MemStore는 모두 Monitoring 대시보드 지원 서비스에 포함됩니다.
- C: Load Balancing과 Pub/Sub은 Monitoring 대시보드 지원 서비스에 포함됩니다.
