## 문제

카카오클라우드 Monitoring의 **기본 대시보드**와 **커스텀 대시보드**에 대한 설명으로 옳은 것은?

## 보기

- A. 기본 대시보드는 사용자가 임의로 수정할 수 있으며, 커스텀 대시보드는 카카오클라우드가 제공하는 고정 대시보드이다.
- B. 기본 대시보드와 커스텀 대시보드 모두 사용자가 자유롭게 수정할 수 있다.
- C. 기본 대시보드는 사용자가 임의 수정이 불가능하고 조회만 가능하며, 커스텀 대시보드는 사용자가 직접 생성하여 원하는 메트릭 차트를 추가할 수 있다.
- D. 기본 대시보드와 커스텀 대시보드 모두 별도 설정 없이 바로 사용 가능하다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 기본 대시보드는 카카오클라우드에서 기본 제공하는 대시보드로 별도 설정 없이 바로 사용 중인 리소스의 메트릭을 조회할 수 있지만, 사용자가 임의로 수정하는 것은 불가능합니다. 커스텀 대시보드는 사용자가 직접 생성하고 원하는 서비스의 메트릭 차트를 추가하여 관리할 수 있는 대시보드입니다.

## 오답 포인트

- A: 기본 대시보드와 커스텀 대시보드의 설명이 반대로 되어 있습니다. 기본 대시보드는 카카오클라우드 제공 고정 대시보드이고, 커스텀 대시보드는 사용자가 생성하는 대시보드입니다.
- B: 기본 대시보드는 사용자가 임의 수정이 불가능합니다. 수정 가능한 것은 커스텀 대시보드입니다.
- D: 별도 설정 없이 바로 사용 가능한 것은 기본 대시보드만 해당됩니다. 커스텀 대시보드는 사용자가 직접 생성해야 합니다.
