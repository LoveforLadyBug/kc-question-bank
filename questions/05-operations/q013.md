## 문제

카카오클라우드 **Monitoring**과 **Cloud Trail** 서비스의 차이점에 대한 설명으로 옳은 것은?

## 보기

- A. Monitoring은 사용자 로그인 및 리소스 변경 이력을, Cloud Trail은 서버 상태를 모니터링한다.
- B. Monitoring은 서버, 네트워크, 애플리케이션, 리소스 상태를 모니터링하며, Cloud Trail은 사용자 활동(로그인, 리소스 생성 및 변경 등)을 추적한다.
- C. 두 서비스는 동일한 기능을 제공하며 중복 사용 가능하다.
- D. Monitoring은 보안 감사에, Cloud Trail은 성능 모니터링에 사용된다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **Monitoring**은 클라우드 환경에서 **서버, 네트워크, 애플리케이션, 리소스 상태를 실시간으로 모니터링**하는 서비스입니다. 반면 **Cloud Trail**은 **사용자 활동(로그인, 리소스 생성 및 변경 등)에 대한 모니터링 및 추적**을 담당합니다. 두 서비스는 서로 다른 목적을 가진 보완적인 서비스입니다.

## 오답 포인트

- A: Monitoring과 Cloud Trail의 역할이 반대로 설명되어 있습니다.
- C: 두 서비스는 목적이 다르며 각기 다른 기능을 제공합니다.
- D: Cloud Trail은 사용자 활동 추적에, Monitoring은 리소스 상태 모니터링에 사용됩니다.
