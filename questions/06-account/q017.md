## 문제

카카오클라우드 API 인증 토큰의 **유효 기간**에 대한 설명으로 옳은 것은?

## 보기

- A. API 인증 토큰의 유효 기간은 UTC 기준 12시간이다.
- B. API 인증 토큰의 유효 기간은 KST(한국 표준시) 기준 12시간이다.
- C. API 인증 토큰의 유효 기간은 3시간이며 연장이 불가능하다.
- D. API 인증 토큰은 만료 기한이 없으며 사용자가 직접 폐기해야 한다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, API 인증 토큰의 `issued_at`과 `expired_at` 시간은 **UTC 기준**으로 표시됩니다. 이 값에 +9시간을 더하면 KST 기준으로 확인할 수 있으며, **실제 토큰 사용 가능 기간은 KST 기준 12시간**입니다. 콘솔에서 만료 시간이 3시간으로 표시되는 것은 UTC와 KST 간의 시간 차이 때문에 발생하는 혼동입니다.

## 오답 포인트

- A: 유효 기간은 UTC 기준이 아닌 KST 기준 12시간입니다. UTC로 표시될 때 3시간처럼 보이는 것은 시간대 차이 때문입니다.
- C: 3시간처럼 보이는 것은 UTC 기준 표시 때문이며, 실제 유효 기간은 KST 기준 12시간입니다.
- D: API 인증 토큰은 KST 기준 12시간의 만료 기한이 있습니다.
