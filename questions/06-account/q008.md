## 문제

카카오클라우드 API 토큰이 **유효 기간(12시간) 이전에 조기 만료**될 수 있는 조건으로 옳은 것을 모두 고른 것은?

## 보기

- A. 토큰 발급에 사용한 IAM 액세스 키가 삭제된 경우
- B. 토큰을 발급한 사용자가 해당 프로젝트에서 역할이 제거된 경우
- C. 사용자가 다른 프로젝트에 새로운 역할을 추가한 경우
- D. A와 B 모두

## 정답

D

## 해설

카카오클라우드 공식 문서에 따르면, API 토큰은 기본 유효 기간인 12시간이 지나기 전에도 다음 조건에서 조기 만료됩니다. ① 토큰 발급에 사용한 IAM 액세스 키가 삭제된 경우, ② 토큰을 발급한 사용자가 해당 프로젝트에서 역할이 제거된 경우. 두 조건 모두 토큰의 조기 만료를 유발합니다.

## 오답 포인트

- A만: IAM 액세스 키 삭제 시 조기 만료되는 것은 맞지만, 역할 제거 시에도 조기 만료되므로 A만 선택하는 것은 불완전합니다.
- B만: 역할 제거 시 조기 만료되는 것은 맞지만, 액세스 키 삭제 시에도 조기 만료되므로 B만 선택하는 것은 불완전합니다.
- C: 다른 프로젝트에 새로운 역할을 추가하는 것은 기존 토큰의 만료에 영향을 미치지 않습니다.
