## 문제

카카오클라우드 IAM에서 사용자를 **그룹에서 제거**할 때 발생하는 동작으로 옳은 것은?

## 보기

- A. 그룹을 통해 부여된 역할과 직접 부여된 역할이 모두 즉시 회수된다.
- B. 그룹에서 제거되어도 그룹을 통해 부여된 역할은 다음 로그인 시까지 유지된다.
- C. 그룹을 통해 부여된 역할은 회수되고, 직접 부여된 역할은 유지되며 해당 그룹을 통해 발급된 토큰은 즉시 만료된다.
- D. 그룹을 통해 발급된 토큰은 만료 기한(12시간)까지 계속 사용할 수 있다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, 사용자가 그룹에서 제거되면 **해당 그룹을 통해 부여된 역할은 즉시 회수**됩니다. 그러나 사용자에게 **직접 부여된 역할은 그대로 유지**됩니다. 또한 **그룹을 통해 발급된 토큰은 즉시 만료**되어 더 이상 사용할 수 없습니다. 이 구조는 권한 변경 이력 추적과 보안 사고 대응에 유용합니다.

## 오답 포인트

- A: 직접 부여된 역할은 그룹 제거 시에도 유지됩니다.
- B: 그룹을 통해 부여된 역할은 즉시 회수됩니다. 다음 로그인을 기다리지 않습니다.
- D: 그룹을 통해 발급된 토큰은 만료 기한과 상관없이 즉시 만료됩니다.
