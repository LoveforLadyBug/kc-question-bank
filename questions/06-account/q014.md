## 문제

카카오클라우드 IAM의 **서비스 에이전트 계정(Service Agent Account)**에 대한 설명으로 옳은 것은?

## 보기

- A. 사용자가 직접 생성할 수 있으며, 필요에 따라 수정 및 삭제도 가능하다.
- B. 카카오클라우드 서비스가 리소스를 조작할 때 자동으로 생성되는 시스템 전용 계정이다.
- C. 서비스 에이전트 계정은 콘솔 로그인이 가능하며 사용자가 직접 API를 호출할 수 있다.
- D. 서비스 에이전트 계정 형식은 `{사용자정의}-{프로젝트이름}@kc.serviceaccount.com`이다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **서비스 에이전트 계정**은 사용자가 직접 생성하지 않으며, **카카오클라우드 서비스가 리소스를 조작할 때 자동으로 생성되는 시스템 전용 계정**입니다. 예를 들어 Kubernetes Engine에서 노드풀을 추가하면, 해당 요청을 처리하기 위해 Kubernetes Engine 전용 서비스 에이전트가 자동으로 생성됩니다. 내부 전용 계정으로 콘솔 로그인 또는 API 호출 등 사용자가 직접 제어할 수 없습니다.

## 오답 포인트

- A: 사용자가 직접 생성, 수정, 삭제할 수 없습니다. 단, 프로젝트 삭제 시 해당 서비스 에이전트 계정도 함께 삭제됩니다.
- C: 내부 전용 계정으로 콘솔 로그인이나 사용자가 직접 API를 호출하는 것이 불가능합니다.
- D: `{사용자정의}-{프로젝트이름}@kc.serviceaccount.com` 형식은 서비스 계정의 형식이며, 서비스 에이전트 계정의 형식은 `{프로젝트이름}@{서비스이름}.kc.serviceaccount.com`입니다.
