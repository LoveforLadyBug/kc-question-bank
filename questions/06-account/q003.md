## 문제

카카오클라우드에서 제공하는 **2단계 인증(OTP)** 기능에 대한 설명으로 옳은 것은?

## 보기

- A. OTP는 SMS 문자메시지로만 전송된다.
- B. OTP 인증은 모든 사용자에게 강제로 적용된다.
- C. OTP는 일회용 비밀번호 방식의 2단계 인증으로, 사용자가 선택적으로 설정할 수 있다.
- D. OTP 인증은 API 호출 시에만 사용되며, 콘솔 로그인에는 적용되지 않는다.

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, OTP(One-Time Password)는 일회용 비밀번호를 사용하는 2단계 인증 방식으로, 2025년 9월 16일부터 제공됩니다. 사용자가 선택적으로 설정하여 보안을 강화할 수 있습니다. 로그인 시 추가적인 인증 단계를 거치게 함으로써 계정 보안을 높여줍니다.

## 오답 포인트

- A: OTP는 SMS 전용이 아닌 일회용 비밀번호(OTP 앱 등) 방식을 사용합니다.
- B: OTP 인증은 모든 사용자에게 강제 적용되는 것이 아니라 선택적으로 설정합니다.
- D: OTP 인증은 콘솔 로그인 시 적용되는 2단계 인증입니다.
