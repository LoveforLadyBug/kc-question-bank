## 문제

카카오클라우드 API 토큰의 기본 **유효 기간**으로 옳은 것은?

## 보기

- A. 1시간
- B. 6시간
- C. 12시간
- D. 24시간

## 정답

C

## 해설

카카오클라우드 공식 문서에 따르면, API 토큰의 기본 유효 기간은 발급 시점으로부터 **12시간**입니다. 토큰은 UTC 기준으로 12시간 후 만료되며, KST(UTC+9)로 환산하면 발급 후 12시간 뒤에 만료됩니다. 토큰이 만료된 후에는 새로운 토큰을 발급받아야 합니다.

## 오답 포인트

- A: API 토큰 유효 기간은 1시간이 아닙니다. 12시간입니다.
- B: API 토큰 유효 기간은 6시간이 아닙니다. 12시간입니다.
- D: API 토큰 유효 기간은 24시간이 아닙니다. 12시간입니다.
