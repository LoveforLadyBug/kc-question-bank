## 문제

카카오클라우드 IAM **서비스 계정**의 특성에 대한 설명으로 옳은 것은?

## 보기

- A. 서비스 계정은 IAM 사용자 목록에 포함되며 사용자 할당량에 영향을 준다.
- B. 서비스 계정은 IAM 사용자 목록에 포함되지 않으며 사용자 할당량에도 영향을 주지 않는다.
- C. 서비스 계정은 조직 단위에서 생성하며 프로젝트와 무관하게 동작한다.
- D. 서비스 계정은 이메일/비밀번호 방식으로 콘솔에 로그인할 수 있다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, **서비스 계정은 IAM 사용자 목록에 포함되지 않으며, IAM 사용자 할당량에도 영향을 주지 않습니다**. 서비스 계정은 별도 관리 항목으로 취급되며, 프로젝트당 최대 100개(비활성 포함)까지 생성할 수 있습니다. 콘솔 로그인이 불가하며 IAM 액세스 키를 통한 API 호출만 가능합니다.

## 오답 포인트

- A: 서비스 계정은 IAM 사용자 목록에 포함되지 않으며, 사용자 할당량에 영향을 주지 않습니다.
- C: 서비스 계정은 프로젝트 단위에서 생성되며, 프로젝트 단위 역할만 부여할 수 있습니다.
- D: 서비스 계정은 콘솔 로그인이 불가하며, API 호출 전용 계정입니다.
