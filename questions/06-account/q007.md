## 문제

카카오클라우드에서 사용자가 **비밀번호를 변경**했을 때 발생하는 동작으로 옳은 것은?

## 보기

- A. 비밀번호 변경은 현재 세션에만 영향을 미치며, 기존에 발급된 API 토큰은 유효 기간까지 계속 사용할 수 있다.
- B. 비밀번호 변경 시 모든 활성 세션과 토큰이 만료된다.
- C. 비밀번호 변경은 다른 기기의 세션에는 영향을 미치지 않는다.
- D. 비밀번호 변경 후 새로운 API 토큰을 발급받으려면 관리자 승인이 필요하다.

## 정답

B

## 해설

카카오클라우드 공식 문서에 따르면, 비밀번호를 변경하면 보안을 위해 모든 활성 세션과 토큰이 만료됩니다. 따라서 비밀번호 변경 후에는 다시 로그인하고 필요한 경우 새로운 API 토큰을 발급받아야 합니다. 이는 계정이 탈취되었을 경우를 대비하여 이전에 발급된 모든 세션을 무효화하는 보안 조치입니다.

## 오답 포인트

- A: 비밀번호 변경은 현재 세션만이 아니라 모든 활성 세션과 토큰에 영향을 미칩니다.
- C: 비밀번호 변경은 다른 기기의 세션을 포함한 모든 세션에 영향을 미칩니다.
- D: 비밀번호 변경 후 새로운 API 토큰을 발급받는 데 관리자 승인이 필요하지 않습니다.
