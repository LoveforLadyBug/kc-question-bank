## 문제

카카오클라우드 Kubernetes Engine에서 클러스터를 생성할 때 반드시 선행되어야 하는 네트워크 인프라 구성 요소는?

## 보기

- A. 인터넷 게이트웨이(Internet Gateway)
- B. VPC(Virtual Private Cloud)와 서브넷(Subnet)
- C. 공인 IP(Public IP) 주소 할당
- D. CDN(Content Delivery Network) 설정

## 정답

B

## 해설

Kubernetes Engine은 VPC 기반으로 동작하므로 클러스터 생성 전 VPC와 해당 VPC 내의 서브넷이 먼저 구성되어 있어야 합니다.
클러스터의 노드와 네트워크 통신은 VPC 내 서브넷을 기반으로 이루어지기 때문에, 이 두 요소는 필수 선행 조건입니다.

## 오답 포인트

- A: 인터넷 게이트웨이는 외부 통신을 위한 구성 요소로, 클러스터 생성의 필수 선행 조건은 아닙니다.
- C: 공인 IP 할당은 클러스터 생성 이후 필요에 따라 구성하는 선택적 설정입니다.
- D: CDN은 콘텐츠 배포 네트워크로 Kubernetes 클러스터 구성과 직접적인 관련이 없습니다.
