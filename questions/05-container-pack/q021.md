## 문제

카카오클라우드 Kubernetes Engine에 대한 설명으로 가장 올바른 것은?

## 보기

- A. 사용자가 직접 etcd와 API 서버를 포함한 컨트롤 플레인을 구성하고 유지해야 한다.
- B. VPC를 기반으로 동작하는 관리형 Kubernetes 서비스로, 클라우드 환경에 최적화되어 있다.
- C. 온프레미스 전용 서비스로, 퍼블릭 클라우드 환경에서는 사용할 수 없다.
- D. Kubernetes 버전 업그레이드는 사용자가 수동으로 직접 수행해야 하며 자동화를 지원하지 않는다.

## 정답

B

## 해설

카카오클라우드 Kubernetes Engine은 VPC를 기반으로 쉽게 Kubernetes를 실행할 수 있는 관리형(Managed) 서비스입니다.
관리형 서비스이므로 컨트롤 플레인 구성·운영 부담을 줄여주고, 카카오클라우드의 클라우드 인프라 위에서 동작합니다.

## 오답 포인트

- A: 관리형 서비스이므로 컨트롤 플레인(etcd, API 서버 등)을 사용자가 직접 구성할 필요가 없습니다.
- C: 카카오클라우드 퍼블릭 클라우드 환경에서 제공되는 서비스로, 온프레미스 전용이 아닙니다.
- D: 관리형 서비스의 주요 이점 중 하나는 버전 관리 및 업그레이드 지원이며, 사용자가 수동으로 전부 처리하도록 강제하지 않습니다.
