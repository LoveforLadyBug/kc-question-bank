## 문제

카카오클라우드 Container Registry에 대한 설명으로 옳은 것은?

## 보기

- A. 이미지를 저장할 때 반드시 퍼블릭으로 공개해야 한다.
- B. Docker Hub와 동일한 공개 레지스트리 서비스이다.
- C. 클라우드 환경에서 컨테이너 이미지를 안전하게 보관·관리하는 비공개 이미지 저장소이다.
- D. 이미지 저장은 가능하지만 Kubernetes Engine과 연동은 지원하지 않는다.

## 정답

C

## 해설

Container Registry는 도커 및 컨테이너 이미지를 클라우드 환경에서 안전하게 보관하고 관리할 수 있는 비공개(프라이빗) 이미지 저장소입니다.
Kubernetes Engine을 포함한 카카오클라우드 서비스와 연동하여 빌드·저장·배포 파이프라인을 구성할 수 있습니다.

## 오답 포인트

- A: Container Registry는 비공개 저장소로 운영되며, 이미지를 반드시 퍼블릭으로 공개할 필요가 없습니다.
- B: Docker Hub는 외부의 공개 레지스트리이며, Container Registry는 카카오클라우드 내부의 프라이빗 레지스트리입니다.
- D: Container Registry는 Kubernetes Engine과 연동하여 이미지를 배포하는 데 사용할 수 있습니다.
