## 문제

카카오클라우드 Container Registry의 핵심 특징으로 가장 적절한 것은?

## 보기

- A. 이미지를 퍼블릭 인터넷에 공개하여 누구나 무제한으로 pull할 수 있는 공개 레지스트리 서비스이다.
- B. Docker Hub와 연동하여 외부 이미지를 자동으로 미러링하는 캐시 전용 서비스이다.
- C. 카카오클라우드 환경에서 컨테이너 이미지를 안전하게 저장하고 관리할 수 있는 비공개 이미지 저장소이다.
- D. 컨테이너 이미지 빌드 파이프라인을 자동으로 실행하여 CI/CD를 전담하는 빌드 서비스이다.

## 정답

C

## 해설

Container Registry는 도커 및 컨테이너 이미지를 클라우드 환경에서 안전하게 보관하고 관리할 수 있는 비공개(프라이빗) 이미지 저장소 서비스입니다.
외부에 이미지를 무단 노출하지 않고 팀·프로젝트 단위로 접근을 제어할 수 있어, 보안이 중요한 엔터프라이즈 환경에서 적합하게 활용됩니다.

## 오답 포인트

- A: Container Registry는 퍼블릭 공개 레지스트리가 아닌 비공개(프라이빗) 저장소로, 인가된 사용자만 이미지에 접근할 수 있습니다.
- B: Docker Hub 미러링 캐시 전용 서비스가 아니며, 사용자가 직접 빌드한 이미지를 저장·관리하는 용도로 사용됩니다.
- D: 이미지 빌드 파이프라인 자동화는 별도의 CI/CD 도구 영역이며, Container Registry 자체는 이미지 저장 및 관리에 집중하는 서비스입니다.
