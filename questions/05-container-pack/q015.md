## 문제

카카오클라우드 Kubernetes Engine이 '관리형(Managed) 서비스'인 것의 가장 핵심적인 의미는?

## 보기

- A. 사용자가 원하는 Kubernetes 버전을 직접 컴파일하여 설치할 수 있다.
- B. 마스터 노드의 설치·운영·업그레이드를 카카오클라우드가 대신 관리한다.
- C. 모든 워커 노드의 애플리케이션 배포를 카카오클라우드가 자동으로 수행한다.
- D. 사용자는 kubectl 명령어를 사용할 수 없고 콘솔만으로 관리한다.

## 정답

B

## 해설

관리형 Kubernetes 서비스의 핵심은 마스터(컨트롤 플레인) 노드를 클라우드 제공자가 직접 관리한다는 점으로, 사용자는 마스터 노드 설치·패치·업그레이드 부담 없이 워커 노드와 애플리케이션에 집중할 수 있습니다.
카카오클라우드 Kubernetes Engine은 VPC 기반의 관리형 서비스로, 컨트롤 플레인 운영을 카카오클라우드가 책임집니다.

## 오답 포인트

- A: Kubernetes 버전을 사용자가 직접 컴파일하는 것은 관리형 서비스의 특징이 아니며, 지원되는 버전 중 선택하는 방식입니다.
- C: 애플리케이션 배포는 사용자가 직접 수행하며, 카카오클라우드가 자동으로 대신하지 않습니다.
- D: 관리형 서비스에서도 사용자는 kubectl 등 표준 도구를 통해 클러스터를 제어할 수 있습니다.
