## 문제

카카오클라우드 Container Pack에 포함된 서비스 조합으로 올바른 것은?

## 보기

- A. Kubernetes Engine, Container Registry
- B. Kubernetes Engine, Object Storage
- C. Container Registry, Virtual Machine
- D. Kubernetes Engine, Load Balancer

## 정답

A

## 해설

카카오클라우드 Container Pack은 컨테이너 환경을 효율적으로 활용하고 관리하기 위한 서비스 그룹으로, Kubernetes Engine과 Container Registry 두 가지 서비스로 구성됩니다.
Object Storage, Virtual Machine, Load Balancer는 Container Pack이 아닌 별도의 서비스 카테고리에 속합니다.

## 오답 포인트

- B: Object Storage는 Container Pack이 아닌 스토리지 서비스 카테고리에 속하는 서비스입니다.
- C: Virtual Machine은 컴퓨트 서비스에 해당하며 Container Pack의 구성 요소가 아닙니다.
- D: Load Balancer는 네트워크 서비스 카테고리에 속하며 Container Pack에 포함되지 않습니다.
