## 문제

카카오클라우드 Kubernetes Engine에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 사용자가 직접 Kubernetes 마스터 노드를 설치하고 운영해야 하는 자체 관리형 서비스이다.
- B. VPC를 기반으로 Kubernetes를 손쉽게 실행할 수 있는 관리형 Kubernetes 서비스이다.
- C. 베어메탈 서버 위에서만 동작하며 가상 네트워크를 지원하지 않는 서비스이다.
- D. 단일 노드 클러스터만 지원하며 고가용성 구성이 불가능한 서비스이다.

## 정답

B

## 해설

카카오클라우드 Kubernetes Engine은 VPC(Virtual Private Cloud)를 기반으로 Kubernetes를 손쉽게 실행할 수 있도록 제공되는 관리형(Managed) Kubernetes 서비스입니다.
관리형 서비스이므로 마스터 노드의 설치·운영을 카카오클라우드가 담당하며, 사용자는 워크로드 운영에 집중할 수 있습니다.

## 오답 포인트

- A: Kubernetes Engine은 관리형 서비스로, 마스터 노드를 사용자가 직접 설치·운영할 필요가 없습니다.
- C: Kubernetes Engine은 VPC 기반 가상 네트워크 환경에서 동작하며 베어메탈 전용 서비스가 아닙니다.
- D: 관리형 Kubernetes 서비스는 다중 노드 클러스터와 고가용성 구성을 지원합니다.
