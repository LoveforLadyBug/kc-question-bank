## 문제

카카오클라우드 Container Pack에 포함된 서비스 구성으로 올바른 것은?

## 보기

- A. Kubernetes Engine과 Block Storage
- B. Container Registry와 Object Storage
- C. Kubernetes Engine과 Container Registry
- D. Container Registry와 Virtual Machine

## 정답

C

## 해설

Container Pack은 컨테이너 환경을 효율적으로 활용하고 관리하기 위한 카카오클라우드의 서비스 그룹으로, Kubernetes Engine과 Container Registry 두 가지 서비스로 구성됩니다.
Kubernetes Engine은 컨테이너 오케스트레이션을, Container Registry는 컨테이너 이미지 저장 및 관리를 담당합니다.

## 오답 포인트

- A: Block Storage는 Container Pack에 포함된 서비스가 아닙니다. Container Pack의 구성 요소는 Kubernetes Engine과 Container Registry입니다.
- B: Object Storage는 별도의 스토리지 서비스이며, Container Pack에 포함되지 않습니다.
- D: Virtual Machine은 컴퓨팅 서비스에 해당하며, Container Pack의 구성 요소가 아닙니다.
