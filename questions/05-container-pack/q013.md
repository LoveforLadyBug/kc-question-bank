## 문제

카카오클라우드 Kubernetes Engine에서 클러스터를 생성할 때 반드시 선택해야 하는 네트워크 기반 구성 요소는?

## 보기

- A. 전용 물리 서버(Bare Metal)와 전용 스위치
- B. VPC(Virtual Private Cloud)와 서브넷
- C. 글로벌 CDN 엔드포인트
- D. 공인 IP가 할당된 퍼블릭 게이트웨이

## 정답

B

## 해설

카카오클라우드 Kubernetes Engine은 VPC 기반 위에서 동작하며, 클러스터 생성 시 VPC와 서브넷을 선택하여 노드 및 파드의 네트워크 대역을 구성합니다.
VPC를 기반으로 클러스터 내부 통신과 외부 접근을 격리·제어할 수 있습니다.

## 오답 포인트

- A: 카카오클라우드 Kubernetes Engine은 전용 물리 서버나 전용 스위치를 별도로 요구하지 않으며, 클라우드 가상 네트워크를 활용합니다.
- C: CDN 엔드포인트는 콘텐츠 배포 가속 서비스로, 클러스터 네트워크 구성과 관련이 없습니다.
- D: 공인 IP 게이트웨이는 선택적 구성 요소이며, 클러스터 생성의 필수 조건이 아닙니다.
