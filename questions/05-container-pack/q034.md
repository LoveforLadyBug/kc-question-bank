## 문제

카카오클라우드 Container Pack에 포함된 서비스 구성으로 올바른 것은?

## 보기

- A. Kubernetes Engine과 Object Storage
- B. Kubernetes Engine과 Container Registry
- C. Container Registry와 Virtual Machine
- D. Kubernetes Engine과 Load Balancer

## 정답

B

## 해설

Container Pack은 컨테이너 환경을 효율적으로 활용하고 관리하기 위한 카카오클라우드의 서비스 그룹으로, Kubernetes Engine과 Container Registry 두 가지 서비스로 구성됩니다.
Kubernetes Engine은 컨테이너 오케스트레이션을, Container Registry는 이미지 저장·관리를 담당하여 컨테이너 생애주기 전반을 지원합니다.

## 오답 포인트

- A: Object Storage는 Container Pack에 포함되지 않으며, 별도의 스토리지 서비스 카테고리에 속합니다.
- C: Virtual Machine은 컴퓨팅 서비스 영역이며, Container Pack의 구성 요소가 아닙니다.
- D: Load Balancer는 네트워크 서비스 영역이며, Container Pack에 독립적으로 포함되는 서비스가 아닙니다.
