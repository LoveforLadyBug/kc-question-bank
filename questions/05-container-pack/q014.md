## 문제

카카오클라우드 Container Pack에 포함된 서비스 조합으로 옳은 것은?

## 보기

- A. Kubernetes Engine과 Block Storage
- B. Container Registry와 Object Storage
- C. Kubernetes Engine과 Container Registry
- D. Container Registry와 VM(Virtual Machine)

## 정답

C

## 해설

Container Pack은 컨테이너 환경을 효율적으로 활용·관리하기 위한 카카오클라우드 서비스 그룹으로, Kubernetes Engine과 Container Registry로 구성됩니다.
두 서비스를 함께 사용하면 이미지 저장부터 클러스터 배포까지 일관된 컨테이너 워크플로를 구성할 수 있습니다.

## 오답 포인트

- A: Block Storage는 Container Pack에 포함되지 않는 별도의 스토리지 서비스입니다.
- B: Object Storage는 Container Pack 구성 서비스가 아닙니다.
- D: VM은 컨테이너 서비스가 아닌 가상 머신 서비스로, Container Pack에 포함되지 않습니다.
