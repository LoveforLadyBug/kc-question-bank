## 문제

카카오클라우드 Kubernetes Engine에서 노드 그룹을 구성할 때, 동일한 노드 그룹에 속하는 워커 노드들이 공유하는 특성으로 가장 적절한 것은?

## 보기

- A. 노드마다 서로 다른 운영체제 이미지를 사용하여 다양한 워크로드를 지원한다.
- B. 동일한 인스턴스 타입과 설정을 공유하며, 그룹 단위로 스케일 조정이 가능하다.
- C. 노드 그룹 내 각 노드는 독립적인 VPC에 배치되어 격리된 네트워크를 가진다.
- D. 노드 그룹은 단일 파드만 실행할 수 있으며, 추가 파드는 새 그룹이 필요하다.

## 정답

B

## 해설

Kubernetes Engine의 노드 그룹은 동일한 인스턴스 사양과 설정을 공유하는 워커 노드들의 묶음 단위입니다.
이를 통해 그룹 단위로 노드 수를 늘리거나 줄이는 스케일 조정이 가능하며, 운영 효율성을 높일 수 있습니다.

## 오답 포인트

- A: 동일 노드 그룹 내 노드들은 동일한 운영체제 이미지와 인스턴스 타입을 사용합니다. 노드마다 다른 이미지를 사용하지 않습니다.
- C: 노드 그룹 내 노드들은 동일한 VPC와 서브넷 내에 배치되며, 각각 독립적인 VPC를 갖지 않습니다.
- D: 노드 그룹과 파드 실행 수는 무관하며, 하나의 노드에 여러 파드를 실행할 수 있습니다.
