## 문제

카카오클라우드 Kubernetes Engine에서 노드 그룹을 추가할 때 설정할 수 있는 항목으로 가장 적절한 것은?

## 보기

- A. 마스터 노드의 OS 커널 버전을 사용자가 직접 지정한다.
- B. 노드 그룹별로 인스턴스 타입, 노드 수, 가용 영역을 설정한다.
- C. 노드 그룹 간 네트워크 통신을 차단하는 방화벽 정책을 필수로 설정한다.
- D. 클러스터당 최대 1개의 노드 그룹만 생성할 수 있다.

## 정답

B

## 해설

Kubernetes Engine의 노드 그룹은 동일한 사양의 워커 노드를 묶는 단위로, 그룹 생성 시 인스턴스 타입·노드 수·가용 영역 등을 설정합니다.
하나의 클러스터에 여러 노드 그룹을 만들 수 있어 워크로드 특성에 맞는 유연한 구성이 가능합니다.

## 오답 포인트

- A: 마스터 노드는 카카오클라우드가 관리하며 사용자가 OS 커널 버전을 직접 지정할 수 없습니다.
- C: 노드 그룹 추가 시 방화벽 정책 설정은 필수 항목이 아니며, 네트워크 정책은 별도로 관리됩니다.
- D: 클러스터에는 여러 노드 그룹을 생성할 수 있어 단일 그룹 제한은 잘못된 설명입니다.
