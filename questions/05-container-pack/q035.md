## 문제

카카오클라우드 Kubernetes Engine의 서비스 형태에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 사용자가 마스터 노드를 직접 설치·운영하는 자체 관리형(Self-managed) 서비스이다.
- B. VPC 기반으로 Kubernetes를 손쉽게 실행할 수 있는 카카오클라우드의 관리형 서비스이다.
- C. 컨테이너 없이 가상 머신만으로 마이크로서비스를 운영할 수 있는 환경을 제공한다.
- D. 온프레미스 Kubernetes 클러스터를 클라우드에서 원격으로 모니터링하는 전용 서비스이다.

## 정답

B

## 해설

카카오클라우드 Kubernetes Engine은 VPC를 기반으로 Kubernetes를 손쉽게 실행할 수 있도록 마스터 노드 운영을 카카오클라우드가 대신 관리해주는 관리형(Managed) 서비스입니다.
사용자는 마스터 노드 구성·유지 관리 부담 없이 워커 노드와 워크로드 운영에만 집중할 수 있어, 운영 효율성이 크게 향상됩니다.

## 오답 포인트

- A: Kubernetes Engine은 카카오클라우드가 마스터 노드를 관리해주는 관리형 서비스이므로, 사용자가 마스터 노드를 직접 설치·운영할 필요가 없습니다.
- C: Kubernetes Engine은 컨테이너 기반 오케스트레이션 서비스이며, 가상 머신만으로 마이크로서비스를 운영하는 것과는 다른 개념입니다.
- D: 온프레미스 클러스터 원격 모니터링 전용 서비스가 아니라, 카카오클라우드 환경에서 Kubernetes 클러스터를 직접 생성·운영하는 서비스입니다.
