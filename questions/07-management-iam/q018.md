## 문제

카카오클라우드 Certificate Manager에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 클라우드 리소스의 메트릭 데이터를 차트로 시각화하여 제공하는 서비스
- B. 사용자가 보유한 SSL 인증서를 카카오클라우드에 등록하여 활용할 수 있는 서비스
- C. 리소스 태그를 기반으로 비용을 분류하고 보고서를 생성하는 서비스
- D. 사용자의 API 호출 이력을 수집하고 이상 징후를 탐지하는 서비스

## 정답

B

## 해설

Certificate Manager는 사용자가 직접 보유한 SSL 인증서를 카카오클라우드에 등록하여 서비스에 활용할 수 있도록 지원하는 서비스입니다.
이를 통해 기존에 발급받은 인증서를 카카오클라우드 환경 내 로드밸런서 등의 서비스와 연동하여 HTTPS 통신을 구성할 수 있습니다.

## 오답 포인트

- A: 메트릭 데이터를 차트로 시각화하는 기능은 대시보드(Dashboard) 또는 Metric 서비스에서 제공합니다.
- C: 리소스 태그 기반 비용 분류 및 보고서 생성은 Resource Manager와 별도의 비용 관리 서비스 영역입니다.
- D: API 호출 이력 수집 및 이상 탐지는 Activity Log 또는 별도 보안 서비스의 기능입니다.
