## 문제

담당자가 다음 요구사항을 모두 충족하는 구성을 계획하고 있다. 각 요구사항에 매핑되는 카카오클라우드 Management 서비스가 올바르게 연결된 것은?

> **요구사항**  
> ① 특정 사용자가 어떤 리소스를 생성·삭제했는지 이력을 추적한다.  
> ② 서버 장애 발생 시 담당자에게 즉시 알림을 전송한다.  
> ③ 인스턴스 CPU 사용률을 실시간으로 수집하고 분석한다.

## 보기

- A. ① Activity Tracker — ② Service Event — ③ Metric
- B. ① Metric — ② Activity Tracker — ③ Service Event
- C. ① Service Event — ② Metric — ③ Activity Tracker
- D. ① Activity Tracker — ② Metric — ③ Service Event

## 정답

A

## 해설

① 사용자 행동(생성·삭제) 이력 추적은 Activity Tracker, ② 장애 발생 시 알림 전송은 Service Event의 정책·수신 채널 설정, ③ CPU 사용률 실시간 수집·분석은 Metric 서비스가 담당합니다.
세 서비스는 각각 감사(Audit), 이벤트 알림, 성능 모니터링 역할을 수행하며, Management 서비스 그룹에서 상호 보완적으로 운영됩니다.

## 오답 포인트

- B: Metric은 성능 데이터 수집 서비스이므로 사용자 행동 이력 추적(①)에 적합하지 않으며, 순서가 잘못되었습니다.
- C: Service Event는 알림 서비스이므로 사용자 이력 추적(①) 역할을 수행하지 않으며, 전체 매핑이 틀렸습니다.
- D: ③의 CPU 사용률 분석은 Metric의 역할인데 Service Event로 잘못 매핑되었고, ②의 알림 기능도 Metric이 아닌 Service Event가 담당합니다.
