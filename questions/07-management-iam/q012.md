## 문제

카카오클라우드 Management 서비스 그룹에서 IAM이 담당하는 기능으로 가장 적절한 것은?

## 보기

- A. 클라우드 리소스의 장애 발생 시 알림을 전송하는 정책 설정
- B. 리소스 메타데이터를 기반으로 태그를 생성하고 분류하는 작업
- C. 사용자의 역할에 맞게 리소스 접근 권한을 제어하고 관리하는 기능
- D. 애플리케이션 엔드포인트를 API로 모니터링하고 업무 프로세스를 자동화하는 기능

## 정답

C

## 해설

IAM(Identity and Access Management)은 사용자 역할에 따라 리소스 접근 권한을 세분화하여 제어하고 관리하는 서비스입니다.
이를 통해 조직 내 다양한 사용자가 각자의 역할에 적합한 수준의 권한만을 행사하도록 클라우드 환경을 통제할 수 있습니다.

## 오답 포인트

- A: 장애 발생 시 알림 정책과 수신 채널 설정은 Service Event 서비스의 역할입니다.
- B: 리소스 메타데이터를 활용한 태그 생성과 분류는 Resource Manager 서비스의 기능입니다.
- D: API를 통한 엔드포인트 모니터링 및 업무 자동화는 별도의 API 활용 서비스에 해당합니다.
