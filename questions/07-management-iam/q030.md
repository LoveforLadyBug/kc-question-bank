## 문제

다음 중 카카오클라우드 Management 서비스 그룹에 포함되는 서비스들의 조합으로 옳은 것은?

## 보기

- A. IAM, Dashboard, Activity Log, Object Storage, Metric
- B. IAM, Dashboard, Service Event, Activity Log, Metric, Certificate Manager, Resource Manager, Maintenance
- C. IAM, VPC, Kubernetes Engine, Activity Log, Certificate Manager
- D. Dashboard, Service Event, Block Storage, Resource Manager, Maintenance

## 정답

B

## 해설

카카오클라우드 Management 서비스 그룹은 IAM(접근 권한 관리), Dashboard(모니터링), Service Event(알림), Activity Log(감사 추적), Metric(메트릭 모니터링), Certificate Manager(SSL 인증서), Resource Manager(리소스 태그/검색), Maintenance(유지관리)로 구성됩니다. 이 서비스들은 모두 클라우드 환경의 관리·제어를 목적으로 한 그룹입니다.
각 서비스는 독립적인 기능을 담당하면서도 통합된 관리 경험을 제공하여 클라우드 운영의 일관성을 높입니다.

## 오답 포인트

- A: Object Storage는 스토리지 서비스로 Management 그룹에 속하지 않습니다.
- C: VPC와 Kubernetes Engine은 각각 네트워크, 컨테이너 서비스로 Management 그룹의 구성 요소가 아닙니다.
- D: Block Storage는 스토리지 서비스로 Management 그룹에 포함되지 않으며, 올바른 서비스 조합이 누락되어 있습니다.
