## 문제

카카오클라우드 Management에서 API를 활용하는 주요 목적으로 가장 적절한 것은?

## 보기

- A. 사용자 비밀번호를 암호화하여 안전하게 저장하고 주기적으로 교체하기 위해
- B. 리소스에 부여된 태그 정보를 바탕으로 비용을 팀별로 자동 분할 청구하기 위해
- C. 애플리케이션 엔드포인트를 모니터링하고 업무 프로세스를 자동화하기 위해
- D. 여러 가용 영역에 리소스를 자동 분산 배치하여 고가용성을 확보하기 위해

## 정답

C

## 해설

카카오클라우드 Management에서 API는 애플리케이션의 엔드포인트를 모니터링하고, 업무 프로세스를 자동화할 수 있도록 지원하는 수단으로 활용됩니다.
API를 통한 자동화는 반복 작업의 효율화, CI/CD 파이프라인 통합, 운영 스크립트 작성 등 다양한 DevOps 시나리오에서 활용됩니다.

## 오답 포인트

- A: 비밀번호 암호화 저장 및 교체 정책은 IAM 또는 Secrets Manager 계열 서비스의 기능입니다.
- B: 태그 기반 팀별 비용 분할 청구는 비용 관리 및 리소스 태그 서비스의 조합으로 이루어지며, API 자동화의 주목적이 아닙니다.
- D: 가용 영역 분산 배치를 통한 고가용성 확보는 컴퓨팅·네트워크 아키텍처 설계 영역으로, Management API 활용 목적과 다릅니다.
