## 문제

카카오클라우드 Activity Log가 자동으로 수집·기록하는 사용자 활동 유형으로 가장 적절한 것은?

## 보기

- A. 클라우드 리소스의 CPU 사용률과 네트워크 트래픽 메트릭
- B. SSL 인증서의 발급 기관 정보와 만료 일정
- C. 사용자 로그인, 리소스 생성·변경·삭제 등의 행동 이력
- D. 애플리케이션 API 엔드포인트의 응답 시간과 오류율

## 정답

C

## 해설

Activity Log는 사용자의 로그인, 리소스 생성·변경·삭제 등의 활동을 자동으로 수집하고 기록하여 쉽게 추적할 수 있도록 지원합니다.
이 서비스는 감사(Audit) 목적으로 클라우드 환경에서 발생하는 모든 주요 행동 이력을 체계적으로 관리합니다.

## 오답 포인트

- A: CPU 사용률·네트워크 트래픽 등 성능 메트릭 수집은 Metric 모니터링 서비스의 기능입니다.
- B: SSL 인증서 발급 기관 정보와 만료 일정 관리는 Certificate Manager 서비스에서 담당합니다.
- D: API 엔드포인트 응답 시간·오류율 모니터링은 별도의 API 모니터링 서비스 영역입니다.
