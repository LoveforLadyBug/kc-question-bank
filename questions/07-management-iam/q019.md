## 문제

수십 개의 프로젝트에 걸쳐 수백 개의 리소스를 운영하는 팀이 특정 환경(예: 운영/개발)에 속한 리소스만 빠르게 조회하고 분류하려 합니다. 카카오클라우드에서 이 요구에 가장 적합한 서비스는?

## 보기

- A. Activity Log
- B. Service Event
- C. Resource Manager
- D. Certificate Manager

## 정답

C

## 해설

Resource Manager는 리소스 메타데이터를 활용하여 리소스를 빠르게 검색하고, 태그를 생성하여 원하는 기준으로 분류·관리할 수 있는 서비스입니다.
운영/개발 환경처럼 다양한 기준으로 리소스를 그룹화해야 할 때 태그 기반 관리가 효과적이며, Resource Manager가 이 기능을 전담합니다.

## 오답 포인트

- A: Activity Log는 사용자 행동(로그인, 리소스 생성·삭제 등)의 이력을 기록·추적하는 서비스로 리소스 분류 기능은 없습니다.
- B: Service Event는 리소스 장애 시 알림 정책을 설정하는 서비스이며, 리소스 검색·분류와는 무관합니다.
- D: Certificate Manager는 SSL 인증서 등록 및 관리 서비스로, 리소스 태그 관리 기능을 제공하지 않습니다.
