## 문제

카카오클라우드 Metric 서비스에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 사용자가 보유한 SSL 인증서를 카카오클라우드에 등록하고 관리하는 서비스이다.
- B. 리소스에 태그를 부여하여 원하는 기준으로 분류하고 검색하는 서비스이다.
- C. 클라우드 환경에서 메트릭 데이터를 실시간으로 수집, 저장, 분석할 수 있는 관리형 모니터링 서비스이다.
- D. API를 활용해 애플리케이션 엔드포인트를 모니터링하고 업무 프로세스를 자동화하는 서비스이다.

## 정답

C

## 해설

Metric은 클라우드 환경에서 발생하는 메트릭 데이터를 실시간으로 수집·저장·분석할 수 있는 관리형 모니터링 서비스입니다.
운영자는 이 서비스를 통해 리소스 상태를 지속적으로 관찰하고, 임계값 초과 시 알림을 받거나 과거 데이터를 분석하는 등 다양한 모니터링 작업을 수행할 수 있습니다.

## 오답 포인트

- A: SSL 인증서 등록 및 관리는 인증서 관리(Certificate Manager) 서비스의 기능입니다.
- B: 태그를 통한 리소스 분류 및 검색은 리소스 관리(Resource Manager) 서비스의 기능입니다.
- D: API 기반 엔드포인트 모니터링 및 업무 자동화는 별도의 API 관리 서비스의 설명에 해당합니다.
