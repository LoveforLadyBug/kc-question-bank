## 문제

카카오클라우드 Management 서비스 중, API를 활용하여 애플리케이션의 엔드포인트를 모니터링하고 업무 프로세스를 자동화하는 데 활용되는 서비스는?

## 보기

- A. Activity Log
- B. Resource Manager
- C. Metric
- D. Service Event

## 정답

C

## 해설

Metric 서비스는 클라우드 환경에서 메트릭 데이터를 실시간으로 수집·저장·분석하는 기능 외에도, API를 통해 애플리케이션 엔드포인트를 모니터링하고 업무 프로세스 자동화를 지원합니다.
API 연동을 통해 외부 시스템이나 애플리케이션과 메트릭 데이터를 연계할 수 있어, 운영 자동화 파이프라인 구성에 활용됩니다.

## 오답 포인트

- A: Activity Log는 사용자 로그인, 리소스 생성·변경·삭제 등의 행동 이력을 감사·추적하는 서비스입니다.
- B: Resource Manager는 리소스 메타데이터를 기반으로 리소스를 검색하고 태그로 분류·관리하는 서비스입니다.
- D: Service Event는 리소스 장애 발생 시 알림을 받을 수 있도록 정책과 수신 채널을 설정하는 서비스입니다.
