## 문제

카카오클라우드 Management의 Dashboard 서비스를 통해 사용자가 수행할 수 있는 작업으로 가장 적절한 것은?

## 보기

- A. 사용자의 로그인 이력과 API 호출 내역을 시간순으로 감사한다.
- B. 원하는 차트를 구성하여 주요 서비스 리소스의 메트릭과 로그를 한눈에 확인한다.
- C. SSL 인증서를 업로드하고 유효기간을 모니터링한다.
- D. 클라우드 리소스에 태그를 부여하고 태그 기반으로 리소스를 검색한다.

## 정답

B

## 해설

Dashboard는 주요 서비스 리소스를 모니터링하고, 사용자가 설정한 차트를 통해 메트릭과 로그를 시각적으로 확인할 수 있는 서비스입니다.
원하는 지표를 차트로 구성하여 여러 리소스의 상태를 통합적으로 파악하는 것이 Dashboard의 핵심 기능입니다.

## 오답 포인트

- A: 로그인 이력 및 API 호출 내역 감사는 Activity Log 서비스의 기능입니다.
- C: SSL 인증서 등록 및 유효기간 관리는 Certificate Manager의 역할입니다.
- D: 태그 부여 및 태그 기반 리소스 검색은 Resource Manager의 기능입니다.
