## 문제

카카오클라우드 Service Event에 대한 설명으로 가장 적절한 것은?

## 보기

- A. 클라우드 환경에서 메트릭 데이터를 실시간으로 수집하고 저장하는 모니터링 서비스
- B. 리소스 장애 발생 시 알림을 받을 수 있도록 정책과 수신 채널을 설정하는 서비스
- C. 사용자가 보유한 SSL 인증서를 카카오클라우드에 등록하여 활용하는 서비스
- D. API를 활용해 애플리케이션 엔드포인트를 모니터링하고 업무를 자동화하는 서비스

## 정답

B

## 해설

Service Event는 리소스 장애 발생 시 알림을 받을 수 있도록 정책과 수신 채널을 설정하는 서비스입니다.
이를 통해 운영자는 장애를 사전에 감지하고 신속하게 대응하여 서비스 가용성을 높일 수 있습니다.

## 오답 포인트

- A: 메트릭 데이터의 실시간 수집·저장은 Metric 모니터링 서비스의 기능입니다.
- C: 사용자 보유 SSL 인증서의 등록 및 활용은 Certificate Manager 서비스에 해당합니다.
- D: API 기반 엔드포인트 모니터링과 업무 자동화는 별도 API 서비스 영역의 기능입니다.
