## 문제

카카오클라우드 Certificate Manager 서비스의 주요 목적으로 가장 적절한 것은?

## 보기

- A. 클라우드 리소스의 접근 권한을 역할 기반으로 정의하고 관리한다.
- B. 사용자가 보유한 SSL 인증서를 카카오클라우드에 등록하여 서비스에 활용한다.
- C. 리소스 장애 이벤트를 감지하고 담당자에게 알림을 발송한다.
- D. API 엔드포인트를 모니터링하고 업무 프로세스를 자동화한다.

## 정답

B

## 해설

Certificate Manager는 사용자가 이미 보유하고 있는 SSL 인증서를 카카오클라우드에 등록하여 서비스에 연동·활용할 수 있도록 지원하는 서비스입니다.
외부에서 발급받은 인증서를 클라우드 환경에서 통합 관리함으로써 보안 연결 설정을 간소화할 수 있습니다.

## 오답 포인트

- A: 역할 기반 접근 권한 정의 및 관리는 IAM 서비스의 역할입니다.
- C: 장애 이벤트 감지 및 알림 발송은 Service Event 서비스의 기능입니다.
- D: API 엔드포인트 모니터링과 업무 자동화는 Metric 및 API 연동 기능 영역에 해당합니다.
