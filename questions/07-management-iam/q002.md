## 문제

카카오클라우드 대시보드(Dashboard)에서 제공하는 기능으로 가장 적절한 것은?

## 보기

- A. 사용자 계정의 비밀번호를 주기적으로 자동 변경하는 정책을 설정한다.
- B. 주요 서비스 리소스를 모니터링하고 설정한 차트로 메트릭과 로그를 확인한다.
- C. SSL 인증서를 자동 발급하고 만료 전 갱신 알림을 발송한다.
- D. API 호출 이력을 기반으로 월별 비용 보고서를 자동 생성한다.

## 정답

B

## 해설

대시보드는 주요 서비스 리소스를 모니터링하고, 설정한 차트를 통해 메트릭과 로그를 한눈에 확인할 수 있는 기능을 제공합니다.
이를 통해 운영자는 클라우드 환경의 현재 상태를 실시간으로 파악하고 이상 징후를 신속하게 감지할 수 있습니다.

## 오답 포인트

- A: 비밀번호 자동 변경 정책은 IAM의 계정 보안 설정 영역이며, 대시보드의 기능이 아닙니다.
- C: SSL 인증서 발급 및 갱신 알림은 인증서 관리 서비스(Certificate Manager)의 기능입니다.
- D: API 호출 이력 기반 비용 보고서 생성은 비용 관리 또는 Activity Tracker 관련 기능이지 대시보드의 기능이 아닙니다.
