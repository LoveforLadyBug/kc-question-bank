## 문제

수십 GB 이상의 데이터를 메모리에 상주시키며 실시간 조회 응답 속도를 극대화해야 하는 인메모리 데이터베이스 운영에 가장 적합한 BCS 인스턴스 유형은?

## 보기

- A. 컴퓨팅 최적화(Compute Optimized) 인스턴스
- B. 범용(General Purpose) 인스턴스
- C. 메모리 최적화(Memory Optimized) 인스턴스
- D. 가속 컴퓨팅(Accelerated Computing) 인스턴스

## 정답

C

## 해설

메모리 최적화(Memory Optimized) 인스턴스는 대용량 메모리를 제공하여 데이터를 디스크가 아닌 메모리에 상주시키는 인메모리 데이터베이스나 대규모 데이터 처리 워크로드에 적합합니다. 실시간 조회 성능이 중요한 환경에서 메모리 대역폭을 최대로 활용할 수 있습니다.

## 오답 포인트

- A: 컴퓨팅 최적화 인스턴스는 CPU 처리량은 높지만 메모리 용량보다 CPU 속도가 우선시되므로 인메모리 DB에는 적합하지 않습니다.
- B: 범용 인스턴스는 CPU와 메모리가 균형을 이루나, 대용량 메모리가 핵심인 인메모리 DB에는 메모리 비율이 부족할 수 있습니다.
- D: 가속 컴퓨팅 인스턴스는 AI/ML 연산을 위한 GPU 가속에 특화되어 있어 인메모리 DB 최적화와는 거리가 있습니다.
