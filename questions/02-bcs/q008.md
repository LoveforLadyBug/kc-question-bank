## 문제

카카오클라우드 BCS에서 머신 러닝 모델 훈련과 같은 AI 워크로드에 적합한 인스턴스 유형은?

## 보기

- A. 범용(General Purpose) 인스턴스
- B. 메모리 최적화(Memory Optimized) 인스턴스
- C. 가속 컴퓨팅(Accelerated Computing) 인스턴스
- D. 컴퓨팅 최적화(Compute Optimized) 인스턴스

## 정답

C

## 해설

BCS의 가속 컴퓨팅 인스턴스는 GPU, NPU 등의 가속기를 탑재하여 머신 러닝, 인공 지능과 같은 분야에 특화된 분산 컴퓨팅 환경 구축을 지원합니다. 공식 문서에서도 GPU 인스턴스는 머신 러닝 모델 훈련에서 우수한 성능을 제공한다고 명시하고 있습니다.

## 오답 포인트

- A: 범용 인스턴스는 CPU, 메모리, 네트워크의 균형 잡힌 구성으로 일반적인 워크로드에 적합하며 AI 훈련에는 최적이 아닙니다.
- B: 메모리 최적화 인스턴스는 대용량 메모리가 필요한 워크로드에 적합하며, GPU 연산 집약적 AI 훈련과는 다릅니다.
- D: 컴퓨팅 최적화 인스턴스는 고성능 CPU 연산에 특화되어 있으나, GPU 기반 AI 훈련에는 가속 컴퓨팅 인스턴스가 더 적합합니다.
