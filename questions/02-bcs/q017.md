## 문제

카카오클라우드 BCS 인스턴스에서 기본으로 제공하는 운영체제(OS) 이미지에 포함되는 것으로 올바른 것은?

## 보기

- A. Ubuntu, CentOS(Linux 계열)와 Microsoft Windows 이미지를 기본 제공한다
- B. BCS 인스턴스는 Debian, Fedora 이미지만 지원한다
- C. BCS VM 인스턴스에는 Microsoft Windows 이미지만 사용 가능하다
- D. BCS VM에서 CentOS만 공식 이미지로 제공되며, Ubuntu는 커스텀 이미지로만 사용 가능하다

## 정답

A

## 해설

카카오클라우드 BCS는 Ubuntu, CentOS 등의 Linux 계열 운영체제와 Microsoft Windows를 기본 이미지로 제공합니다.
사용자는 필요에 따라 공식 이미지 외에 커스텀 이미지를 사용할 수도 있으며, 카카오클라우드는 지속적으로 이미지를 최신 상태로 업데이트합니다.

## 오답 포인트

- B: Debian, Fedora는 카카오클라우드 BCS VM 인스턴스의 공식 지원 목록에 없으며, 공식 문서는 Ubuntu와 CentOS를 명시합니다.
- C: BCS VM 인스턴스는 Linux와 Windows 이미지를 모두 지원하며, Windows 전용이 아닙니다.
- D: BCS VM에서 Ubuntu는 공식 이미지로 제공되며, 커스텀 이미지로만 사용해야 하는 제약이 없습니다.
