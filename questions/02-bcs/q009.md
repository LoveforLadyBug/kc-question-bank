## 문제

카카오클라우드 BCS에서 지원하는 운영 체제에 대한 설명으로 옳은 것은?

## 보기

- A. Linux 계열 운영 체제만 지원하며, Windows는 별도 계약이 필요하다
- B. Ubuntu, CentOS 등의 Linux와 Microsoft Windows를 포함한 다양한 운영 체제를 기본으로 제공한다
- C. 운영 체제는 사용자가 직접 설치해야 하며, 사전 구성된 이미지는 제공되지 않는다
- D. 운영 체제 업데이트는 사용자가 수동으로 직접 수행해야 하며, 자동 업데이트 기능은 없다

## 정답

B

## 해설

카카오클라우드 BCS는 Ubuntu, CentOS 등의 Linux 배포판과 Microsoft Windows를 포함한 다양한 운영 체제를 기본으로 제공합니다. 또한 지속적인 인프라 업데이트를 통해 사용자가 항상 최신 버전의 이미지와 기술을 사용할 수 있도록 지원합니다.

## 오답 포인트

- A: BCS는 Linux뿐 아니라 Microsoft Windows도 기본 지원 운영 체제로 제공합니다.
- C: BCS는 운영 체제 소프트웨어를 포함한 사전 구성된 이미지를 선택하여 인스턴스를 시작할 수 있도록 지원합니다.
- D: 카카오클라우드는 소프트웨어 업데이트, 패치 적용 등의 일상적 작업을 자동화하여 최신 상태를 유지합니다.
