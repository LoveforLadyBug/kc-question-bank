## 문제

BCS SmartNIC 기술의 주된 역할로 가장 올바른 것은?

## 보기

- A. 인스턴스의 CPU 코어 수를 동적으로 늘려 연산 성능을 향상시킨다
- B. 네트워크 및 스토리지 I/O 처리를 호스트 CPU 대신 전담하여 인스턴스 성능을 개선한다
- C. GPU 연산을 오프로드하여 딥러닝 학습 속도를 높인다
- D. 물리 서버의 전력 소비를 줄이기 위해 유휴 네트워크 패킷을 압축 처리한다

## 정답

B

## 해설

SmartNIC은 네트워크 패킷 처리, 스토리지 I/O 등의 작업을 호스트 CPU 대신 별도의 전용 칩이 처리하도록 오프로드하는 기술입니다.
이를 통해 호스트 CPU의 부담이 줄어들어 인스턴스에서 실행되는 애플리케이션이 더 많은 컴퓨팅 자원을 활용할 수 있게 됩니다.

## 오답 포인트

- A: CPU 코어 수 동적 확장은 SmartNIC의 역할이 아니라 Scale-up 또는 CPU 오버프로비저닝 관련 기능입니다.
- C: GPU 연산 오프로드는 SmartNIC이 아닌 GPU 가속기(CUDA 등)의 역할입니다.
- D: 전력 절감을 위한 패킷 압축은 SmartNIC의 주된 기능 정의와 다르며, 핵심 목적은 컴퓨팅 성능 향상입니다.
