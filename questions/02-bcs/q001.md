## 문제

카카오클라우드 Virtual Machine(VM)에서 인스턴스의 보안 접근 제어에 사용되는 두 가지 요소는?

## 보기

- A. 보안 그룹(Security Group)과 키 페어(Key Pair)
- B. VPC와 서브넷(Subnet)
- C. IAM 역할과 액세스 키
- D. 로드 밸런서와 오토 스케일링

## 정답

A

## 해설

카카오클라우드 VM은 보안 그룹(Security Group)으로 인바운드/아웃바운드 트래픽을 제어하고, 키 페어(Key Pair)로 SSH 접근을 인증합니다. 두 요소가 함께 인스턴스 보안 접근을 담당합니다.

## 오답 포인트

- B: VPC와 서브넷은 네트워크 격리와 구성에 사용되며, 인스턴스 접근 제어가 아닙니다.
- C: IAM 역할과 액세스 키는 API/콘솔 권한 관리에 사용되며, VM 직접 접근 제어와 다릅니다.
- D: 로드 밸런서와 오토 스케일링은 트래픽 분산 및 확장성을 위한 구성 요소입니다.
