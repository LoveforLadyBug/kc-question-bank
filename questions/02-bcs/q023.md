## 문제

카카오클라우드 BCS GPU 인스턴스에서 지원하는 NVIDIA 라이브러리로 올바른 것은?

## 보기

- A. TensorRT, cuDNN, NCCL
- B. BCS GPU 인스턴스에서 OpenMP, MKL, BLAS를 지원한다
- C. BCS 가속 컴퓨팅 인스턴스에서 Spark, Hadoop, Hive를 활용한다
- D. BCS GPU 인스턴스는 Ansible, Terraform, Docker 라이브러리를 지원한다

## 정답

A

## 해설

카카오클라우드 BCS GPU 인스턴스는 TensorRT(추론 최적화), cuDNN(딥러닝 연산 가속), NCCL(분산 학습 통신) 등 NVIDIA GPU 라이브러리를 지원합니다.
이를 통해 AI/ML 분야에서 BCS GPU 가속 컴퓨팅 환경을 효율적으로 구축할 수 있습니다.

## 오답 포인트

- B: BCS GPU 인스턴스에서는 OpenMP, MKL, BLAS는 CPU 기반 병렬 연산 라이브러리로, GPU 가속 라이브러리가 아닙니다.
- C: BCS 가속 컴퓨팅 인스턴스에서 Spark, Hadoop, Hive는 대규모 분산 데이터 처리 프레임워크이며 NVIDIA GPU 전용 라이브러리가 아닙니다.
- D: BCS GPU 인스턴스에 Ansible, Terraform, Docker는 인프라 자동화 및 컨테이너 도구이며 GPU 연산 라이브러리와 무관합니다.
