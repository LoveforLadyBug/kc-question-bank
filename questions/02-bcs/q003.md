## 문제

딥러닝 모델 학습처럼 대규모 병렬 연산이 필요한 워크로드에 가장 적합한 카카오클라우드 BCS 인스턴스 유형은?

## 보기

- A. 범용(General Purpose) 인스턴스
- B. 메모리 최적화(Memory Optimized) 인스턴스
- C. GPU 인스턴스
- D. Bare Metal Server

## 정답

C

## 해설

GPU 인스턴스는 대규모 병렬 연산에 특화된 그래픽 처리 장치(GPU)를 탑재하고 있어 딥러닝, 머신러닝 학습, 영상 처리 등 고성능 연산 워크로드에 적합합니다.

## 오답 포인트

- A: 범용 인스턴스는 웹 서버, 소규모 데이터베이스 등 일반적인 워크로드에 적합합니다.
- B: 메모리 최적화 인스턴스는 대용량 인메모리 DB, 캐시 서버에 적합하며 병렬 연산에 특화되지 않습니다.
- D: Bare Metal Server는 하이퍼바이저 없이 물리 서버를 직접 사용하는 형태로, GPU 병렬 연산 전용이 아닙니다.
