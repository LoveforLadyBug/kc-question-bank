## 문제

BCS에서 '가상화 기술을 활용하여 물리적 하드웨어 시스템을 가상으로 재구성한 환경'을 설명하는 구성 요소로 가장 적절한 것은?

## 보기

- A. Bare Metal 서버
- B. 가상 머신(VM) 인스턴스
- C. 물리 전용 스토리지
- D. 하드웨어 보안 모듈(HSM)

## 정답

B

## 해설

BCS의 VM 인스턴스는 가상화 기술을 활용하여 물리적 하드웨어 시스템을 가상으로 재구성한 클라우드 컴퓨팅 환경을 제공합니다.
이는 하이퍼바이저를 통해 하나의 물리 서버 위에 여러 독립된 가상 서버를 구성할 수 있는 구조를 의미합니다.

## 오답 포인트

- A: Bare Metal 서버는 가상화 계층 없이 물리 서버를 단독으로 사용하는 방식으로, '가상화 기술'을 활용하는 설명과 반대됩니다.
- C: 물리 전용 스토리지는 데이터 저장 장치를 가리키며, 컴퓨팅 환경 재구성과 관련 없습니다.
- D: HSM은 암호화 키 관리를 위한 보안 장치로, 컴퓨팅 환경 가상화와 무관합니다.
