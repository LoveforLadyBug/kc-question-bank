## 문제

BCS(Beyond Compute Service)의 핵심 개념을 가장 잘 설명한 것은?

## 보기

- A. 물리적 서버를 구매한 사용자에게만 제공되는 전용 호스팅 서비스이다
- B. 가상화 기술을 활용해 물리 하드웨어를 가상으로 재구성한 클라우드 컴퓨팅 환경이다
- C. 사용자가 직접 데이터센터를 운영하면서 가상 서버를 관리하는 방식이다
- D. 인터넷 연결 없이 내부 네트워크에서만 동작하는 프라이빗 서버 서비스이다

## 정답

B

## 해설

BCS는 가상화 기술을 활용하여 물리적 하드웨어 시스템을 가상으로 재구성한 클라우드 컴퓨팅 환경으로, 사용자가 물리적 인프라를 직접 소유하거나 유지할 필요 없이 필요한 만큼 가상 서버를 생성·사용할 수 있습니다.
이는 온디맨드 방식의 클라우드 컴퓨팅의 대표적인 특징으로, 유연성과 확장성을 제공하는 것이 BCS의 핵심 가치입니다.

## 오답 포인트

- A: BCS는 물리 서버 구매 없이 클라우드 환경에서 가상 서버를 이용하는 서비스로, 전용 호스팅과는 다릅니다.
- C: BCS는 카카오클라우드가 인프라를 관리하며, 사용자가 직접 데이터센터를 운영하지 않습니다.
- D: BCS는 인터넷 기반 클라우드 서비스이며, 내부 네트워크 전용 서버 서비스가 아닙니다.
