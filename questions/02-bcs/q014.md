## 문제

카카오클라우드 BCS에서 CPU와 메모리 자원을 균형 있게 제공하여 웹 서버, 중소형 데이터베이스, 개발·테스트 환경에 적합한 인스턴스 유형은?

## 보기

- A. 범용(General Purpose) 인스턴스
- B. 컴퓨팅 최적화(Compute Optimized) 인스턴스
- C. 메모리 최적화(Memory Optimized) 인스턴스
- D. 가속 컴퓨팅(Accelerated Computing) 인스턴스

## 정답

A

## 해설

범용(General Purpose) 인스턴스는 CPU와 메모리를 균형 있게 제공하여 특정 자원에 편중되지 않는 워크로드에 적합합니다. 웹 서버, 중소형 데이터베이스, 개발 및 테스트 환경이 대표적인 사용 사례입니다.

## 오답 포인트

- B: 컴퓨팅 최적화 인스턴스는 높은 CPU 성능이 필요한 배치 처리, HPC, 게임 서버, ML 추론에 적합합니다.
- C: 메모리 최적화 인스턴스는 대용량 메모리가 필요한 인메모리 DB, 대규모 데이터 분석에 적합합니다.
- D: 가속 컴퓨팅 인스턴스는 GPU/NPU 하드웨어 가속이 필요한 딥러닝, AI 워크로드에 적합합니다.
