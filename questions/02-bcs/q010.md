## 문제

카카오클라우드 BCS에서 퍼블릭 IP 주소를 활용하는 시나리오로 가장 적절한 것은?

## 보기

- A. 동일 VPC 내부에서 인스턴스 간 프라이빗 통신을 할 때 퍼블릭 IP가 필수로 필요하다
- B. IP 주소를 변경하지 않고 인스턴스를 다른 네트워크로 이동시키거나, 인터넷을 통해 전 세계 사용자가 접근해야 하는 서비스에 활용된다
- C. 퍼블릭 IP는 인스턴스 생성 시 자동으로 고정 할당되며, 삭제 후에도 계속 유지된다
- D. 퍼블릭 IP 주소는 보안 강화를 위해 내부 서브넷 통신에만 사용하는 것이 권장된다

## 정답

B

## 해설

퍼블릭 IP 주소는 인터넷에서 접근 가능한 고정 IP 주소로, 웹 서버·메일 서버 등 인터넷 접근이 필요한 인스턴스에 할당합니다. 또한 IP 주소를 변경하지 않고 클라우드 인스턴스를 서로 다른 네트워크로 이동시킬 수 있어 다운타임 위험을 줄이고 유연한 인프라 관리가 가능합니다.

## 오답 포인트

- A: 동일 VPC 내부의 인스턴스 간 통신은 프라이빗 IP로 이루어지며, 퍼블릭 IP가 필수는 아닙니다.
- C: 퍼블릭 IP는 자동 고정 할당이 아니라 사용자가 필요에 따라 할당·해제할 수 있는 구조입니다.
- D: 퍼블릭 IP는 인터넷을 통한 외부 접근을 위해 사용하는 것이며, 내부 서브넷 통신 전용이 아닙니다.
