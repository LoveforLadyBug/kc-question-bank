## 문제

카카오클라우드 BCS에서 공인 IP 주소를 인스턴스에 할당할 때 발생하는 비용에 관한 설명으로 옳은 것은?

## 보기

- A. 공인 IP는 BCS 인스턴스에 자동 할당되며 무료이고 할당 개수에 제한이 없다
- B. 공인 IP는 BCS 인스턴스 실행 여부와 관계없이 할당 시 별도 비용이 발생한다
- C. 공인 IP는 BCS 인스턴스가 실행(running) 중일 때만 요금이 부과된다
- D. 공인 IP는 BCS VPC 생성 시 자동으로 1개가 무료 제공된다

## 정답

B

## 해설

카카오클라우드 BCS에서 공인 IP 주소는 별도의 비용이 발생하는 유료 자원입니다.
공인 IP를 BCS 인스턴스에 할당하면 인스턴스 실행 상태와 무관하게 해당 IP에 대한 요금이 청구됩니다.

## 오답 포인트

- A: BCS 인스턴스에서 공인 IP는 무료가 아니며, 사용량에 따라 요금이 부과됩니다.
- C: BCS 인스턴스에서 공인 IP 요금은 인스턴스 실행 여부가 아닌 IP 할당 여부에 따라 책정됩니다.
- D: 공인 IP는 BCS VPC 생성 시 자동으로 무료 제공되지 않으며, 별도로 신청하고 비용을 지불해야 합니다.
