## 문제

딥러닝 모델 학습 작업을 위해 카카오클라우드 BCS 인스턴스를 선택할 때 가장 적합한 인스턴스 유형은?

## 보기

- A. 범용(General Purpose) 인스턴스
- B. 메모리 최적화(Memory Optimized) 인스턴스
- C. 가속 컴퓨팅(Accelerated Computing) 인스턴스
- D. 컴퓨팅 최적화(Compute Optimized) 인스턴스

## 정답

C

## 해설

가속 컴퓨팅(Accelerated Computing) 인스턴스는 GPU/NPU 하드웨어 가속을 활용하여 딥러닝, AI, 데이터 분석 등의 대규모 병렬 연산에 최적화되어 있습니다. 카카오클라우드 BCS는 GPU와 NPU 인스턴스를 제공하여 AI/ML 분산 컴퓨팅 환경 구축을 지원합니다.

## 오답 포인트

- A: 범용 인스턴스는 CPU와 메모리가 균형 잡혀 있어 딥러닝처럼 GPU 연산이 집중되는 워크로드에는 적합하지 않습니다.
- B: 메모리 최적화 인스턴스는 대용량 메모리를 요구하는 인메모리 DB에 특화되며, GPU 가속이 없어 딥러닝에 부적합합니다.
- D: 컴퓨팅 최적화 인스턴스는 CPU 성능이 높지만 딥러닝의 병렬 행렬 연산에는 GPU 가속이 필수이므로 적합하지 않습니다.
