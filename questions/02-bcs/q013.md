## 문제

카카오클라우드 BCS의 CPU 옵션 최적화 기능에 대한 설명으로 옳은 것은?

## 보기

- A. 멀티스레딩은 모든 워크로드에서 성능을 저하시키므로 항상 비활성화하는 것이 권장된다
- B. 고성능 컴퓨팅(HPC)과 같은 특정 워크로드에서는 멀티스레딩을 비활성화하고 CPU 코어당 단일 스레드를 지정하면 성능을 최적화할 수 있다
- C. CPU 옵션 최적화 기능은 메모리 최적화 인스턴스에서만 사용 가능하다
- D. 멀티스레딩 비활성화는 네트워크 처리 성능을 높이기 위한 기능으로, 컴퓨팅 워크로드와는 무관하다

## 정답

B

## 해설

BCS 인스턴스는 멀티스레딩을 지원하며 일반적으로 성능 향상에 도움이 됩니다. 그러나 HPC 워크로드와 같은 특정 환경에서는 멀티스레딩을 비활성화하고 CPU 코어당 단일 스레드를 지정하는 것이 오히려 성능을 최적화할 수 있습니다. 카카오클라우드 BCS는 이러한 멀티스레딩 비활성화 기능을 지원하여 고객의 워크로드 요구 사항에 최적화된 서비스를 제공합니다.

## 오답 포인트

- A: 멀티스레딩은 일반적으로 성능 향상에 도움이 되며, 모든 워크로드에서 항상 비활성화를 권장하는 것은 사실과 다릅니다.
- C: CPU 옵션 최적화 기능이 메모리 최적화 인스턴스에만 한정된다는 내용은 공식 문서에 없으며, BCS 인스턴스 전반에 걸쳐 지원됩니다.
- D: 멀티스레딩 비활성화는 네트워크 처리가 아닌 CPU 코어 성능 최적화를 위한 기능으로, 컴퓨팅 워크로드와 직접적으로 연관됩니다.
