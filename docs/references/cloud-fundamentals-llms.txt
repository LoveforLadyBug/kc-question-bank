### 카카오클라우드 서비스를 자세히 살펴보세요


#### 주요 서비스 개념 이해하기 ​

카카오클라우드 기술 문서의 주요 개념 설명을 통해 각 서비스의 특징과 구성 요소를 자세히 이해할 수 있습니다.
버스터블 인스턴스(t1i)를 소개합니다. 버스터블 인스턴스의 동작 방식, 크레딧 사용에 대한 자세한 정보를 확인하세요.
Data Catalog 서비스에서는 Apache Iceberg 테이블을 손쉽게 등록하고 메타데이터를 통합 관리할 수 있습니다.
자동 백업과 바이너리 로그를 기반으로, 데이터베이스를 원하는 시점으로 되돌려 새 인스턴스 그룹으로 복구할 수 있습니다.

#### How-to Guides ​

카카오클라우드 서비스를 더욱 쉽고 편리하게 사용하기 위한 How-to Guides를 살펴보세요.
카카오클라우드에서 Kubeflow 사용을 위해 필요한 사전 작업과 Kubeflow 생성 및 관리 방법을 상세하게 안내합니다.
Hadoop Eco 서비스에서 클러스터에 설치된 Hive 쿼리, Kafka, Druid 등 컴포넌트를 사용하는 방법을 안내합니다.
카카오클라우드 쿠버네티스 관리형 서비스 Kubernetes Engine에서 클러스터를 생성하는 방법을 알아보세요.
코드 기반의 API를 제공하여 사용자가 프로그래밍을 통해 Object Storage에 접근하고 관리할 수 있도록 지원합니다.
조직 레벨과 프로젝트 레벨에서 역할을 추가, 삭제, 변경하는 방법을 설명합니다.
Monitoring의 Metric Export 엔드포인트를 이용하여 서드파티에서 모니터링 데이터를 확인하는 방법을 안내합니다.
