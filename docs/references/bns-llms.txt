Networking은 카카오클라우드의 네트워킹 서비스 그룹으로 VPC, Transit Gateway, Load Balancing 등이 포함됩니다.
카카오클라우드의 서비스는 각각 특화된 네트워크 기술을 보유하고 있으며, 최신 인프라를 활용하여 안정적이고 빠른 클라우드 환경의 기반을 제공합니다.

#### 서비스 구성 ​

VPC(Virtual Private Cloud)는 논리적으로 격리된 가상의 네트워크 공간을 제공하는 서비스입니다.
사용자 환경에 적절한 트래픽 분산 설정을 통해 효율적으로 처리해주는 서비스입니다.
사용자가 사이트에 접속하기 위해 입력한 도메인을 해당 사이트를 호스팅하는 컴퓨터의 실제 IP 주소로 응답하는 서비스입니다.
다수의 VPC와 온프레미스 네트워크를 연결하여 중앙 집중식 네트워크를 구성하고, 다중 AZ 환경을 지원합니다.
다양한 정보를 튜토리얼과 블로그에서 확인하실 수 있습니다.

Load Balancing 서비스는 사전 설정에 따라 여러 개의 서버로 트래픽을 분산해주는 서비스입니다.
사용자 환경에 적절한 트래픽 분산 설정을 통해 효율적으로 처리함으로써 무중단 서비스를 유지할 수 있습니다.
카카오클라우드는 Application Load Balancer(ALB)와 Network Load Balancer(NLB), Direct Server Return Network Load Balancer(DSRNLB)의 3가지 로드 밸런서 유형를 제공합니다.
사용자는 요구 사항에 따라 원하는 유형의 서비스를 선택하여 사용할 수 있습니다.
